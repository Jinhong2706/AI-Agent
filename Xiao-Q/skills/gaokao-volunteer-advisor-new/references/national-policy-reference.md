# 全国高考政策参考手册

## 一、志愿填报模式分类

### 新高考省份（院校专业组模式）
> 广东、江苏、湖南、湖北、福建、河北、辽宁、重庆（2021年起）
> 广西、安徽、江西、贵州（2024年起）

**投档规则**：考生在同一院校专业组内填报若干专业志愿，平行志愿按"位次优先、遵循志愿、一轮投档"原则执行。未录到专业时，视是否服从组内调剂决定是否退档。

**志愿数量参考**：
- 广东：本科批最多可填45个院校专业组，每组最多6个专业
- 江苏：本科批最多可填40个院校专业组，每组最多6个专业
- 湖南：本科批最多可填45个院校专业组

### 新高考省份（专业+院校模式）
> 浙江、上海、北京、天津、山东（首批改革省份）

**浙江**：专业平行志愿，考生以"专业（类）+学校"为志愿单位，最多填80个志愿，投档到专业（类），不存在退档调剂问题。

**上海**：综合评价+普通批结合，普通批最多24个志愿。

**山东**：以"专业（类）+学校"为单元，96个平行志愿。

### 传统文理科省份（院校平行志愿）
> 河南、山西、陕西、内蒙古、新疆、西藏、青海、宁夏、甘肃、云南、四川（部分批次）、黑龙江、吉林、海南（2024年新高考过渡中）

**投档规则**：以院校为志愿单位，平行志愿按"分数优先、遵循志愿"原则，投档到院校后由院校按专业志愿顺序录取。是否服从专业调剂影响退档概率。

## 三、特殊类型招生规则摘要

### 军校/国防生
- 报名条件：应届高中毕业生，政治审查合格，体检达到军检标准
- 填报时机：本科提前批（部分为提前批特殊类型）
- 注意：录取后实行军事化管理，毕业后服现役，与普通本科批志愿**互不影响**

### 公安/司法院校
- 报名条件：政审、体检（含面试体能测试），裸分+面试综合录取
- 填报时机：提前批
- 注意：各省招生计划少，竞争激烈；有户籍/生源地要求

### 公费师范生（教育部直属六所）
> 北师大、华东师大、东北师大、华中师大、陕师大、西南大学
- 免学费、住宿费，毕业后定向从教10年
- 填报时机：本科提前批（部分省份）或普通本科批
- 注意：协议约束，违约需退还费用

### 定向医学生
- 农村订单定向免费医学生（5年制临床医学）
- 填报时机：本科提前批
- 毕业后回定向县级以下医疗机构服务6年

### 综合评价招生
> 北京大学博雅计划、清华大学领军计划、浙大、复旦、上交等
- 仅限部分省份考生，需提前报名参加学校测试
- 录取分：高考成绩×85% + 学校测试×15%（各校比例不同）

### 少数民族专项/地方专项
- 农村专项计划（国家、高校、地方三类）
- 报名条件：户籍、就读地双满足，不同省份要求不同
- 投档分数线通常低于普通批次

## 五、投档规则详解

### 平行志愿投档（大多数省份普通批次）
1. 按考生位次从高到低排序
2. 每位考生按填报志愿顺序（A→B→C…）依次检索
3. 某志愿院校有计划余额且考生满足条件，即投档
4. 投档后不再检索后续志愿
5. **关键风险**：投档后若专业录取失败（分不够、体检不合格等），且不服从调剂→退档，退档后不再检索该批次其余志愿，**直接滑档**

### 顺序志愿投档（少数省份/批次）
- 第一志愿优先，第一志愿满额后才启用第二志愿
- 风险极高：第一志愿落选后竞争第二志愿的考生多为第一志愿无缘的高分生
- 填报策略：第一志愿必须稳妥，不宜冲高

### 服从专业调剂说明
- **勾选服从**：所报专业均录满时，院校在其他未填专业中调剂，录取概率大幅提升
- **不勾选服从**：所报专业均录满时，直接退档，风险高
- **建议**：稳/保志愿建议勾选服从调剂；若对专业有明确要求则慎重考虑

## 七、就业参考数据口径说明

- **麦可思就业报告**：主要参考应届就业率、起薪、专业对口率（年度更新）
- **教育部绿/黄/红牌专业**：根据失业量、薪资水平综合判断（红牌=警戒，绿牌=优质）
- **学科评估结果**：教育部学位中心发布，A+=全国前2%/前2名，B类=前10%-30%
- **国家级一流专业**：教育部"双万计划"认定，分为国家级和省级两档

## 八、常见"天坑专业"与"伪热门专业"提醒

### 天坑专业（就业难/收入低/发展受限）
- 生化环材（生物、化学、环境、材料）——科研为主，就业市场窄
- 哲学、历史学——体制内或学术路线为主
- 纯文学类（汉语言文学等）——就业宽泛但竞争激烈，起薪偏低

### 名称陷阱专业
- "数字媒体艺术"≠"数字媒体技术"（前者艺术类，后者工科）
- "金融工程"≠"金融学"（前者工学，数学要求极高）
- "信息管理与信息系统"≠"计算机科学"（管理学，非纯CS）
- "公共事业管理"≠政府职能部门直通（就业方向宽泛，不专精）

### 伪热门专业
- 人工智能（本科阶段实为应用为主，研究需读研）
- 大数据管理与应用（新设专业，实力分化大）
- 网络空间安全（精英赛道，普通院校竞争力有限）

## 九、选科限制快速参考（新高考）

| 专业类 | 必选科目要求 |
|--------|-------------|
| 临床医学/口腔医学 | 必选化学+生物（多数院校） |
| 计算机/人工智能/电气工程 | 必选物理（3+1+2须选物理组） |
| 土木/建筑/机械/航空 | 必选物理 |
| 法学/文学/新闻/外语 | 无严格限制，历史选科更友好 |
| 经济/管理/财务 | 无限制，数学能力重要 |
| 护理学 | 要求化学或生物（各校不同） |
| 艺术类 | 参加专业联考+文化成绩 |
