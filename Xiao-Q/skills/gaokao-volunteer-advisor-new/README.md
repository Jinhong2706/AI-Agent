# 高考志愿填报助手 (gaokao-volunteer-advisor)

> 全国通用专业高考志愿填报智能助手，精通全国31省高考政策、录取规则、历年投档数据，提供全流程志愿规划与冲稳保方案。

---

## ✨ 功能亮点

| 功能模块         | 说明                                     |
| ------------ | -------------------------------------- |
| 📋 前置信息收集    | 8项必填维度，缺一不进行正式分析，确保方案精准                |
| 📊 分数/位次精准定位 | 基于全省位次，划分冲(20%)/稳(50%)/保(25%)/底(5%)四梯队 |
| 🗺️ 地域+家庭适配  | 家庭经济条件与意向城市双维度交叉分析，推荐性价比最优方案           |
| 🏫 院校深度解析    | 每所院校输出：基础标签、核心优势、明显短板、录取参考、适配人群        |
| 📚 专业深度解析    | 学科评级、优劣势、报考提醒、转专业难度全覆盖                 |
| 💼 职业前景6维评估  | 就业率 / 方向 / 薪资 / 晋升 / 深造 / 长期趋势全面分析     |
| 📝 分批次志愿方案   | 提前批+本科批+专科批完整志愿表，每条注明推荐理由与风险           |
| ⚠️ 填报风险提醒    | 投档规则风险、专业坑、院校坑、政策坑全维度避坑                |
| 🔄 每日数据自动更新  | 每天07:30自动抓取高校招生动态、省份分数线、就业报告（保留7天）     |

## 🚀 快速开始

### 安装方式

1. 将 `gaokao-volunteer-advisor.zip` 导入 WorkBuddy Skill 管理界面
2. 安装完成后，输入以下任意关键词自动触发：

```
高考志愿 / 志愿填报 / 选大学 / 报考 / 专业选择
志愿方案 / 大学推荐 / 冲稳保 / 录取分数线 / 位次 / 高考报考
```

### 使用流程

```
第一步：AI 列出信息收集清单（8项）
         ↓
第二步：用户逐项填写完整信息
         ↓
第三步：AI 生成完整六板块分析报告
         ↓
第四步：用户可随时提出二次修改需求
```

## 🔄 每日自动数据更新

本 Skill 配套一个每日自动化任务，每天 **07:30** 自动执行：

**抓取内容：**

- 🏛️ **全国政策**：教育部通知、新规变化、专业目录调整
- 🎓 **重点院校**：清北复交浙等985院校招生简章、计划人数变化
- 📈 **各省分数线**：一分一段表发布动态、提前批录取结果
- 💼 **就业报告**：麦可思最新数据、红牌/绿牌专业变化

**结果存储：**

- 自动写入 `references/daily-update.md`
- 滚动保留 **最近7天** 数据，旧记录自动清理

**数据优先级：**

```
实时抓取数据 > references 内置数据 > 模型内置知识
```

## 📊 输出报告格式

分析完成后，AI 按以下结构输出完整报告：

```
【考生基础信息汇总】
  └─ 整理确认用户提供的所有信息

【分数位次定位 & 整体报考区间】
  └─ 可报考层次 + 冲稳保兜底梯队划分

【地域 + 家庭条件综合适配建议】
  └─ 城市分析 + 经济维度院校类型建议

【分批次志愿方案（冲/稳/保完整清单）】
  └─ 每条志愿：推荐理由 + 适配点 + 风险提示

【院校 + 专业逐一深度解析】
  └─ 每所院校/每个专业的优劣势 + 就业前景6维分析

【填报风险提醒 & 实操建议】
  └─ 投档规则 + 专业坑 + 院校坑 + 政策坑
```

## ⚠️ 免责声明

- 本助手所有分析基于**公开数据与历史参考趋势**，仅供参考
- 录取数据存在年份波动，最终以**当年各省招考院官方公布**为准
- 就业前景分析标注"普遍情况"与"参考趋势"，**不构成就业或薪资承诺**
- 建议同步查阅各院校官网最新招生简章，以官方信息为最终依据
