## 详细描述模板（直接复制粘贴）

```markdown
# 健康管理 — 智能健康评估助手

基于普睿科公司"蚩尤智核CFC"的 AI 健康管理助手。

## 核心能力

- **25种常见疾病风险评估**：上传体检报告/化验单，AI即时评估各项疾病的患病概率
- **个性化干预方案**：根据评估结果生成精准、个性化的健康干预建议
- **完整报告呈现**：API 返回结果完整、无损展示，保留所有格式（表格、颜色标识等）
- **HTML健康报告生成**：可生成美观的一体化健康疾病风险评估报告
- **癌症筛查引导**：涉及癌症相关风险时自动显示健康筛查预约链接和咨询热线

## 工作流程

1. 用户上传体检报告/化验单或输入指标数据
2. 助手引导补充：姓名、年龄、性别、地区、职业、既往病史、用药情况等
3. 调用后端医疗大模型 API 进行综合评估
4. 完整呈现 API 返回的分阶段评估结果（阶段一~阶段五）
5. 可生成 HTML 版健康疾病风险评估报告

## 信息采集范围

- 基础人口学特征（姓名、年龄、性别、地区、职业）
- 健康病史（既往病史、家族遗传史、生活方式）
- 用药与特殊状态（当前用药、过敏史、女性特殊状态）

## 安装方式

将整个文件夹复制到 `~/.workbuddy/skills/` 目录下，重启 WorkBuddy 即可使用。

## 技术支持
```
