# 每日热点新闻技能

## 技能描述

为用户提供每日热点新闻简报，采用"60秒早间阅读"格式，让用户快速掌握今日大事。以国内新闻为主体，兼顾国际重要事件。

## 触发条件

当用户说以下任一内容时触发：
- "说新闻"
- "新闻"
- "今日新闻"
- "热点新闻"

**不适用场景**：深度专题调研、特定领域分析、长篇报告

## 执行流程

### 1. 确定目标日期
- 用户未指定日期：使用当前真实日期
- 用户指定日期：使用指定日期
- **重要**：搜索关键词必须包含具体日期（格式："X月X日"），绝不使用"今天"、"今日"等相对时间词

### 2. 三层搜索策略（并行执行）

#### 第一层：主流媒体定向搜索
**国内官方和主流媒体**（关键词需包含媒体名称）：
- 新华社、央视新闻、人民日报、中国新闻网、环球时报、澎湃新闻、新京报、界面新闻
- 同时用 `web_fetch` 直接读取新华网当日要闻页面

**国内商业门户和平台媒体**（关键词需包含平台名称）：
- 腾讯新闻、网易新闻、新浪新闻、搜狐新闻

**国际主流媒体**（关键词需包含媒体名称）：
- 路透社、美联社、BBC、CNN、NHK、韩联社

#### 第二层：领域细分搜索
并行搜索以下领域，每个领域至少获取2-3条结果：
- 房价数据（老百姓钱袋子）
- 科技产品发布
- 汽车新车发布
- 航天太空重大突破（仅限突破性成就，非普通发射）
- 科学研究重大进展
- 人工智能最新消息
- 财经A股黄金
- 民生政策（公积金、医保）
- 教育学校和假期
- 旅游消费机票
- 社会趋势变化
- 热搜热点话题
- 传染病疫情公共卫生

#### 第三层：日期筛选
严格筛选搜索结果，只采用与目标日期一致的新闻

### 3. 新闻价值判断（双维度）

**维度一：价值判断**（首要标准）
- 人类文明进步（科技突破、太空探索里程碑）
- 人类生死存亡（传染病、重大自然灾害）
- 全球平等（不因病贫国家而忽略）
- 民生直接影响（公积金、医保、教育等）
- 社会变迁信号（反映深层变化的现象）
- 全民关注热点（热搜话题）

**维度二：分类判断**（确保覆盖面）
- 先判断价值，再检查是否遗漏领域
- 有价值但被忽视的领域要主动搜索补齐

### 4. 选稿偏好（按优先级排序）
1. 人类文明进步优先（突破性成就，非例行操作）
2. 人类生死存亡优先（无歧视，全球同等关注）
3. 民生直接影响优先
4. 社会变迁信号优先
5. 全民关注热点优先
6. 全球平等（不以国家贫富决定新闻价值）

**控制比例**：
- 纯政治新闻（元首会晤、外交声明）：≤2条
- 战争军事冲突：≤2条
- 体育赛事：≤1条（除非重大突破性赛果）

### 5. 交叉验证规则

**国内新闻**：
- 至少2家官方/主流媒体报道且内容一致
- 官方媒体：新华社、央视新闻、人民日报、中国新闻网
- 主流媒体：澎湃新闻、新京报、界面新闻、第一财经、环球时报
- 单一来源除非是官方通报，否则视为存疑

**国外新闻**：
- 至少2家媒体报道且核心事实一致
- 优先参考：路透社、美联社、BBC、CNN、NHK、韩联社
- 国内媒体转引外媒需确认原文可查

**数据验证**：
- 具体数据必须来自搜索结果明确提及的数值
- 不同来源数据冲突时，以官方/权威来源为准
- 无法确认的数据用"多名"、"巨额"等模糊表述

### 6. 存疑标注规则

重要新闻不应因验证不足而遗漏，但存疑之处必须透明标注。

**标注场景**：
- 来源单一：仅一个媒体/渠道报道 → 标注"仅单一来源报道，待确认"
- 数据冲突：不同媒体关键数据不一致 → 标注具体分歧
- 事实存疑：核心事实描述矛盾 → 标注具体矛盾点
- 未经证实：仅为传言爆料 → 标注"未经官方证实"

**格式**：使用红色字体 `<font>（存疑内容）</font>`

### 7. 绝对禁止事项
1. 编造新闻（虚构事件、人物、数据）
2. 编造数据（凭空捏造数字）
3. 拼凑信息（从不同事件碎片拼凑"新闻"）
4. 猜测填补（靠猜测补充缺失细节）
5. 隐瞒存疑（存疑新闻假装确凿）

### 8. 筛选精炼
- **去重**：同一事件合并为一条，保留最完整版本
- **条数**：基础12-15条，新闻多时可推到18-25条甚至更多
- **均衡**：
  - 国内新闻：60-70%
  - 国际新闻：30-40%
  - 至少覆盖6个不同领域
  - 同一领域：≤3条
  - 同一事件/主题：≤2条
  - 类型多样：政策数据、生活消费、科技前沿等

### 9. 输出格式

```
# 2026年6月19日 星期五 农历五月初四 60秒早间阅读

1. 新闻内容一句话摘要；
2. 新闻内容一句话摘要；
3. 外媒：新闻内容一句话摘要<font>（存疑内容）</font>；
...

【每日金句】积极向上的短句

来源：新华社、央视新闻、人民日报...
```

**格式要求**：
- 首行：井号 + 年月日、星期、农历日期、"60秒早间阅读"
- 每条新闻：阿拉伯数字编号 + 顿号开头，分号结尾
- 内容：一句话摘要，不分行、不加粗、不加emoji
- 来源标注：国内新闻不标注；国外新闻标注"外媒："、"美媒："等
- 存疑标注：红色字体括号，在分号之前
- 末尾：每日金句 + 来源列表（仅列实际搜索到的媒体）

## 参数说明

本技能无需额外参数，自动根据触发时的日期执行。

## 注意事项

1. 搜索时必须使用具体日期，禁止用"今天"、"今日"
2. 严格执行三层搜索策略，确保覆盖面
3. 新闻价值判断优先，不符合价值的不推
4. 交叉验证必须到位，存疑必须标注
5. 输出格式严格按规范，不使用分类标题、emoji前缀或分组
6. 来源列表必须如实标注，不添加未搜索的媒体

## 示例

**用户**：说新闻

**执行**：
1. 确定日期：2026年6月19日
2. 执行三层搜索（并行）
3. 价值判断 + 交叉验证
4. 按格式输出简报

**输出**：
```

2. 国家医保局发布通知，2026年居民医保人均财政补助标准新增30元；

5. 教育部公布2026年全国高考报名人数1291万人，再创历史新高；
...

【每日金句】每一天都是新的开始，让我们一起努力，创造美好未来！

来源：新华社、央视新闻、人民日报、中国新闻网、澎湃新闻、路透社、美联社
```
