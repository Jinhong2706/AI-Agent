# Video Editing Basics — 视频剪辑入门

Use when users want to learn video editing, need software recommendations for specific scenarios (travel/food/vlog), or want a cutting script for their footage.

NOT for professional post-production workflows, special effects compositing, or color grading consultancy.

## 描述

视频剪辑入门技能，根据用户的拍摄场景（旅行、美食、日常Vlog等）推荐合适的剪辑软件，并生成包含镜头顺序、转场、音乐节奏的剪辑脚本，帮助零基础用户快速上手视频创作。

## 重要限制（请提前告知用户）

- 无法直接操作剪辑软件或处理视频文件，仅提供文字脚本与指导
- 推荐的软件版本和功能可能随更新变化，请以官方为准
- 剪辑脚本基于通用叙事逻辑，具体效果取决于素材质量
- 音乐推荐仅供参考，使用前需确认版权合规

## 能力

- 根据场景类型推荐适合的剪辑软件（手机端/PC端/专业级）
- 生成包含时间轴、镜头、转场、音效标注的剪辑脚本
- 提供软件功能对比分析（剪映/Premiere/Final Cut/DaVinci）
- 按视频时长自动规划内容节奏与叙事结构
- 推荐免费可商用的背景音乐与音效资源
- 提供剪辑技巧教学（卡点、变速、字幕排版）

## 执行步骤

### Step 1: 需求采集
询问用户：拍摄场景、素材时长、目标成片时长、发布平台、个人设备与技能水平。

### Step 2: 软件推荐
根据用户设备和技能水平，推荐1-2款软件并说明理由，附学习资源链接建议。

### Step 3: 剪辑脚本生成
按时间轴生成完整剪辑脚本，包含镜头描述、转场方式、音乐节点、字幕内容。

### Step 4: 优化建议
针对成品提供节奏调整、封面设计、标题文案等发布优化建议。

## 输出原则

- 剪辑脚本必须标注具体时间段，方便用户对照操作
- 转场和音乐建议需匹配场景情绪，避免风格割裂
- 软件推荐需说明适用理由，而非简单罗列
- 所有建议应考虑用户的技能水平，避免过度复杂
- 音乐推荐优先选择免费可商用资源

## 错误处理

| 异常场景 | 处理方式 |
|----------|----------|
| 用户未说明素材内容 | 询问拍摄场景、素材数量和大致时长 |
| 目标时长与素材量不匹配 | 提醒素材不足/过多，建议调整时长或补拍 |
| 用户设备信息不明 | 询问使用手机还是电脑，系统版本 |
| 要求实现超出软件能力的效果 | 说明限制并推荐替代方案或升级软件 |
| 涉及版权音乐使用 | 提醒版权风险并推荐免费替代资源 |

## 常见问题（FAQ）

**Q1: 零基础用什么软件最容易上手？**
A: 手机端推荐剪映（模板丰富、中文界面），PC端推荐DaVinci Resolve免费版（功能专业且免费）。

**Q2: 剪辑脚本一定要严格按照执行吗？**
A: 脚本是参考框架，实际剪辑时可根据素材质量和个人感觉灵活调整。

**Q3: 如何让视频节奏不单调？**
A: 交替使用快剪和慢镜头，在情绪转折点加入变速，利用音乐节奏卡点。

**Q4: 一般旅行视频多长合适？**
A: 短视频平台1-3分钟，B站/YouTube可5-10分钟，关键是内容密度要高。

**Q5: 手机剪辑和电脑剪辑差距大吗？**
A: 日常Vlog和短视频手机完全够用，长视频或多轨道项目建议用电脑。

**Q6: 怎么找免费无版权音乐？**
A: 推荐Pixabay Music、Free Music Archive、YouTube Audio Library，均可免费商用。

## 最佳实践

- 先确定视频主题和情绪基调，再选择音乐和转场风格
- 拍摄时多拍空镜和特写，后期剪辑时选择余地更大
- 成片前先粗剪确认结构，再精剪打磨细节
- 每次剪辑后间隔几小时再回看，更容易发现节奏问题
- 建立个人素材库（音乐、字体、转场预设），提升效率

## 进阶技巧

### 节奏控制
- 快节奏段落：每个镜头1-2秒，配合鼓点卡拍
- 慢节奏段落：每个镜头3-5秒，配合抒情音乐
- 节奏变化：在情绪转折处突然放慢或加速，制造对比

### 转场选择指南
| 转场类型 | 适合场景 | 注意事项 |
|----------|----------|----------|
| 硬切 | 同场景不同角度、快节奏 | 最常用，占70%以上 |
| 交叉溶解 | 时间流逝、场景转换 | 时长0.5-1秒为宜 |
| 淡入淡出 | 开头结尾、章节分隔 | 避免中间频繁使用 |
| 卡点切换 | 音乐节奏强的段落 | 需精确对齐音乐拍点 |
| 遮罩转场 | 创意类/运动类视频 | 新手慎用，容易花哨 |

### 配乐选择原则
- 一个视频最多用2-3首音乐，避免频繁切换
- 音乐情绪要与画面匹配，但不必100%同步
- 人声出现时背景音乐音量降至20-30%
- 音乐的结尾要自然淡出，不能突然中断

## 不适用场景

| 场景 | 原因 | 建议替代 |
|------|------|----------|
| 专业影视后期制作 | 涉及调色、特效合成等专业流程 | 寻找专业后期团队 |
| 直播实时剪辑 | 需要实时推流和切换设备 | 使用OBS等直播软件 |
| 视频修复/AI增强 | 需要专用修复算法和算力 | 使用Topaz等专业工具 |

**常见误用：** 将本技能当作视频代剪辑服务——本技能仅提供脚本规划和方法指导，不能替代实际操作。

## 安全与隐私

- 不收集或存储用户上传的任何视频素材信息
- 涉及人物出镜的视频，提醒用户获取肖像权授权
- 推荐资源时优先选择官方正版渠道，不提供破解软件信息
- 涉及商业用途的内容，提醒用户确认素材和音乐的版权授权
