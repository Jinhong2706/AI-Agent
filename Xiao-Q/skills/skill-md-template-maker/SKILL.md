# SKILL.md 模板生成器（腾讯SkillHub专用）

## 一、使用方法
你只需要告诉我一句话：你的技能是做什么的。
例如：
- 我要做一个法拍房风险评估工具
- 我要做一个二手房交易助手
- 我要做一个文案生成小工具
- 我要做一个学习资料整理助手

我会立即为你生成**完整标准的 SKILL.md 全文**，你直接复制粘贴到记事本保存即可上传。

## 二、我会为你自动生成
1. 标准 name（唯一标识）
2. 显示名称 display_name
3. 简介 description
4. 标签 tags
5. 完整使用流程
6. 输入方式
7. 功能说明
8. 输出结果
9. 风险提示与免责声明

## 三、生成后的格式
生成内容为纯文本标准格式，可直接保存为 SKILL.md 并上传到腾讯SkillHub，无需任何修改。

## 四、重要提示
生成内容仅为模板辅助，不涉及违法、违规、投资建议，请合规使用。
