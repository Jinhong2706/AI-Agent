# 美食海报出图技能（SKILL）
## 技能名称
清新治愈风美食食谱竖版图文笔记海报制作

## 适用场景
小红书食谱笔记、美食手账、美食杂志类手机竖版海报，主打家常菜品食谱展示

## 核心风格
融合**小红书爆款排版+日系手账+轻手绘UI+美食杂志风**，整体清新治愈，拒绝过度卡通、低质插画与明显AI绘画感，还原专业美食博主实拍图文质感

## 基础视觉规范
1. 版式：**手机竖版**，4K高清，画面信息饱满且排版整洁
2. 底色：温暖奶油系（米白/浅奶咖），带轻微纸张纹理
3. 配色：整体明亮柔和不刺眼，菜品色彩鲜亮有食欲，元素阴影柔和
4. 装饰：统一使用轻手绘元素（可爱箭头、涂鸦线条、虚线边框、圆角卡片、小图标）点缀

## 固定页面结构（自上而下）
### 1. 顶部标题区
- 主体：大号字体展示菜品名称
- 搭配：蒸汽/爱心/星星/辣椒等手绘小图标
- 附加：1句食欲短句（突出美味、易上手）

### 2. 主视觉区
- 位置：画面视觉中心，占比突出
- 内容：菜品成品特写，还原刚出锅状态，自带光泽、热气效果，采用专业美食摄影质感

### 3. 食材准备模块
- 样式：圆角卡片式布局
- 内容：罗列主料+辅料，每项搭配对应小图标/实物图，清晰标注食材用量

### 4. 制作步骤模块
- 形式：数字编号+箭头串联流程
- 内容：单步配实操过程图+精简文字说明，步骤直观易懂

### 5. 烹饪小贴士模块
- 样式：手绘便签风+虚线边框+小灯泡图标
- 内容：火候、防翻车、增香、食材替代、保存等实用技巧

### 6. 底部氛围文案
搭配短句强化食欲，突出菜品搭配/口感优势

## 避坑要求
杜绝AI绘画感、低质量插画、夸张卡通风格，整体趋近真人美食博主实拍精修图文效果
