# 海外化妆品：ODM/OEM 代工厂线（印刷包装潜客）

## 适用场景

- 目标为**海外**化妆品 **ODM、OEM、合同制造商（CM）**：为品牌方代工成品，通常存在**稳定、批量**的纸盒、外箱、标签说明书需求。
- 决策链多为**供应链、采购、项目部**；与直接找品牌相比，单客户订单可能更集中。
- 我方：**印刷包装**供应商（彩盒、礼盒、运输外箱、说明书等）。

## 公开联系邮箱与联系人（本 Skill 能做什么）

- **可以**：从工厂官网 **Contact、Sales、Business Development** 及展会**官方展商页**摘录公示邮箱、总机、表单入口 URL；若新闻稿或官网写出**具名商务联系人**，可记录并附来源。
- **不可以**：将 LinkedIn 等平台上**未在官网重复公示**的私人邮箱当作官方渠道抄录（除非用户明确要求且符合当地法律与平台规则，本 Skill 默认不这么做）；编造采购经理邮箱。
- **结果预期**：工厂常见 `info@`、`sales@` 或仅在线表单，表中如实填写。

## 用户应提供的输入

- 目标**国家或区域**。
- 代工类型侧重：**护肤灌装**、**彩妆压粉/灌装**、**香氛灌装**或不限。
- 是否只保留具备 **GMP/ISO 等公开认证描述**的工厂（可选，用于初筛）。

## 研究流程（按顺序执行）

1. **区分工厂类型**
   - **ODM**：常提供配方与成品交付；**OEM**：按品牌方配方与包材规范生产；**CM**：合同制造，范围因厂而异。
   - 输出中标注类型依据（官网自述、展会分类、名录分类），不确定则写「待官网确认」。

2. **高信号来源**
   - **展会名录**：Cosmoprof、in-cosmetics 等展区中 **Contract Manufacturing / Private Label / Turnkey** 类目展商。
   - **行业协会**：制造商会员名录（国家或跨境欧洲组织）。
   - **行业媒体 / 目录站**：化妆品制造业排名、地区工厂列表（必须再回官网核对）。

3. **包材与包装相关信号**
   - 官网 **Capabilities / Services** 是否列出 **packaging sourcing、secondary packaging、kitting、gift set assembly** 等。
   - 是否展示 **turnkey**、**full service**（常意味包材纳入总包或强协调）。
   - **案例或客户类型**：服务哪些渠道品牌（仅概括，不泄露未公开客户细节）。

4. **合规与质量公开信息**
   - 记录官网列出的 **GMP、ISO、FDA 注册（若宣称）** 等（原文引用页面，不夸大解读）。

5. **公开联系方式（每条工厂做一次）**
   - 查阅 Contact/Sales/Request a quote 页及展会展商页；记录明文邮箱或「仅联系表单」及 URL。
   - **具名联系人**：仅官网或新闻稿明示姓名+职务时填写。

6. **核验**
   - 工厂须有**独立域名官网**或**展会官方展商页**至少其一；仅有黄页电话无官网的条目标注「待核验」或剔除（在输出开头说明采用的规则）。

## 合规与边界

- 工厂客户名单若未在官网公开，不推测具体品牌客户名称。
- 不使用侵犯商标或内部资料的信息；优先官网与展会官方材料。
- 跨境报价与认证声明以工厂官网及合同为准，本 Skill 不产生法律或合规结论。

## 与企业线、DTC 线的分工

- **企业线**：集团品牌方。
- **DTC 线**：自有品牌、轻资产营销。
- **本线**：**B2B 代工主体**，触达话术侧重产能、交付、包材一致性与认证配合。
