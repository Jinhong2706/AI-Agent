# 自我批评自我反省 / Self-Improving Agent

> 通用AI代理自我反思+自我批评+自我学习记忆系统。

**自我批评自我反省** 是一个通用记忆系统，帮助AI代理从纠错中学习、反思已完成的工作，并随着时间永久改进。适用于 Claude Code、Codex、Copilot 及任何AI代理。

## 核心优势

- **自我反思** — 完成任务后自动评估表现
- **纠错学习** — 从用户纠正中学习模式
- **三级存储** — 热层/温层/冷层分层管理
- **自动晋升** — 3次重复后确认成为永久规则
- **跨平台** — 支持所有AI代理和操作系统

## 安装方式

### 方式一：自动化安装

```bash
# 下载并运行安装脚本
curl -sL <setup-url> | bash

# 或本地运行
bash scripts/setup.sh --tier normal
```

### 方式二：手动安装

1. 创建目录结构
```bash
mkdir -p ~/self-improving/{projects,domains,archive}
```

2. 复制模板文件
```bash
cp templates/memory.md ~/self-improving/memory.md
cp templates/corrections.md ~/self-improving/corrections.md
cp templates/index.md ~/self-improving/index.md
```

3. 集成到工作区
```bash
# 添加到 CLAUDE.md
# 添加到 AGENTS.md
# 添加到 HEARTBEAT.md
```

### 快速验证

```bash

bash ~/self-improving/verify.sh

# 验证安装详情
bash scripts/stats.sh
```

## 典型使用场景

### 场景1: 学习用户纠正

**用户说:** "不，那不对。总是使用2个空格缩进。"

**代理行为:**
1. 记录到 `corrections.md`
2. 计数+1 (1/3)
3. 等待重复触发

**用户再次说:** "我说的是2个空格缩进，不是4个。"

**代理行为:**
1. 计数+1 (2/3)
2. 继续等待

**用户第三次说:** "停止使用4个空格。我已经告诉你两次了。"

**代理行为:**
1. 计数+1 (3/3)
2. 询问确认: "我应该总是使用2个空格缩进吗？"
3. 用户确认后晋升到 `memory.md` (热层)

### 场景2: 自我反思

完成任务后，代理自动评估:

```
上下文(CONTEXT)：构建Flutter UI
反思(REFLECTION)：间距看起来不对，不得不重做
教训(LESSON)：展示给用户前检查视觉间距
```
