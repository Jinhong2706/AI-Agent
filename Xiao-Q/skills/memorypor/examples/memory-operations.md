# 记忆操作示例

## 快速查询

| 用户说 | 动作 |
|-----------|--------|
| "你对X了解多少？" | 在所有层级中搜索X |
| "你学到了什么？" | 显示 `corrections.md` 最近10条 |
| "显示我的模式" | 列出 `memory.md`（热层） |
| "显示[项目]模式" | 加载 `projects/{name}.md` |
| "温层有什么？" | 列出 `projects/` + `domains/` 中的文件 |
| "记忆统计" | 显示每层计数 |
| "忘记X" | 从所有层中移除（先确认） |
| "导出记忆" | ZIP所有文件 |

最近活动（7天）：
  记录的纠错：8

  降级到温层：1

## 导出/导入示例

**导出：**
```bash
bash scripts/export.sh
# 创建：self-improving-backup-2026-05-25.zip
```

**导入：**
```bash
bash scripts/import.sh ~/self-improving-backup-2026-05-25.zip
# 恢复所有记忆文件并备份
```

---

## 分层存储行为

| 层级 | 位置 | 大小限制 | 行为 |
|------|----------|------------|----------|
| 热层 | memory.md | ≤200行 | 始终加载 |
| 温层 | projects/, domains/ | 每文件≤500行 | 按上下文匹配加载 |
| 冷层 | archive/ | 无限 | 按明确查询加载 |

---

## 自动晋升示例

1. 用户3次纠正相同问题
2. 代理提示："我应该把它作为永久规则吗？"
3. 用户确认
4. 规则晋升到热层记忆
5. 代理引用："使用来自纠错的规则（3次确认）"
