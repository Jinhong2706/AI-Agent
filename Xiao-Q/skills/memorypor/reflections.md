# 自我反思日志

跟踪已完成工作中的自我反思。每个条目捕获代理从评估自己的输出中学到什么。

## 格式

```
## [日期] — [任务类型]

**我做了什么：** 简要描述
**结果：** 发生了什么（成功、部分、失败）
**反思：** 我注意到关于我工作的地方
**教训：** 下次要做的不同之处
**状态：** ⏳ candidate（候选）| ✅ promoted（已晋升）| 📦 archived（已归档）
```

## 条目示例

```
## 2026-02-25 — Flutter UI构建

**我做了什么：** 构建了带切换开关的设置屏幕
**结果：** 用户说"间距看起来不对"
**反思：** 我专注于功能，没有可视化检查结果
**教训：** 展示给用户前始终截图评估视觉平衡
**状态：** ✅ 已晋升到 domains/flutter.md
```
