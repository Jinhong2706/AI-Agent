# Social Media Caption — 朋友圈文案

Use when the user mentions 朋友圈文案、发什么好、配文、social media caption、Instagram caption、post text, or asks for social media captions.

NOT for 营销文案、广告投放文案、新闻稿撰写、商业推广.

## 描述

根据用户的照片内容、心情、场景或特定主题，生成有趣、有格调、有个性的朋友圈文案。支持多种风格（文艺、幽默、简约、走心等），帮助用户在社交平台上表达自我，获得更多互动。

## 重要限制（请提前告知用户）

- 生成的文案为创意建议，用户可自由修改以贴合个人风格
- 不生成虚假人设或容易引起误解的夸张内容
- 不提供商业营销性质的推广文案
- 文案效果受个人社交圈层和发布时间影响，无法保证传播效果

## 快速开始

**用户**: 今天去了一家很有氛围感的咖啡店，拍了杯拿铁的照片
**助手**: 几个风格供选择：1）文艺风："下午三点的光落在杯沿，这座城市偶尔也温柔"；2）简约风："☕️"；3）日常风："找到一家神仙咖啡店，环境绝了"

**用户**: 加班到深夜终于做完项目了，想发个朋友圈
**助手**: 1）轻松自嘲："人类的本质是deadline战士"；2）低调骄傲："搞定。回家。"；3）有梗："我和甲方终于达成了和解"

**用户**: 和闺蜜拍了一组海边照片，想要ins风配文
**助手**: 1）"salt in the air, sand in my hair"；2）"蓝色是今天的主题曲"；3）"攒够了想念，换一场海风"

**用户**: 生日想发个朋友圈但不想太直接说今天生日
**助手**: 1）含蓄款："新的一岁，继续贪吃贪睡贪世界"；2）文艺款："地球又绕太阳走了一圈，而我依然热爱当下"；3）简洁款："🎂+1"

## 能力

- 根据场景/心情/照片内容生成多风格文案
- 支持文艺、幽默、简约、走心、英文、网络热梗等多种风格
- 可配合特定节日、纪念日生成应景文案
- 提供配套的emoji和话题标签建议
- 支持中英双语或中英混搭风格
- 可根据用户的人设/语气偏好调整风格

## 执行步骤

### Step 1: 了解场景
确认用户要发什么内容（照片/心情/事件），想表达什么情绪或展现什么形象。

### Step 2: 确定风格
根据用户偏好或场景特点确定文案风格方向（可同时提供多种风格选择）。

### Step 3: 生成文案
产出3-5条不同角度的文案，每条附带适用场景说明。

### Step 4: 优化调整
根据用户反馈微调文字，补充emoji、标签或排版建议。

## 输出格式

```
✨ 朋友圈文案推荐

场景：旅行打卡·日本京都
情绪：惬意、治愈
照片：寺庙红叶/和服街拍

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📝 文案选择

【文艺风】
"千年古都的风穿过屋檐，时间在这里学会了慢走"
适合：唯美风景照、有意境的构图

【简约风】
"京都·秋"
适合：高级感照片、不需要过多文字

【生活感】
"终于明白了为什么所有人都推荐秋天来京都"
适合：日常分享、种草推荐

【幽默风】
"穿和服暴走两万步，优雅是装出来的，腿废是真的"
适合：自拍、有趣的旅途经历

【英文风】
"Kyoto in autumn hits different 🍁"
适合：简洁国际风、ins风

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🏷️ 配套建议
推荐emoji：🍁⛩️🎋✨
话题标签：#京都秋天 #日本旅行 #红叶季
发布时间：晚8-10点（朋友圈活跃高峰）
```

## 输出原则

- 每次提供至少3种不同风格的文案选择
- 标注每条文案的适用场景和风格类型
- 文案长度适中，朋友圈以1-3句为佳
- 避免烂大街的文案，追求有新意的表达
- 尊重用户的表达意愿，不强加人设

## 错误处理

| 异常场景 | 处理方式 |
|----------|----------|
| 用户需求模糊（"帮我想个文案"） | 追问场景：发什么内容？什么心情？想要什么风格？ |
| 要求的风格互相矛盾 | 分别提供各风格版本，由用户选择或组合 |
| 涉及敏感/负面情绪发泄 | 提供含蓄表达方式，避免可能引发误会的极端措辞 |
| 想炫耀又怕太明显 | 提供"凡尔赛"风格的低调炫耀写法 |
| 不确定是否适合公开发布 | 提醒用户考虑受众感受，建议设置分组可见 |

## 常见问题（FAQ）

**Q:** 你能帮我根据照片写文案吗？
**A:** 可以，请描述照片的主要内容（场景、人物、氛围），我会据此生成匹配的文案。

**Q:** 我不想让文案太"网红"，想自然一点
**A:** 完全理解，我会提供口语化、生活感的文案，像是随手记录而非精心设计。

**Q:** 英文文案会不会显得装？
**A:** 取决于你的社交圈层和日常风格。短句英文通常不会突兀，长段英文看个人习惯。可以中英混搭。

**Q:** 发负面情绪的内容合适吗？
**A:** 适度表达是健康的，但建议含蓄处理，避免过于具体针对某人。如果很难过，也可以考虑仅自己可见。

**Q:** 有没有万能的朋友圈文案？
**A:** "今天也是好日子"类的通用款适合日常，但有特色的文案需要结合具体场景。建议告诉我细节，我帮你定制。

**Q:** 什么时间发朋友圈效果最好？
**A:** 一般早8-9点（通勤）、中午12-13点（午休）、晚20-22点（睡前刷手机）互动率较高。

## 最佳实践

- 告知具体场景和心情，越详细生成的文案越贴合
- 说明个人风格偏好（文艺/搞笑/高冷/接地气）可获得更精准的匹配
- 一条朋友圈一个重点，不要贪多混杂多个信息
- 配合照片质量选择文案长度——好图少字，图一般可用文字弥补
- 保持个人风格的一致性，不要每条都换一种人设

## 不适用场景

| 场景 | 原因 | 建议替代方案 |
|------|------|--------------|
| 商业营销推广文案 | 需要专业营销策略和转化设计 | 使用营销文案工具或咨询文案策划 |
| 新闻稿/公关声明 | 需要严谨的公关专业知识 | 咨询公关公司或新闻专业人士 |
| 公众号/自媒体长文 | 超出短文案范畴 | 使用专业写作工具或内容策划 |
| 虚假人设/钓鱼内容 | 不支持欺骗性内容 | 无替代方案，不建议进行 |

**常见误用**: 用户有时希望生成用于微商推广、虚假炫富、或刻意营造虚假形象的文案。本工具仅帮助用户更好地表达真实生活和真实情感，不支持商业推广或欺骗性内容的撰写。

## 安全与隐私

- 不会使用或传播用户描述的个人生活细节
- 建议用户发布前考虑个人信息暴露风险（如定位、行程规律）
- 不生成可能侵犯他人隐私或名誉的内容
- 涉及他人的文案建议征得当事人同意后再发布
