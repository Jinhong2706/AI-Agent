# 风格分类和艺术家参考

## 摄影风格

### 时尚/商业
| 风格 | 特点 | 适用场景 |
|-----|------|---------|
| Editorial | 高端杂志风格，环境简洁 | 时尚、服装 |
| Catalog | 清晰展示产品，无干扰 | 电商、产品 |
| High Fashion | 戏剧化、独特角度 | 奢侈品、广告 |
| Street Style | 自然、街头场景 | 日常穿搭 |

### 纪实/人文
| 风格 | 特点 | 适用场景 |
|-----|------|---------|
| Documentary | 真实瞬间，故事性 | 新闻、记录 |
| Street | 城市生活，抓拍 | 城市纪实 |
| Travel | 地域特色，文化 | 旅游攻略 |
| Portrait | 人物神韵，性格 | 个人写真 |

### 艺术/创意
| 风格 | 特点 | 适用场景 |
|-----|------|---------|
| Fine Art | 画廊感，美学至上 | 艺术创作 |
| Conceptual | 概念表达，创意 | 概念摄影 |
| Experimental | 突破常规，新技术 | 实验作品 |
| Abstract | 非具象，形式美感 | 抽象表达 |

## 艺术家风格参考

### 经典摄影大师
- **Ansel Adams** - 风光摄影，自然壮美，区域曝光法
- **Henri Cartier-Bresson** - 决定性瞬间，街头摄影
- **Annie Leibovitz** - 时尚人像，戏剧化场景
- **Steve McCurry** - 纪实摄影，色彩浓郁
- **Richard Avedon** - 极简人像，纯色背景

### 当代艺术家
- **Erik Almas** - 超现实合成，梦幻感
- ** Platon** - 肖像摄影，强光比，纯色
- **Mickalene Thomas** - 色彩大胆，女性视角
- **Ryan McGinley** - 青春气息，户外自然光
- **Mario Testino** - 时尚摄影，奢华明亮

### 数字艺术风格
- **Beeple** - 赛博朋克，未来主义
- **Magritte 超现实** - 蓝天白云+奇异物体
- **Greg Rutkowski** - 奇幻插画风格
- **ArtStation 风格** - 游戏概念设计感

## 艺术流派风格

| 流派的 | 特点 | 关键词 |
|-------|------|--------|
| 印象派 | 笔触模糊，色彩光影 | impressionist, loose brushwork |
| 野兽派 | 色彩强烈，形体夸张 | Fauvism, vibrant colors |
| 立体主义 | 多角度碎片化 | cubist, geometric |
| 超现实主义 | 梦境般，潜意识 | surrealist, dreamlike |
| 波普艺术 | 流行文化，大众 | pop art, comic style |
| 极简主义 | 简洁纯净 | minimalist, clean |
| 赛博朋克 | 霓虹科技，夜景 | cyberpunk, neon, futuristic |
| 蒸汽朋克 | 维多利亚+机械 | steampunk, vintage machinery |
| 哥特式 | 黑暗浪漫 | gothic, dark romantic |
| 浮世绘 | 日式传统 | ukiyo-e, Japanese woodblock |
| 水彩 | 柔和透明 | watercolor, soft edges |
| 油画 | 质感厚重 | oil painting, canvas texture |
