# 光线、构图、摄影术语

## 光线术语

### 光质 (Light Quality)
| 中文 | 英文 | 说明 |
|-----|------|-----|
| 硬光 | hard light | 边缘清晰，明暗对比强 |
| 柔光 | soft light | 边缘柔和，过渡平缓 |
| 散射光 | diffused light | 经过漫射的光，柔和无刺 |

### 光位 (Light Position)
| 中文 | 英文 | 说明 |
|-----|------|-----|
| 正面光 | front light | 从正面照射，主体受光均匀 |
| 侧光 | side light | 从侧面照射，制造立体感 |
| 逆光 | back light | 从背后照射，轮廓光/剪影 |
| 顶光 | top light | 从上方照射，正午阳光 |
| 脚光 | bottom light | 从下方照射，诡异效果 |
| 45度光 | 45-degree lighting | 经典人像光位 |

### 光比 (Lighting Ratio)
| 中文 | 英文 | 说明 |
|-----|------|-----|
| 平光 | flat lighting | 光比小，明暗接近 |
| 短光 | short lighting | 暗部对着相机 |
| 全光 | full lighting | 受光面面对相机 |
| 显宽光 | broad lighting | 受光面较宽 |
| 显瘦光 | short lighting | 受光面较窄 |

### 特殊光效
| 中文 | 英文 | 说明 |
|-----|------|-----|
| 黄金时段 | golden hour | 日出/日落暖光 |
| 蓝色时段 | blue hour | 日出前/日落后的冷调 |
| 窗口光 | window light | 柔和侧光来源 |
| 眼神光 | catchlight | 眼睛中的光点 |
| 边缘光 | rim light | 勾勒轮廓 |
| 发丝光 | hair light | 头发上的高光 |
| 蝴蝶光 | butterfly light | 鼻子下方的蝴蝶形阴影 |

## 构图术语

### 基本构图法
| 中文 | 英文 | 说明 |
|-----|------|-----|
| 三分法 | rule of thirds | 将画面分成九宫格 |
| 对称构图 | symmetrical composition | 左右/上下平衡 |
| 对角线构图 | diagonal composition | 动态张力 |
| 框架构图 | frame within frame | 用前景做框架 |
| 引导线 | leading lines | 引导视线走向 |
| 留白 | negative space | 简洁呼吸空间 |
| 前景 | foreground | 画面前部元素 |
| 背景 | background | 画面后部元素 |

### 景别 (Shot Size)
| 中文 | 英文 | 说明 |
|-----|------|-----|
| 远景 | extreme wide shot / establishing shot | 交代环境全貌 |
| 全景 | full shot | 完整展示主体 |
| 中景 | medium shot | 膝盖/腰部以上 |
| 半身 | cowboy shot / medium full | 大腿到头顶 |
| 近景 | close-up | 局部重点 |
| 特写 | extreme close-up | 细节极致 |

### 角度 (Camera Angle)
| 中文 | 英文 | 说明 |
|-----|------|-----|
| 鸟瞰 | bird's eye view | 正上方俯视 |
| 俯视 | high angle | 从上往下微斜 |
| 平视 | eye level | 正常水平视角 |
| 仰视 | low angle | 从下往上 |
| 蚁视 | worm's eye view | 超低仰角 |

## 摄影技术参数

### 相机设置
| 中文 | 英文 | 效果 |
|-----|------|-----|
| 大光圈 | wide aperture / f/1.4 - f/2.8 | 背景虚化 |
| 小光圈 | narrow aperture / f/8 - f/16 | 前后都清晰 |
| 广角 | wide angle (24-35mm) | 视野宽，有畸变 |
| 标准 | standard (50mm) | 接近人眼视角 |
| 长焦 | telephoto (85-200mm) | 压缩空间，虚化背景 |
| 高速快门 | fast shutter | 凝固瞬间 |
| 慢速快门 | slow shutter | 运动模糊 |

### 焦距效果
| 焦距 | 特点 |
|-----|------|
| 14-24mm | 超广角，风光/建筑，有透视变形 |
| 35mm | 纪实摄影，自然视角 |
| 50mm | 标准镜头，人文/人像 |
| 85mm | 人像黄金焦距，虚化优美 |
| 135mm | 人像特写，空间压缩 |
| 200mm+ | 远摄，打鸟/运动 |
