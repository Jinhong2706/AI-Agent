# GPT Image 2 提示词生成器

根据用户描述，生成专业级的 GPT Image 2 图像提示词。

## 核心规则

1. **语言转换**：将用户的中文描述转化为英文提示词（GPT Image 2 英文效果更好）
2. **结构化输出**：提示词包含主体、风格、环境、光线、构图等模块
3. **智能增强**：根据描述自动补充专业摄影/艺术术语
4. **负面提示词**：自动生成负面提示词避免常见问题

## 使用流程

### 1. 解析用户需求

用户可能提供：
- **主体描述**：想要画什么（人物、场景、物体）
- **风格偏好**：摄影风格、艺术家风格、艺术流派
- **技术参数**：光线、视角、构图要求
- **用途说明**：照片、海报、Logo 等

### 2. 生成提示词

调用 `references/prompt-templates.md` 获取模板结构和常用表达。

### 3. 构建完整提示词

标准结构：
```
[主体描述], [环境/背景], [光线/照明], [风格/艺术处理], [视角/构图], [技术参数], [质量修饰词]
```

## 额外功能

根据用户需求，可添加以下增强：

| 功能 | 说明 | 触发关键词 |
|-----|------|----------|
| 风格迁移 | 参考特定艺术家/风格 | "做成XX风格"、"参考XX" |
| 光线设计 | 精确控制光线 | "逆光"、"伦勃朗光"、"柔光" |
| 视角控制 | 调整观看角度 | "俯视"、"仰视"、"特写" |
| 构图规则 | 经典构图方法 | "三分法"、"对称构图"、"对角线" |
| 负面提示 | 指定不要的元素 | "不要XX"、"避免XX" |
| 参数微调 | 细节调整 | "暖色调"、"高饱和度"、"景深" |

## 参考资源

- `references/prompt-templates.md` - 提示词模板和常用表达
- `references/style-guide.md` - 风格分类和艺术家参考
- `references/technical-terms.md` - 光线、构图、摄影术语

## 输出格式

生成后，提供：
1. **英文提示词**（可直接用于 GPT Image 2）
2. **负面提示词**（可选）
3. **参数说明**（解释使用了哪些风格/技术）

## 注意事项

- 保持描述简洁具体，避免过于抽象的概念
- 优先使用具体名词而非形容词
- 风格标签放在描述之后
- 质量修饰词放在最后
