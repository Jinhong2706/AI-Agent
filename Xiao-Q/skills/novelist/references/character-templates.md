## 人物设定示例

### 示例1：玄幻小说主角

```
【姓名】林墨
【年龄】18岁（故事开始时）
【性别】男

【外貌特征】
- 身高体型：身材修长但不健壮，有些单薄
- 面容特点：眉目清秀，但眼神中有一种超出年龄的沉稳
- 标志性特征：左手腕有一道环形伤疤
- 穿着风格：常年一袭青色长衫，袖口挽起

【性格特点】
- 核心性格：冷静、坚韧、表面淡然内心执着
- 优点：处事不惊、重承诺、学习能力强
- 缺点：不善表达情感、过度自责、有时过于谨慎
- 说话方式：话语不多但句句在理，很少使用感叹词
- 行为习惯：思考时会摩挲左手的伤疤

【背景故事】
- 出生/成长环境：小镇书香门第，父母早逝
- 重要经历：10岁时目睹宗门灭门，被师父救走
- 心理创伤/执念：找到灭门仇人，但逐渐发现真相并非表面那样

【核心动机】
- 表面目标：修成强者，为宗门报仇
- 深层渴望：找到归属感，证明自己的价值
- 恐惧的事物：再次失去重要的人

【能力设定】
- 主要能力：罕见的双灵根，擅长剑术与阵法
- 弱点/限制：过度使用灵力会引发旧伤
- 成长空间：逐渐觉醒隐藏血脉

【人物弧线】
- 开始时：被仇恨驱动，孤僻冷漠
- 变化过程：在旅途中结识伙伴，学会信任与接纳
- 结束时：放下仇恨，选择守护而非复仇
```

### 示例2：都市小说配角

```
【姓名】沈晴
【与主角关系】同事 → 挚友 → 恋人

【核心特征】
- 外貌印象：短发干练，戴着细框眼镜，职业女性气质
- 性格关键词：独立、毒舌、外冷内热
- 对主角的意义：让主角学会正视自己的情感

【功能定位】
- 剧情作用：推动者——多次在关键时刻点醒主角
- 情感作用：支持者——表面嫌弃实则关心

【关键剧情】
- 出场时机：主角入职新公司
- 重要行为：
  1. 第一次见面就在会议上反驳主角的方案
  2. 在主角低谷期默默帮他完成项目
  3. 醉酒后说出真心话
- 结局走向：与主角修成正果
```

## 人物设计技巧

### 避免人物脸谱化

1. **给缺点空间**
   - 完美的主角不可爱
   - 让优点和缺点相互制衡
   - 缺点应该是情节的种子

2. **言行一致但有小矛盾**
   - 核心性格稳定
   - 但在细节上可以有反差萌
   - 例如：冷酷杀手喜欢养花

3. **动机要有层次**
   - 表面动机：明显、直接
   - 深层动机：隐藏、揭示后让人恍然大悟

### 人物关系要动态变化

- 开篇关系 ≠ 结局关系
- 关系的转变要有事件支撑
- 读者关心的是"如何改变"而非"最终结果"
