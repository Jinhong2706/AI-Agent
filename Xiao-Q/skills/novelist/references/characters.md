# 人物设计参考

## 人物塑造要素

### 基础设定
- 姓名、年龄、性别
- 外貌特征
- 职业/身份
- 家庭背景

### 性格特征
- 主要性格特点（2-3个）
- 优点与缺点
- 价值观与信念
- 恐惧与渴望

### 人物弧光
- 初始状态
- 成长变化
- 最终状态

## 人物关系模板

| 关系类型 | 描述示例 |
|---------|---------|
| 亲情 | 父子矛盾、兄妹情深 |
| 爱情 | 暗恋、双向奔赴、虐恋 |
| 友情 | 生死之交、亦敌亦友 |
| 对立 | 宿敌、商业对手 |
| 师徒 | 传承、代沟 |

## 人物类型参考

### 主角类型
- 成长型 - 从弱变强
- 英雄型 - 天选之人
- 反英雄 - 有缺陷的好人
- 普通人 - 在困境中挣扎

### 反派类型
- 纯粹恶 - 纯粹的邪恶
- 悲剧型 - 有苦衷的反派
- 理念型 - 信念不同
- 灰色角色 - 难以定义善恶

## 人物塑造技巧

1. **Show, don't tell** - 用行动展示性格
2. **矛盾性** - 给人物添加矛盾特质
3. **秘密** - 每个人都有秘密
4. **成长** - 人物要有变化
