# 旅游拍照万能助手

> 套公式就能出片，告别游客照 📸

## 核心定位

好看旅行照 = 主体 + 层次 + 光影 + 留白 + 情绪 + 减法

从构图、层次、光影、角度、动作、减法六大维度，提供极简、好记、可复制的拍照技巧。支持景区/景点拍照建议、最佳时间点、动作指导、道具推荐，单人照/多人照模式全覆盖。

## 6大落地创意公式

| 公式 | 核心内容 |
|------|---------|
| **构图创意** 🖼️ | 三分交点 + 前景遮挡 + 引导线条 + 框架取景 |
| **层次氛围** 🏔️ | 前景 + 中景（人/景点）+ 远景 |
| **光线氛围** ☀️ | 黄金时段 + 侧逆光 + 柔和漫射光 |
| **角度比例** 📐 | 低机位拉长腿 + 高机位收脸 + 侧身45° |
| **动作情绪** 💃 | 不看镜头 + 微动态 + 松弛体态 |
| **高级感减法** ✨ | 去掉杂物 + 缩小主体 + 大面积留白 |

## 3大爆款场景公式

| 场景 | 公式 |
|------|------|
| **古镇/古城** 🏮 | 复古感 = 古风浅穿搭 + 门框框架构图 + 慢动作抓拍 + 暖色调 |
| **海边/湖边** 🌊 | 松弛感 = 大留白天空/海面 + 逆光 + 侧身走动 + 蓝白配色 |
| **山林/山野** 🌲 | 治愈感 = 绿植当前景 + 林间柔光 + 远景群山 + 低饱和色调 |

## 单人照 vs 多人照

| 模式 | 核心 | 要点 |
|------|------|------|
| **单人照** 👤 | 突出个人，人景融合 | 人占1/3，自然动作 |
| **多人照** 👥 | 互动感，层次感 | 前后错落，有互动 |

## 示例

**输入**："乌镇，下午3点，单人，复古氛围"

**输出**：
- 时间调整：建议4:30-5:30（光线柔和）
- 推荐点位：石桥、回廊、水边、古门
- 穿搭建议：白色长裙+油纸伞
- 动作指导：撑伞过桥、倚门回望、坐石阶
- 构图要点：门框框架、石板路引导线
