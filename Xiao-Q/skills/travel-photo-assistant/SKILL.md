# 旅游拍照万能助手

## 概述

**极简、好记、可复制的旅游拍照万能公式。** 📸

好看旅行照 = 主体 + 层次 + 光影 + 留白 + 情绪 + 减法

从构图、层次、光影、角度、动作、减法六大维度，提供直接套用的拍照技巧。支持景区/景点拍照建议、最佳时间点、动作指导、道具推荐，单人照/多人照模式全覆盖。

> **核心理念：套公式就能出片，告别游客照。**

---

## 核心定位

### 服务对象

- 旅游爱好者
- 喜欢拍照打卡的人
- 想提升拍照技术的人
- 导游/旅游博主
- 想拍出朋友圈爆款的人

### 使用场景

- 旅游前做拍照攻略
- 到达景点不知道怎么拍
- 想拍出有氛围感的照片
- 不知道怎么摆姿势
- 不知道什么时间点拍最好

### 核心目标

> **套公式就能出片，让每个人都能拍出好看的旅行照。**

## 核心万能总公式

```
好看旅行照 = 主体 + 层次 + 光影 + 留白 + 情绪 + 减法
```

| 要素 | 作用 | 占比 |
|------|------|------|
| **主体** | 画面焦点 | 20% |
| **层次** | 画面深度 | 20% |
| **光影** | 氛围感 | 20% |
| **留白** | 高级感 | 15% |
| **情绪** | 感染力 | 15% |
| **减法** | 干净度 | 10% |

## 6大落地创意拆分公式

---

### 公式1：构图创意公式 🖼️

```
构图 = 三分交点 + 前景遮挡 + 引导线条 + 框架取景
```

#### 1.1 三分交点

**九宫格构图法**：

```
┌────┬────┬────┐
│    │ ★  │    │  ← 交点1（左上）
├────┼────┼────┤
│ ★  │    │ ★  │  ← 交点2（左中）、交点3（右中）
├────┼────┼────┤
│    │ ★  │    │  ← 交点4（左下）
└────┴────┴────┘

人物放在4个★位置，画面更平衡
```

**手机设置**：
- iPhone：设置 → 相机 → 网格
- 安卓：相机设置 → 参考线/网格

#### 1.2 前景遮挡

**天然画框**：

| 场景 | 前景元素 | 效果 |
|------|---------|------|
| **古镇** | 门框、窗框 | 复古感 |
| **自然** | 树叶、花朵 | 层次感 |
| **建筑** | 栏杆、柱子 | 框架感 |
| **街道** | 遮阳伞、招牌 | 氛围感 |

**拍摄技巧**：
- 前景虚化（靠近前景拍摄）
- 人站在框架中间
- 前景占画面1/4

#### 1.3 引导线条

**自然引导线**：

| 场景 | 引导线 | 效果 |
|------|--------|------|
| **道路** | 公路、小路 | 纵深感 |
| **楼梯** | 台阶、扶手 | 向上感 |
| **海岸** | 海岸线、沙滩 | 延伸感 |
| **河流** | 溪流、桥梁 | 流动感 |

**拍摄技巧**：
- 人站在引导线尽头或中间
- 低角度拍摄，强化线条
- 线条从画面一角延伸到对角

#### 1.4 框架取景

**人造框架**：

| 框架类型 | 示例 | 适用场景 |
|---------|------|---------|
| **门窗** | 古门、花窗 | 古镇、建筑 |
| **植物** | 树枝、花丛 | 自然、公园 |
| **建筑** | 拱门、回廊 | 城市、景区 |
| **道具** | 镜子、相框 | 创意、室内 |

### 公式2：层次氛围感公式 🏔️

```
画面层次 = 前景 + 中景（人/景点）+ 远景
```

#### 2.1 三层构图

```
┌─────────────────────┐
│                     │
│    【远景】          │  ← 山、海、城市天际线
│                     │
│      ★人物★         │  ← 【中景】你本人/地标
│                     │
│   【前景】🌿         │  ← 花、草、树枝、玻璃
│                     │
└─────────────────────┘

一层叠一层，告别扁平游客照
```

#### 2.2 前景选择

| 场景 | 前景元素 | 效果 |
|------|---------|------|
| **花丛** | 花朵、花瓣 | 浪漫感 |
| **树林** | 树叶、树枝 | 自然感 |
| **古镇** | 灯笼、招牌 | 氛围感 |
| **海边** | 礁石、浪花 | 层次感 |
| **城市** | 栏杆、玻璃 | 现代感 |

#### 2.3 中景主体

**人物位置**：
- 占画面1/3到1/2
- 不要居中（用三分法）
- 与背景有互动

**景点地标**：
- 标志性建筑
- 特色景观
- 文化符号

#### 2.4 远景背景

| 场景 | 远景元素 | 效果 |
|------|---------|------|
| **山水** | 远山、云海 | 壮阔感 |
| **海边** | 海平面、天空 | 松弛感 |
| **城市** | 天际线、高楼 | 现代感 |
| **古镇** | 屋檐、街巷 | 历史感 |

### 公式3：光线氛围公式 ☀️

```
绝美光线 = 黄金时段 + 侧逆光 + 柔和漫射光
```

#### 3.1 黄金时段

| 时段 | 时间 | 特点 | 效果 |
|------|------|------|------|
| **日出** | 5:00-7:00 | 柔和、冷调 | 清新感 |
| **上午** | 8:00-10:00 | 明亮、自然 | 活力感 |
| **日落前** | 16:00-18:00 | 温暖、 golden hour | 氛围感 |
| **蓝调** | 日落后30分钟 | 蓝紫色调 | 神秘感 |

**最佳拍摄时间**：
- 上午9-10点
- 下午4-6点（日落前1-2小时）

#### 3.2 光线方向

| 光线类型 | 特点 | 效果 | 适用 |
|---------|------|------|------|
| **顺光** | 光源在身后 | 明亮、清晰 | 风景照 |
| **侧光** | 光源在侧面 | 立体、有层次 | 人像 |
| **逆光** | 光源在身后 | 剪影、发光 | 氛围感 |
| **顶光** | 光源在头顶 | 阴影重 | 避免正午 |

#### 3.3 特殊光线效果

**逆光拍摄**：
- 发丝发光
- 温柔氛围
- 剪影效果

**侧光拍摄**：
- 五官立体
- 有层次感
- 轮廓分明

**阴天拍摄**：
- 天然柔光
- 不翻车
- 适合人像

### 公式4：角度比例公式 📐

```
人像出片角度 = 低机位拉长腿 + 高机位收脸 + 侧身45°
```

#### 4.1 低机位（拍全身）

**技巧**：
- 手机放低（膝盖高度）
- 微微仰拍（15-30度）
- 脚尖抵画面底部

**效果**：
- 拉长腿型
- 显高显瘦
- 气场增强

#### 4.2 高机位（拍头像）

**技巧**：
- 手机举高（眼睛以上）
- 微微俯拍（15-30度）
- 眼睛看镜头上方

**效果**：
- 收脸显瘦
- 眼睛有神
- 可爱感

#### 4.3 侧身45°

**技巧**：
- 身体侧对镜头45度
- 头转回来看镜头
- 重心在后腿

**效果**：
- 显瘦
- 有层次
- 不死板

#### 4.4 避免正面死板

**不要**：
- ❌ 正对镜头，站得笔直
- ❌ 双脚并拢，僵硬站立
- ❌ 平视拍摄，无角度

**要**：
- ✅ 侧身、转体、有角度
- ✅ 重心转移，有曲线
- ✅ 高低角度，有变化

---

### 公式5：动作情绪公式 💃

```
自然动作 = 不看镜头 + 微动态 + 松弛体态
```

#### 5.1 不看镜头

**视线方向**：

| 方向 | 效果 | 场景 |
|------|------|------|
| **侧看** | 自然、随性 | 街拍、风景 |
| **回望** | 故事感 | 走路、回头 |
| **远眺** | 意境感 | 山顶、海边 |
| **低头** | 温柔感 | 花丛、阅读 |
| **闭眼** | 享受感 | 阳光、微风 |

#### 5.2 微动态

**自然动作清单**：

| 动作 | 效果 | 场景 |
|------|------|------|
| **走路** | 自然、随性 | 街道、小路 |
| **回头** | 故事感、惊喜 | 任何场景 |
| **抬手** | 活泼、舒展 | 户外、自然 |
| **扶帽** | 休闲、时尚 | 街拍、旅行 |
| **摸头发** | 温柔、自然 | 任何场景 |
| **眺望** | 意境、思考 | 山顶、海边 |
| **侧身倚靠** | 放松、随性 | 墙壁、栏杆 |
| **低头浅笑** | 温柔、内敛 | 花丛、室内 |
| **甩头发** | 动感、活力 | 海边、风大 |
| **伸展双臂** | 自由、舒展 | 山顶、开阔地 |

#### 5.3 松弛体态

**核心原则**：
- 不僵硬，自然放松
- 有曲线，不笔直
- 有互动，不孤立

**人融景，不硬打卡**：
- 与环境互动
- 做自然动作
- 捕捉瞬间

---

### 公式6：高级感减法公式 ✨

```
高级感 = 去掉杂物 + 缩小主体 + 大面积留白
```

#### 6.1 去掉杂物

**避开**：
- ❌ 垃圾桶
- ❌ 路人（等或避开）
- ❌ 电线、杆子
- ❌ 杂乱背景
- ❌ 反光、眩光

#### 6.2 缩小主体

**技巧**：
- 人占画面1/3以下
- 突出环境氛围
- 人景融合

#### 6.3 大面积留白

**留白元素**：

| 场景 | 留白 | 效果 |
|------|------|------|
| **天空** | 蓝天、云彩 | 开阔感 |
| **水面** | 湖面、海面 | 宁静感 |
| **墙面** | 纯色墙 | 简洁感 |
| **地面** | 草地、沙滩 | 自然感 |

**留白比例**：
- 留白占画面50%以上
- 主体在黄金分割点
- 画面干净立刻高级

## 3个爆款场景专属公式

---

### 场景1：古镇/古城 🏮

```
复古感 = 古风浅穿搭 + 门框框架构图 + 慢动作抓拍 + 暖色调
```

#### 穿搭建议

| 元素 | 推荐 | 避免 |
|------|------|------|
| **颜色** | 米色、浅蓝、淡粉 | 荧光色、大logo |
| **风格** | 汉服、旗袍、文艺风 | 运动装、西装 |
| **配饰** | 油纸伞、扇子、发簪 | 现代包、墨镜 |

#### 构图要点

- 用门框、窗框做框架
- 灯笼、回廊做引导线
- 石板路增加纵深感

#### 拍摄时间

- 上午9-10点（人少光线好）
- 傍晚5-6点（暖光氛围）
- 夜晚（灯笼亮起，夜景美）

#### 动作建议

- 撑伞走过石桥
- 倚门回望
- 坐在台阶上
- 轻抚墙壁

---

### 场景2：海边/湖边 🌊

```
松弛感 = 大留白天空/海面 + 逆光 + 侧身走动 + 蓝白配色
```

#### 穿搭建议

| 元素 | 推荐 | 避免 |
|------|------|------|
| **颜色** | 白色、蓝色、米色 | 黑色、深色 |
| **风格** | 飘逸长裙、宽松衬衫 | 紧身、厚重 |
| **配饰** | 草帽、草编包 | 皮鞋、皮包 |

#### 构图要点

- 海平面占画面1/3
- 大面积天空留白
- 人在画面一侧

#### 拍摄时间

- 日出（5-7点，金色海面）
- 日落前（4-6点，逆光氛围）
- 蓝调时刻（日落后，紫蓝色）

#### 动作建议

- 侧身走路（抓拍）
- 甩动裙摆
- 抬手遮阳
- 赤脚漫步
- 回望镜头

---

### 场景3：山林/山野 🌲

```
治愈感 = 绿植当前景 + 林间柔光 + 远景群山 + 低饱和色调
```

#### 穿搭建议

| 元素 | 推荐 | 避免 |
|------|------|------|
| **颜色** | 米色、卡其、墨绿 | 亮红、亮黄 |

#### 构图要点

- 树叶做前景虚化

- 上午（林间光线柔和）

- 抬头看树
- 伸手触摸树叶

## 单人照 vs 多人照模式

---

### 单人照模式 👤

**核心**：突出个人，人景融合

#### 构图

- 人占画面1/3
- 用三分法
- 与环境互动

#### 动作

- 自然随性
- 不看镜头
- 有故事感

#### 光线

- 侧光显立体
- 逆光出氛围
- 避免顶光

---

### 多人照模式 👥

**核心**：互动感，层次感

#### 站位

| 站位 | 效果 | 人数 |
|------|------|------|
| **前后错落** | 层次感 | 2-3人 |
| **高低错落** | 立体感 | 3-5人 |
| **分散站位** | 自然感 | 3-6人 |
| **对称站位** | 仪式感 | 2-4人 |

#### 互动

- 对视
- 搭肩
- 牵手
- 共同看远方
- 做同一动作

#### 避免

- ❌ 排排站
- ❌ 动作一致
- ❌ 都正面看镜头
- ❌ 间距相等

## 示例演示

### 示例：古镇拍照

**用户输入**：
> "我在乌镇，下午3点，单人，想要复古氛围感"

**Skill输出方案**：

**场景分析**：
- 乌镇：江南水乡，白墙黑瓦，小桥流水
- 下午3点：光线偏硬，建议找阴凉处或等到4-5点
- 单人：突出人与景的融合
- 复古氛围：暖色调，框架构图

**拍摄方案**：

**时间调整**：
- 建议4:30-5:30拍摄（光线柔和，暖色调）

**推荐点位**：
1. 石桥（框架构图）
2. 回廊（引导线）
3. 水边（倒影层次）
4. 古门（框架取景）

**穿搭建议**：
- 白色或米色长裙
- 油纸伞道具
- 低马尾或盘发

**动作指导**：
- 撑伞走过石桥（抓拍）
- 倚门回望
- 坐在河边石阶
