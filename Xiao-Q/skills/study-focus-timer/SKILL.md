# Study Focus Timer — 学习专注计时

Use when the user mentions 学习计时、专注学习、番茄钟学习、study timer、focus study、pomodoro study, or asks to track study time. NOT for 工作专注、冥想、睡眠.

## 描述

基于番茄工作法帮助用户规划学习时段，制定科学的专注-休息节奏，记录学习时长并分析专注效率，解决学习容易分心、时间规划混乱的问题。

## 重要限制（请提前告知用户）

- **无法实时计时**：不能后台运行倒计时，需要用户配合手机/手表自行计时
- **无法强制专注**：只提供规划和建议，无法屏蔽手机通知或锁屏
- **记录需手动更新**：学习时长需要用户告知，无法自动检测
- **不替代自律**：工具只是辅助，关键在于用户的执行力

## 快速开始

```
用户：我要开始学习了
→ 了解学习科目和时长目标，制定番茄钟计划

用户：学习2小时数学
→ 规划4个25分钟番茄钟+休息，提供专注建议

用户：我学完了，记录一下
→ 记录学习时长，给出效率评估和鼓励

用户：今天学了多久？
→ 汇总今日各科学习时长和番茄钟完成情况

用户：我老是走神怎么办？
→ 分析分心原因，给出针对性的专注技巧
```

## 能力

- 根据学习目标规划番茄钟安排
- 提供科学的专注-休息节奏建议
- 记录并汇总学习时长
- 分析学习效率和专注度
- 提供防分心技巧
- 生成学习计划模板

## 执行步骤

### Step 1: 了解学习需求
1. 确认学习科目和目标时长
2. 了解用户的专注能力（能连续专注多久）
3. 确定学习环境（在家/图书馆/咖啡厅）

### Step 2: 制定计划
1. 根据总时长分配番茄钟数量
2. 设置合理的专注和休息时间
3. 安排大休息节点
4. 提供专注建议

### Step 3: 记录反馈
1. 接收用户的完成汇报
2. 记录实际学习时长
3. 计算完成率和效率
4. 给出改进建议

## 输出格式

```
📚 学习计划 | [科目]
━━━━━━━━━━━━━━━━━━━━
目标时长：[X]小时
番茄钟数：[X]个（每个25分钟）
预计完成时间：[时间]

## 学习安排

🍅 第1个番茄钟 (25min) — [具体任务]
☕ 短休息 (5min) — 起身活动、喝水
🍅 第2个番茄钟 (25min) — [具体任务]
☕ 短休息 (5min)
🍅 第3个番茄钟 (25min) — [具体任务]
☕ 短休息 (5min)
🍅 第4个番茄钟 (25min) — [具体任务]
🌴 长休息 (15-20min) — 散步、吃点东西

## 专注建议
- 手机静音并放到视线之外
- 准备好所有学习材料再开始
- 中途想到别的事记在纸上，番茄钟结束后再处理
- 如果走神了，深呼吸3次拉回注意力

---
⏱️ 准备好了吗？设好计时器，开始！
```

## 输出原则

1. **计划具体可执行**：每个番茄钟都有明确的任务内容
2. **节奏科学合理**：遵循25+5或50+10的专注-休息比
3. **鼓励正向反馈**：完成后给予积极反馈，建立成就感
4. **灵活调整**：根据用户反馈调整难度和时长
5. **关注可持续性**：不追求单次极限，注重长期习惯培养

## 错误处理

| 异常场景 | 提示语 |
|----------|--------|
| 目标时长不明 | "⏰ 你今天打算学多久？我来帮你安排番茄钟" |
| 目标过长（超5小时） | "💡 建议分上下午安排，中间留1小时以上休息，避免疲劳" |
| 频繁走神 | "🧠 频繁走神可能是任务太难或太无聊，试试拆解成更小的任务" |
| 无法坚持 | "🌱 可以从短番茄钟（15min）开始，逐步延长到25min" |
| 科目不明 | "📖 今天学什么科目？我好帮你分配每个番茄钟的具体任务" |

## 常见问题（FAQ）

**Q: 一定要25分钟一个番茄钟吗？**
A: 不一定。25分钟是经典设置，但你可以根据自己的专注力调整，15-50分钟都可以。

**Q: 休息时间可以刷手机吗？**
A: 短休息建议做物理活动（喝水、伸展），避免刷手机——容易超时且大脑得不到真正休息。

**Q: 学到一半被打断怎么办？**
A: 被打断的番茄钟不算完成。处理完中断后重新开始一个新番茄钟。

**Q: 不同科目怎么安排？**
A: 建议难的科目放在精力最好的时段，不同科目之间用大休息分隔。

**Q: 一天多少个番茄钟合适？**
A: 学生建议8-12个（4-6小时有效学习），质量比数量更重要。

**Q: 怎么记录学习时长？**
A: 每完成一组番茄钟后告诉我，我会帮你累计。

## 最佳实践

1. **固定学习时段**：每天在相同时间开始学习，形成生物钟
2. **先难后易**：精力充沛时攻克难题，疲劳时做复习
3. **提前准备**：开始前把所有材料准备好，避免中途找东西
4. **一次一事**：每个番茄钟只做一件事，不多任务切换
5. **及时复盘**：每天结束时回顾完成情况，调整明天计划

## 不适用场景

| 场景 | 原因 | 替代方案 |
|------|------|----------|
| 需要实时计时提醒 | 无法后台计时 | 使用手机番茄钟APP |
| 强制屏蔽手机 | 无法控制设备 | 使用专注锁屏APP |
| 长期学习数据分析 | 对话记录不持久 | 使用学习记录APP |
| 工作任务管理 | 本技能面向学习 | 使用项目管理工具 |

### 常见误用

- **误用 1**：只规划不执行 → 计划再好不开始等于零
- **误用 2**：一次规划太多番茄钟 → 先完成4个再追加，避免压力过大
- **误用 3**：休息时间不休息 → 休息是为了下一轮更专注，不是浪费时间

## 安全与隐私

- 学习记录仅在当前对话中临时保存
- 不存储用户的学习习惯数据
- 建议用户同步到其他工具做长期追踪
