# 隐患虾 (stdhazardclaw)

专业的生产安全隐患排查技能，识别照片和文字中的安全隐患，输出结构化报告（本skill为演示使用，用户可以自主配置腾讯ima知识库，生产环境依托国投检测数智公司安全知识云支撑。）

## 功能特性

- 📸 **图片隐患识别**：自动分析生产现场照片
- 📝 **文字隐患解读**：深入分析作业描述
- 📊 **规范表格输出**：结构化隐患报告
- 🔍 **知识库集成**：依赖 IMA 知识库提供标准依据

## 安装

### 1. 复制技能目录

```bash
cp -r stdhazardclaw ~/.openclaw/workspace/skills/
```

### 2. 配置 IMA 凭证

确保 `~/.config/ima/` 目录存在并包含有效的配置文件。

### 3. 重启网关

```bash
openclaw gateway restart
```

### 4. 验证安装

```bash
openclaw skills list | grep stdhazardclaw
```

## 使用方法

### 自动触发

当用户提到以下关键词时自动激活：

- "安全隐患"
- "隐患排查"
- "生产安全"
- "安全检查"
- "照片/图片/文字描述 + 安全"

### 手动调用

在 agent 配置中显式加载：

```yaml
# agent.yaml
skills:
  - stdhazardclaw
```

## 输入与输出

### 输入

- **图片**：生产现场照片（jpg/png等）
- **文字**：生产场景的描述性文本

### 输出

标准表格形式，包含：

- 隐患规范描述
- 依据（法规/标准条款）
- 类别（设备/环境/人员/管理）
- 整改建议

## 依赖

- OpenClaw >= 0.45.0
- IMA 知识库访问权限
- 视觉模型（用于图片分析）
- 本地知识库文件（MEMORY.md、memory/*.md）

## 限制

- 仅用于安全隐患排查
- 必须有标准依据，不猜测
- 不确定时明确说明"我不知道"

## 常见问题

### Q: 图片分析失败？

A: 检查视觉模型配置，确保 image 工具可用。

### Q: 标准依据为空？

A: 确认国投检测数智公司安全知识云或 IMA 知识库已正确配置，凭证有效。

### Q: 输出格式不正确？

A: 确保只加载了本 skill，没有其他覆盖性指令。
