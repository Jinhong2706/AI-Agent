# 石家庄中考志愿填报助手

## 项目身份

本 Skill 提供石家庄中考志愿填报助手项目的完整上下文——包括概率引擎、数据资产、项目结构、技术架构和升级路线图。

项目不是对话机器人，而是完整的**概率计算 + 志愿决策系统**。使命：帮助石家庄家长把"我能上哪个学校"这个模糊焦虑，转化为可验证的录取概率。

**项目根目录**: 用户本地 `sjz-zhongkao/` 目录（首次使用需克隆或初始化项目结构）

## 核心算法（v2.1 逐校σ模型 + 反向查询 + 组合优化）

### 一步到位公式

```
录取概率 = 1 - NormCDF( (学生分数 - 学校统招线) / σ_combined )

σ_combined = √(σ_估分² + σ_school²)
σ_估分 = 12  (估分误差标准差，固定)
σ_school = 该校三年统招线年际波动标准差（逐校独立，范围 3.0 ~ 71.5）
```

**v2.0 核心升级**：旧版所有学校统一 σ=15.62，新版每所学校独立 σ。
- 极度稳定校（二中/一中）：σ_combined ≈ 12.4~12.7
- 中等波动校（二十四中/九中）：σ_combined ≈ 13~14
- 剧烈波动校（河北联邦外国语）：σ_combined ≈ 72.5

### NormCDF 实现

使用 Abramowitz & Stegun 多项式逼近，纯数学实现，零外部依赖：

```python
import math

def _erf(x):
    a1, a2, a3, a4, a5 = 0.254829592, -0.284496736, 1.421413741, -1.453152027, 1.061405429
    p = 0.3275911
    sign = 1 if x >= 0 else -1
    x = abs(x)
    t = 1.0 / (1.0 + p * x)
    y = 1.0 - (((((a5 * t + a4) * t) + a3) * t + a2) * t + a1) * t * math.exp(-x * x)
    return sign * y

def norm_cdf(x):
    return 0.5 * (1.0 + _erf(x / math.sqrt(2.0)))
```

JavaScript 版本已在 webapp/index.html 中完整实现。核心计算：

```javascript
// z = (学生分数 - 学校统招线) / σ_combined
var zScore = (studentScore - school.tongzhao) / school.combinedStd;
var prob = 1 - normCDF(zScore);
```

### v2.1 新增：反向查询引擎（normSInv）

使用有理逼近实现逆正态CDF（probit），精度 ±0.0004：

```javascript
function normSInv(p) {
  // 有理逼近 inverse error function
  var a = [0.886226899, -1.645349621, 0.914624893, -0.140543331];
  var b = [-2.118377725, 1.442710462, -0.329097515, 0.012229801];
  // ...有理函数计算
  return Math.SQRT2 * erfinv(2*p - 1);
}
```

反向查询公式：

```
所需分数 = 统招线 + σ_combined × Φ⁻¹(1 - 期望概率)
```

### 风险分级（v2.1 更新）

| 概率区间 | 风险等级 | 颜色 | 建议 |
|---------|---------|------|------|
| >= 80% | 保 | 绿 | 保底首选 |
| 50%-80% | 稳 | 黄 | 主力志愿 |
| < 50% | 冲 | 红 | 不能作为保底志愿 |

### 分配生逻辑

分配生录取概率 = 三步判断：
1. 分数 >= 统招线 - 50 ? 是→继续 / 否→0
2. 校内排名 <= 配额 ? 是→0.92 / 否→如果 <= 配额×1.3 → 线性递减0.92→0.03 / 否→0.03
3. P_分配生 = P_资格 × P_配额内

