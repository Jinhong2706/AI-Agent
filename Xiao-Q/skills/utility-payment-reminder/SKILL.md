# Utility Payment Reminder — 水电煤缴费提醒

Use when the user mentions 缴费提醒、水电煤、忘记缴费、utility payment、bill reminder、pay bills, or asks to remind about utility payments. NOT for 直接缴费、账单查询、开户.

## 描述

帮助用户记录水电燃气等生活缴费的户号、缴费日期和缴费方式，提供到期提醒和缴费清单，解决经常忘记缴费导致被停水停电、产生滞纳金的问题。

## 重要限制（请提前告知用户）

- **无法自动提醒**：不能定时推送通知，需要用户主动来查询
- **无法直接缴费**：只做记录和提醒，缴费需在对应平台自行操作
- **需要手动录入**：户号和缴费日等信息需用户提供
- **记录不跨对话**：信息在对话结束后不保存

## 快速开始

```
用户：帮我记录水电缴费信息
→ 请告诉我各项户号和缴费截止日

用户：电费户号123456，每月15号前缴
→ 记录信息，计算下次缴费日期

用户：这个月该缴什么费了？
→ 列出本月待缴费项目和截止日

用户：水费怎么交最方便？
→ 推荐缴费渠道

用户：忘了缴费会怎样？
→ 说明欠费后果和补救方法
```

## 能力

- 记录各类生活缴费户号和截止日
- 计算各项费用的下次缴费时间
- 按月生成缴费清单
- 推荐缴费渠道和方式
- 提醒即将到期的项目
- 说明欠费后果和处理方法

## 执行步骤

### Step 1: 录入信息
1. 记录各项费用的户号
2. 记录缴费周期（每月/每两月）
3. 记录缴费截止日
4. 了解当前缴费方式

### Step 2: 生成提醒
1. 计算各项距截止日的天数
2. 标注即将到期项目
3. 列出缴费渠道
4. 建议自动扣费设置

### Step 3: 优化建议
1. 推荐便捷缴费方式
2. 建议设置自动扣费
3. 提供欠费处理方法

## 输出原则

1. **到期紧急优先**：距截止日近的排在最前面
2. **操作路径明确**：告诉用户在哪里缴、怎么操作
3. **自动化建议**：推荐设置代扣，从根本上解决忘缴问题
4. **后果说明**：对即将到期的强调欠费后果
5. **简洁清晰**：一目了然看出哪些需要处理

## 常见问题（FAQ）

**Q: 忘记缴费会停水停电吗？**
A: 通常欠费后有3-7天宽限期才会停供。但建议不要拖，可能产生滞纳金。

**Q: 怎么设置自动代扣？**
A: 支付宝/微信→生活缴费→缴完后选择"开通自动缴费"。或去银行APP绑定代扣。

**Q: 户号在哪里找？**
A: 缴费通知单上、电表/水表旁边的标签上、或上次缴费的凭证/APP记录里。

**Q: 可以帮我查余额吗？**
A: 不能，查询余额需要在对应缴费平台操作。支付宝可以直接查。

**Q: 物业费也能管吗？**
A: 可以，物业费、停车费、网费等定期费用都可以记录进来。

**Q: 搬家了户号要换吗？**
A: 是的，搬家后户号会变。入住新房后确认新户号告诉我更新。

## 最佳实践

1. **统一管理**：把所有定期缴费项目都记录在一起
2. **开通代扣**：核心费用（水电燃气）开通自动代扣最省心
3. **设日历提醒**：在手机日历设置每月缴费提醒
4. **月初检查**：每月1号花1分钟看看本月有什么要缴
5. **保留凭证**：截图保存缴费成功记录

## 不适用场景

| 场景 | 原因 | 替代方案 |
|------|------|----------|
| 直接缴费 | 无法操作缴费系统 | 使用支付宝/微信缴费 |
| 查询用量/余额 | 无法连接系统 | 在缴费平台查询 |
| 开户/销户 | 涉及线下办理 | 去营业厅办理 |
| 费用争议 | 需要专业处理 | 联系供应商客服 |

### 常见误用

- **误用 1**：只记录不查看 → 记录的目的是提醒，要定期来问我
- **误用 2**：不设自动代扣 → 开通代扣是解决忘缴的最佳方案
- **误用 3**：搬家不更新户号 → 新地址的户号要及时更新

## 安全与隐私

- 户号信息仅在当前对话中临时保存
- 建议户号等信息同步到安全的备忘录保存
- 不连接任何缴费系统
- 不存储用户的支付信息
