# Reverse Engineering Framework

本 Skill 在内部遵循以下逆向分析顺序：

1. 布局结构
   - 页面是单列、双列、卡片流、杂志式、海报式还是网格式
   - 主容器宽度、列宽、可视边界、首屏节奏

2. 视觉系统
   - 色板、层级、字体、阴影、圆角、边框、质感、渐变、模糊

3. 组件系统
   - 导航、卡片、按钮、标签、列表、媒体模块、数据模块、表单模块

4. 交互系统
   - 可点击对象、状态变化、可能的 hover / active / focus

5. 设计意图
   - 页面用途、面向人群、品牌气质、内容表达方式、信息密度策略

6. 实现策略
   - HTML 语义化、CSS 变量、响应式、少量必要 JS

7. 风格抽象
   - 将页面的局部视觉现象抽象成可复用规范与设计令牌，供 `frontend-skill` / `frontend-slides` 继续扩页使用
