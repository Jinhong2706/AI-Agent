# Style Lock Principles

高保真不仅是“像”，更是“后续继续做页面也不会变味”。

本 Skill 的风格锁定遵循三层原则：

## 1. Must Keep（必须保留）
- 版式密度与垂直节奏
- 色彩关系与主辅层级
- 标题与正文层级
- 组件语言（按钮、卡片、标签、导航）
- 圆角与阴影语义
- 图像裁切与蒙版方式
- 页面整体气质

## 2. May Adapt（允许适配）
- 内容长度
- 模块数量
- 数据与文案
- 非关键装饰素材
- 不破坏风格前提下的窄屏重排

## 3. Must Not Do（必须禁止）
- 擅自引入新主色
- 擅自切换为另一套字体体系
- 用默认模板覆盖原图节奏
- 把克制风格做成营销页
- 把杂志风做成 SaaS 控制台
- 把原图的简洁质感换成廉价重阴影或过度玻璃拟态

只有同时写清这三层，后续 Skill 才不容易跑偏。
