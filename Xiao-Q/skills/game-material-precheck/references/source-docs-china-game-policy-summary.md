自2000年以来,国务院、国家市场监督管理总局、国家新闻出版署等多部门都陆续印发了支持、规范游戏行业的发展政策,内容涉及游戏发展技术路线、游戏发展规范、未成年人游戏监管等多个方面。

2.1 版号审批与出版管理
《网络游戏管理办法》(已废止)

主要内容:规范网络游戏的出版、运营等活动
状态:2019年废止
