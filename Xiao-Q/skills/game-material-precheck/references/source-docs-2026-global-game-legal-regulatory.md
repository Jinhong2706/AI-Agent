一、中国:AI监管新规来了,7月15日起施行
1.1 版号审批常态化,但审核更精细
2026年3月,国家新闻出版署审批通过133款游戏(国产130款+进口3款),Q1累计发放约417个版号,同比增长28%。

| 变化 | 数据对比 |

核心信号:监管从"形式合规"转向"实质有效",第三方穿透性测试报告成为刚需。
1.2 五部门联合出手:AI拟人化互动服务纳入监管
4月10日,国家网信办、发改委、工信部、公安部、市场监管总局五部门联合发布《人工智能拟人化互动服务管理暂行办法》,2026年7月15日起施行。
游戏企业最需要关注的三点:
严禁向未成年人提供虚拟伴侣等服务
14岁以下使用需监护人同意
不得实施AI情感操纵、诱导过度沉迷

日本扭蛋机制不被归类为赌博(区别在于获得的是数字资产而非现金奖励),但监管持续严格:
| 要求 | 内容 |

四、韩国:外国企业必须指定本地代表
GIPA修订案正式施行
2024年9月韩国国会通过《游戏产业促进法》修订案,2025年4月23日起正式施行:
在韩国提供游戏服务的外国公司必须指定韩国国内代表
须在提供服务前取得运营许可证
须配合韩国监管机构的执法行动
适用范围:所有在韩国运营的外国游戏企业。
五、欧盟:消费者保护连出重拳
5.1 虚拟货币:七项核心原则
2025年3月,欧盟消费者保护合作(CPC)网络发布七项核心原则,规范游戏内虚拟货币:
| 原则 | 要点 |
| 1 | 明确披露虚拟货币与真实货币的兑换比例 |
| 2 | 禁止强制购买额外虚拟货币才能使用已购货币 |
| 3 | 提供退款和退出选项 |
| 4 | 保护儿童免受虚拟货币营销影响 |
| 5 | 披露开箱概率和物品稀缺性 |
| 6 | 限制诱导性营销实践 |
| 7 | 提供消费者争议解决机制 |

5.2 年龄验证应用正式发布
4月15日,欧盟委员会发布免费、开源、跨平台的年龄验证应用:
支持护照/身份证验证
完全匿名,保护隐私
完全开放源代码,其他国家可采用
