# 女性角色局部露肤审核细化规则

导入时间：2026-05-08
示例图：`assets/female-exposure-redline-example.png`

## 规则来源

本规则来自用户对素材审核口径的明确补充：在女性角色胸部位置，两条红线中间的位置禁止露肤。后续审核女性角色宣发素材时，应将该区域作为重点检查项。

## 适用范围

适用于以下素材：

- 女性角色、女性真人/CG角色、类女性体型角色。
- 游戏宣发图、角色立绘、广告图、商店图、社媒图、官网图、版本活动图。
- 中国大陆投放素材默认从严；海外素材按目标市场叠加审核。

## 核心判定

以 `assets/female-exposure-redline-example.png` 为基准，女性胸部位置两条红线中间的区域定义为「胸部敏感露肤红线区」。

该区域原则上禁止出现：

- 裸露皮肤或明显肤色区域。
- 低胸、深V导致的中间沟壑外露。
- 胸部下缘、内侧轮廓、明显曲线被服装开口暴露。
- 通过裁切、红框、箭头、光影、文案等方式强调该区域。

## 风险等级

- 若红线区内出现明显连续肤色裸露：至少判定为 `P1 高风险`，不建议直接上线。
- 若红线区附近有少量皮肤但未进入红线区，且整体非性感化：可按 `P2 中风险` 或 `P3 低风险` 结合渠道判断。
- 若素材用于广告投放、应用商店、开屏、信息流、官网首屏等强曝光渠道：从严处理，红线区露肤直接按 `P1 高风险`。
- 若叠加低龄化、学生感、幼态脸、可爱化角色：风险上调一级。

## 审核步骤

1. 先定位女性角色胸部位置。
2. 参考示例图两条红线之间的相对区域，判断是否存在裸露皮肤。
3. 判断露肤是否由服装开口、低胸、深V、裁切、光影或姿态造成。
4. 判断画面是否通过构图、标注、文案强调该区域。
5. 结合投放渠道判断最终风险等级。

## 修改建议

- 增加内搭、胸前护甲、拉链、围巾、战术装备遮挡红线区。
- 拉远镜头或调整构图，让视觉焦点转向脸部、武器装备、动作、场景。
- 避免红框、箭头、线条、文案等任何形式强调胸部区域。
- 对强曝光渠道，优先使用上半身非低领版本或全身战术姿态版本。
- 如必须保留角色服装设定，应提供更保守版本用于国内宣发。

## 报告写法

推荐表述：

- 「女性胸部敏感露肤红线区存在明显肤色裸露，按本技能女性角色局部露肤细化规则，判定为 P1 高风险，不建议直接上线。」
- 「该素材在胸部红线区附近存在露肤趋势，但未完全进入禁止区域，建议按 P2 中风险处理，调整构图/增加遮挡后复审。」
- 「当前素材未见胸部红线区露肤，女性角色露肤项暂未发现明显问题。」
