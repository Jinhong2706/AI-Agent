2024-2025年,游戏监管已从传统的内容审查扩展到数据保护、未成年人保护、反洗钱、税收合规等多个维度。

三、主要海外市场监管政策
3.1 欧盟
GDPR与《数字服务法》(DSA)
GDPR累计处罚超56.5亿欧元
数据跨境传输为执法重点,TikTok因违规被罚5.3亿欧元
DSA要求游戏内置实时内容过滤器,大型平台需定期提交合规报告
成员国执法差异
西班牙:高频小额处罚
爱尔兰:以高额罚单著称
法国:强调法语优先
德国:严控暴力内容
3.2 美国
联邦层面
FTC强化游戏内购监管- 禁止向16岁以下儿童售战利品箱,要求披露奖励概率与直接购买选项
COPPA- 对13岁以下儿童数据收集设严格限制
各州政策分化
加州CCPA:赋予消费者广泛数据权利
伊利诺伊州BIPA:严控生物识别信息使用
纽约州:关注暴力内容监管
3.3 印尼
2026年1月起强制实施IGRS分级制度:
按3+、7+、13+、15+、18+划分年龄等级
含禁售内容直接拒绝分级
所有游戏及UGC需完成注册、自评、审核流程
获证后需在全渠道公示分级信息
计划接入eKYC强化年龄验证
3.4 韩国
2025年10月起实施海外游戏公司代理人制度:
符合全球营收超52亿人民币或韩区日均下载超1000次等条件的企业需指定本地代理人
代理人需公示道具概率、配合监管调查
未合规每次最高罚2000万韩元
概率虚假披露可处三倍惩罚性赔偿
3.5 越南
游戏分类与准入:
按交互模式将游戏分G1-G4四类
外资需设持股≤49%的合资公司
G1类需申请许可证(审批30天)
强制要求境内设服务器
对18岁以下玩家设每日3小时游戏限制
2025年起App Store上架需提供许可证编号
3.6 泰国
正制定首部《游戏产业法》
游戏分发需经电影和视频委员会批准
已撤销扑克禁令,计划合法化赌场
3.7 新加坡
以《广播法》和《个人数据保护法》为核心
监管相对宽松,重点限制色情、暴力等敏感内容
未设置强制分级或代理人等严苛准入要求
3.8 巴西
2025年起实施全面监管:
许可证费用500万美元(有效期5年)
游戏收入税提至18%
强制生物识别验证(每7天1次)、地理定位监测
需遵守反洗钱条例,违规最高罚20亿雷亚尔
2026年3月起禁止向18岁以下售战利品箱

企业所得税30%,叠加联邦博彩税1%及州级IEPS税7%
申请许可证需6-12个月,最低注册资本200万比索
2025年提案要求纯数字运营商增资至5000万比索、境内设镜像服务器

全面禁止赌博及相关外资准入
游戏需经3-6个月宗教审查
2023年实施PDPL数据保护法,违规最高罚500万沙特里亚尔
Roblox因合规要求屏蔽30万款游戏并禁用聊天功能

五、结语
全球游戏监管正在经历深刻变革,合规已成为游戏出海的入场券,而非可选项。
不合规连出海的机会都没有。中小型游戏企业应借助专业化公共服务平台的赋能,以合规为基石,以创新为动力,在全球市场中实现稳健发展。
