# 本地知识库原文区

本文件用于粘贴、沉淀和更新「游戏素材预审」使用的原始/半原始知识库内容。

## 使用方式

- 将乐享知识库中的文档正文直接粘贴到本文件。
- 每篇文档建议使用二级标题分隔，例如：`## 网络游戏公司全流程合规指南`。
- 每次更新时，在对应文档标题下替换为最新内容，或新增「更新日期」小节。
- 如果内容很长，可以继续拆分为多个 Markdown 文件，例如：
  - `source-docs-game-compliance.md`
  - `source-docs-game-policy.md`
  - `source-docs-game-legal.md`
  - `source-docs-game-pr.md`
  - `source-docs-global-cultural-risk.md`
