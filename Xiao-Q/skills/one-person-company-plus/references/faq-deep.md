## 内容创作 · 进阶问题

**Q：我同时运营多个平台（公众号+小红书+微博），内容要全部不同吗？**

A：不需要全部重写，但要做平台适配。一份核心内容，分发时做以下调整：

| 平台 | 调整重点 |
|-----|---------|
| 公众号 | 长文深度，加目录，可引导关注/订阅 |
| 小红书 | 短句+emoji，图文分离，关键词带#话题 |
| 微博 | 140字核心观点+配图，结尾设互动问题 |
| Newsletter | 个人化语气，独家内容，引导点击率 |

告诉我"把这篇文章改成小红书版本"，我会自动处理所有差异。

---

**Q：我想建立自己的内容风格，怎么让 AI 记住我的文风？**

A：在每次对话开始时，发送你的"风格卡片"：

```
我的内容风格：
- 语气：轻松幽默，偶尔自嘲
- 受众：25-35岁独立创业者
- 惯用句式：短句 + 转折（"但是…"）
- 禁用词：赋能、链接、底层逻辑
- 参考作者：[你喜欢的作者名]
```

这份风格卡片越具体，后续每次输出越接近你的真实风格。

## 商业运营 · 进阶问题

**Q：竞品分析需要我提供什么信息？**

A：最少提供：
1. **竞品名称或链接**（我可以基于公开信息分析）
2. **你想了解的维度**（定价/产品/内容/渠道/用户口碑）
3. **你自己的产品情况**（用于对比分析）

注意：我只分析公开可获取的信息，不提供任何私密竞品数据。

---

**Q：我的产品刚上线没有真实评价，如何在文案中建立信任感？**

A：没有真实评价时，用以下替代方案：
1. **制作过程透明化**：展示原材料、制作工艺、质检步骤
2. **创始人故事**：为什么做这个产品，解决了你自己什么问题
3. **无条件退款承诺**：降低购买风险，比100条好评更有效
4. **小批量试用计划**：邀请真实用户免费试用换取真实反馈

我可以帮你写以上任何一种内容，告诉我产品信息就行。

## 客户服务 · 进阶问题

**Q：有客户在公开平台（小红书/微博）给差评怎么办？**

A：公开差评的处理分两步：

**第一步：公开回复（在评论区，所有人看得见）**
```
目标：展示你的专业和诚意，给旁观者看
原则：不辩解、不攻击、承认问题、说明处理方式
示例："非常抱歉您有这样的体验，已私信联系您，
       我们会认真处理，感谢您的反馈 🙏"
```

**第二步：私信跟进（实质性解决问题）**
按投诉处理标准话术执行，解决后请求对方更新评价（注意：是请求，不是要求）。

---

**Q：客户一直追问货物发出多久了，我很烦，怎么设置自动回复减少这类询问？**

A：设置"订单状态自动通知"可以从根源减少催货询问。
以下是可以粘贴到微店/有赞等平台自动回复的话术：

```
【发货确认消息模板】
您好！您的订单已发出 📦
运单号：[单号]
物流：[快递名]
预计到达：[时间]
查询方式：菜鸟/快递100 搜索单号

如有问题，随时私信我 😊
```

这条消息在发货时主动发送，可以减少 70% 的催货询问。

**Q：我用 Notion/飞书做任务管理，输出格式能兼容吗？**

A：可以。告诉我你用哪个工具，我会调整输出格式：

- **Notion**：输出 Markdown 表格，可直接粘贴
- **飞书文档**：输出带标题层级的 Markdown
- **Excel/WPS**：输出 CSV 格式数据表
- **滴答清单/Things**：输出纯文本任务列表，带截止日期格式

默认输出 Markdown 格式，兼容性最广。

## 安全与隐私

**Q：我能把客户的真实信息（姓名、联系方式、购买记录）发给 AI 处理吗？**

A：**强烈建议不要这样做。** 请遵循以下原则：
- 用"客户A"、"用户B"替代真实姓名
- 隐去手机号、邮箱等联系方式
- 金额数据可以保留（如需分析）
- 地址信息请替换成省市级别即可

我处理的是业务逻辑，不需要真实的个人识别信息。
