## 关键动作

### 上线节奏
- 多平台分别时间
- 是否独占 / 限时独占 / 真独占
- 上线节点是否绑定特殊事件（年度奖项 / 周年庆 / 平台直播节目）

### 平台资源
- State of Play / Direct / Showcase 露出
- 商店首页 Banner / 推荐位
- 平台会员包（PS Plus 限免组合包 / GP Day1 等）
- 限定硬件（捆绑机 / 限定手柄）
- Pro 版增强 / VR 适配 / Cloud Gaming
- 实体版 / 收藏版

### KOL / 主播 / 社区
- 头部主播预热 / Twitch Drops（海外）
- 区域 KOL 矩阵（B 站抖音 / Vtuber / Watch Party 等）
- 玩家共创 / 二创扩散
- 社区运营策略

### 跨界 / IP 联动 / 文化叙事
- IP 联动 / 跨界品牌合作
- 文化机构联动（国家博物馆 / 文旅部门）
- 主题乐园 / 实体化（环球影城等）

### 上线后的长尾运营
- 大版本免费追加
- 付费 DLC
- 奖项二次传播（TGA / DICE）
- 后置 Game Pass / 二次冲榜

## 写作时的提醒

1. **关键动作**这一节最重要——必须具体到"实际做了什么"，而不是只写做了什么类型
2. **数据点**必须每行都有可点开来源
3. **对发行方的参考意义**是案例的真正价值——不要只复盘案例本身
4. 如果有内部一手资料 / 不可外发的报告，单独存放在 `shared/internal_reports/`，并标注"⚠️ 禁止外发"
