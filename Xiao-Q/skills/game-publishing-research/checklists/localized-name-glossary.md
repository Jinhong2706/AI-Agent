## 使用规则

### 第一次出现
当一个游戏 / 厂商 / 平台首次在文档中出现时，可用如下格式：

```
绝地潜兵 2（Helldivers 2）
索尼互动娱乐（SIE）
任天堂直面会（Nintendo Direct）
```

### 后续引用
首次提及后，所有后续引用一律用中文名，不再带英文括号注释。

### 例外情况
以下情况可保留英文：
- 简短英文已被中文用户广泛接受（如 PS5 / Xbox / Switch / Steam / TGA / SGF）
- 没有公认中文译名（如 Pocketpair / FromSoft / Embark）
- 玩家社区原生英文梗（如 "super popular to hate"）

### 不要使用的情况
- ❌ 标题里夹英文：「Marvel Rivals 案例分析」
- ✅ 标题用中文：「漫威争锋 案例分析」
- ❌ 表格里中英混用：「Helldivers 2 / 幻兽帕鲁」
- ✅ 全表统一中文：「绝地潜兵 2 / 幻兽帕鲁」
