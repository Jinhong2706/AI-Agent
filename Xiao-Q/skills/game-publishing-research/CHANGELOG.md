### Design principles

- NEVER preset path taxonomy — induce from current case data
- Every data point must have a citable source
- TL;DR first
- Counter-examples only as foil, never as standalone chapter
- Highlight viral phrases verbatim
- Multi-choice over open-ended questions
- Practical takeaway for publishers in every case
