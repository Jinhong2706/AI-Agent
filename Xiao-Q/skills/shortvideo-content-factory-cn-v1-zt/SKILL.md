# Short Video Growth Suite

## Goal
Provide one unified workflow from content planning to conversion execution.

## Supported Scenarios
1. 短视频脚本与矩阵内容生产
2. 抖音直播带货话术
3. 小红书笔记结构化产出
4. 微信私域转化文案序列
5. 本地商家活动营销文案
6. 知识付费课程发售内容

## Input Collection
Collect only necessary fields:
- 行业/赛道
- 目标受众（人群 + 痛点 + 消费能力）
- 素材池（卖点/案例/FAQ/用户反馈）
- 平台（抖音/小红书/视频号/私域）
- 目标动作（私信/留资/下单/预约/到店）
- 交付周期（7天/14天/30天）

If key fields are missing, ask at most 3 clarifying questions, then proceed with practical defaults.

## Output Contract
Always output in this order:

1. **内容策略**
- 3-5个内容支柱（认知/信任/转化）
- 每个支柱对应目标动作

2. **选题与排期**
- 7天或14天清单
- 每天主题、平台、发布时间建议

3. **短视频脚本包**
- 分镜（镜头/动作/字幕/时长）
- 口播稿（可直接念）
- CTA（单一动作）

4. **标题与封面**
- 标题10-15条（分风格）
- 封面短句5条（6-14字）

5. **转化文案模块（按用户场景启用）**
- 直播：开场、转品、异议处理、收口
- 私域：欢迎语、7天跟进、召回
- 本地活动：预热、爆发、返场
- 课程发售：预热、开售、收口

6. **矩阵改写**
- 同主题 A/B/C 三版，避免同质化

7. **执行检查清单**
- 拍摄前 / 发布前 / 复盘后

## Style Rules
- 简洁、可执行、可复制
- 每条内容只保留1个核心卖点
- 先结论后解释，避免长段空话
- 默认中文口语化表达

## Platform Adaptation
- Douyin: 强钩子前3秒，节奏快，强CTA
- Xiaohongshu: 场景化表达，真实经验感
- WeChat Channels: 信息密度均衡，强调可信证据

## Compliance
Must reject or rewrite:
- 医疗疗效绝对化
- 金融收益承诺
- 夸大/虚假宣传
- 违法引流与违规承诺

When rejecting, provide safe alternative wording.

## Quality Checklist
Before final output, verify:
- [ ] 输出内容与用户行业强相关
- [ ] 每条脚本可直接拍、口播可直接读
- [ ] 排期与用户产能匹配
- [ ] 文案无违规承诺词
- [ ] 至少给出1个可复用模板
