## 四、发布配置模板

```

1. 内容检查
   □ 视频画质清晰

   □ 无敏感内容
   □ 字幕准确

2. 文案检查
   □ 标题吸引人
   □ 话题标签相关
   □ @提及正确
   □ 位置定位（如适用）

3. 时机检查
   □ 发布时间合适
   □ 封面吸引人
   □ 描述完整

4. 互动准备
   □ 首评已准备
   □ 置评已准备
   □ 定时发布设置
```

## 七、话术模板库

### 7.1 开场话术

```
暖场话术：
"欢迎{名字}进入直播间！（读名字）
感谢关注！
刚进来的小伙伴打个招呼～
今天直播间有{福利}，先别走开！"

价值预告：
"今天给大家带来{数量}款{产品}，
{好处}！直播间专属价，只有今天有！"
```

### 7.2 促单话术

```
紧迫感：
"库存只剩{数量}件了！"
"价格只限今天直播间！"
"这波福利结束就没了！"

福利：
"拍2件送{赠品}，拍3件送{赠品2}！"
"现在下单的粉丝额外送{赠品}！"

从众：
"已经{数量}人下单了！"
"刚才那个姐姐一下拍了{数量}件！"
```

### 7.3 互动话术

```

"点赞到{数量}抽{奖品}！"

"{问题}问得最多，给大家统一回复..."
```
