# 长篇小说真相文件系统

## 核心功能
结构化追踪长篇小说创作中的四大核心要素：
- 角色设定与行为一致性
- 伏笔埋设与回收进度
- 世界观规则与细节
- 时间线逻辑连贯性

## 配套模板
模板文件位于 `templates/` 目录：
1. `写前自检表.md`：创作前核查设定一致性的检查清单
2. `写后结算表.md`：章节完成后复盘伏笔、角色、时间线的结算模板

## 使用流程
1. 用户说「加载真相文件系统」/「检查小说设定一致性」时触发
2. 根据当前创作进度，调用对应模板生成检查清单
3. 创作完成后，用写后结算表复盘一致性问
4. 所有追踪记录自动存入记忆，下次创作可直接调用

## 注意事项
- 所有追踪数据仅存用户明确要求记录的内容，仅用户可访问
- 模板可重复使用，每次创作前自动重置状态
