# 北京市二年级下学期口算练习生成器

## 概述

本 Skill 用于生成北京市二年级下学期数学口算练习题，最终输出为 docx 文件（A4横版、无边框表格）。练习题涵盖三个核心知识点，共70道题目，仅包含题目，不含答案。生成前会先询问用户需要生成的套数（1～20套），根据用户输入生成对应数量的练习题。

## 交互流程

1. **询问套数**：用户触发skill后，首先询问"请输入需要生成的套数（1～20）："
2. **生成题目**：根据用户输入的套数，调用脚本生成对应的docx文件
3. **输出结果**：多套题目合并到单个docx文件中，每套占一页，文件名包含日期

## 知识点与题量

| 知识点 | 题量 | 排列方式 |
|--------|------|----------|
| 万以内数的加法和减法 | 20题 | 4行×5列 |
| 有余数的除法 | 25题 | 5行×5列 |
| 混合运算 | 25题 | 5行×5列 |

## 使用方法

### 基本用法

运行 `scripts/generate_merged.py` 脚本生成合并的练习题 docx 文件：

```bash
python3 scripts/generate_merged.py -n 3
```

默认在当前目录生成 `oral_calc_practice_merged_年-月-日.docx` 文件，包含3套题目，每套占一页。

### 脚本说明

| 脚本 | 说明 |
|------|------|
| `scripts/generate_practice.py` | 生成独立的多套docx文件（每套一个文件） |
| `scripts/generate_merged.py` | 生成合并的单个docx文件（多套合并在一个文件中，每套占一页） |

### 命令行参数（generate_merged.py）

| 参数 | 缩写 | 说明 | 默认值 |
|------|------|------|--------|
| `--output` | `-o` | 输出文件路径 | 当前目录/oral_calc_practice_merged_年-月-日.docx |
| `--seed` | `-s` | 随机种子，用于生成不同题目 | 随机 |
| `--count` | `-n` | 生成套数（1～20） | 1 |

### 命令行参数（generate_practice.py）

| 参数 | 缩写 | 说明 | 默认值 |
|------|------|------|--------|
| `--output` | `-o` | 输出文件路径 | 当前目录/oral_calc_practice_年-月-日.docx |
| `--seed` | `-s` | 随机种子，用于生成不同题目 | 随机 |
| `--title` | `-t` | 练习题标题 | 北京市二年级下学期口算练习 年-月-日 |
| `--count` | `-n` | 生成套数（1～20） | 1 |
| `--format` | `-f` | 输出格式：docx 或 md | docx |

### 使用示例

生成3套合并的练习题：

```bash
python3 scripts/generate_merged.py -n 3
```

生成5套合并的练习题并指定输出路径：

```bash
python3 scripts/generate_merged.py -n 5 -o /path/to/output.docx
```

生成1套独立练习题：

```bash
python3 scripts/generate_practice.py
```

## 输出格式

### docx 文件（最终输出）

- 页面设置：A4 横版，页边距上下0.8cm、左右1.5cm
- 正文行间距：1倍行间距（标题、小节标题等段落）
- 表格行间距：1.4倍行间距
- 字体：全文统一为宋体
- 标题：宋体，14号，居中，加粗，包含日期（如"北京市二年级下学期口算练习 2026-05-24"）
- 小节标题：宋体，10.5号，加粗
- 题目表格：每行5题，5列排列（无边框，宋体，10.5号，左对齐）
- 作答方式：等号后留白，学生直接在等号后书写答案
- 内容限制：每套题目确保在单页内完整显示
- 多套合并：每套题目之间用分页符分隔

### 文件命名

- 合并文件：`oral_calc_practice_merged_2026-05-24.docx`
- 独立文件：`oral_calc_practice_2026-05-24.docx`

## 题目生成规则

### 万以内数的加法和减法

- 涵盖4种题型：
  - 四位数加减法（如 `3000 + 5000 =`、`5000 - 800 =`）
  - 三位数加减法（如 `510 - 240 =`、`980 + 870 =`）
- 加法结果不超过9999，减法结果为正数
- 格式：`a + b =` 或 `a - b =`

### 有余数的除法

- 被除数范围：5～99
- 除数范围：2～9
- 余数严格小于除数且大于0
- 商范围：1～9（商为一位数）
- 格式：`a ÷ b =`

### 混合运算

- 运算类型涵盖6种模板：
  - `d - a ÷ b =`（如 `44 - 35 ÷ 5 =`）
  - `d + a ÷ b =`（如 `50 + 56 ÷ 8 =`）
  - `d - a × b =`（如 `87 - 9 × 5 =`）
  - `d + a × b =`（如 `7 × 7 + 45 =`）
  - `a ÷ b - d =`（如 `45 ÷ 9 - 6 =`）
  - `a ÷ b + d =`（如 `24 ÷ 8 + 6 =`）
- 遵循运算优先级：先乘除后加减
- 所有中间结果和最终结果均为正整数
- 格式：`表达式 =`
