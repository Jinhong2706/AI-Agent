# 极端行情应对

## 概述

极端行情是检验投资者功力的时刻。**行情极端时，最重要的是活下来**。本文介绍天地板、核按钮、翘板等极端行情的识别和应对方法。

---

## 一、天地板与地天板

### 1. 天地板（一字跌停）

**定义**：从涨停到跌停，当日最大跌幅20%

**特征**：
- 开盘即涨停，盘中突然炸板
- 快速跌停，K线呈直线
- 成交量突然放大

**出现场景**：
- 情绪高潮后的退潮
- 利空消息突发
- 主力出货

**应对策略**：

| 情况 | 应对 |
|------|------|
| 持有中遇到 | 次日竞价挂单，等翘板机会 |
| 追高遇到 | 当日不能卖出，等待次日 |
| 翘板失败 | 果断止损，不要幻想 |

**注意事项**：
- 天地板次日往往惯性下跌
- 不要在跌停价补仓摊低成本
- 记录天地板股，作为情绪信号

### 2. 地天板（一字涨停）

**定义**：从跌停到涨停，当日最大涨幅20%

**特征**：
- 开盘即跌停，盘中突然拉升
- 快速涨停，K线呈直线
- 成交量明显放大

**出现场景**：
- 情绪冰点后的转折
- 利空出尽
- 主力自救

**应对策略**：

| 情况 | 应对 |
|------|------|
| 持有中遇到 | 持有观察，不要急于卖出 |
| 低吸遇到 | 可以小仓位参与 |
| 追涨遇到 | 注意特停风险 |

**注意事项**：
- 地天板次日往往有溢价
- 注意成交量是否持续
- 关注特停风险

## 二、核按钮

### 1. 核按钮定义

**核按钮**：用大单直接将股价砸至跌停的行为

**特征**：
- 涨停股突然被大单砸盘
- 快速下跌，不给反应时间
- 往往是情绪转弱的信号

### 2. 核按钮原因

| 原因 | 说明 |
|------|------|
| 情绪退潮 | 资金恐慌出逃 |
| 主力砸盘 | 主力获利了结 |
| 利空消息 | 突发负面消息 |
| 踩踏出逃 | 恐慌情绪蔓延 |

### 3. 核按钮应对

**预防措施**：
- 不追高位股
- 控制仓位
- 设置止损线

**遇到核按钮**：
- 当日无法卖出
- 次日竞价挂单
- 观察是否有翘板
- 翘板失败果断止损

**核按钮股操作**：
- 翘板成功可以持有
- 翘板失败必须走
- 不幻想，不死扛

---

## 三、翘板操作

### 1. 翘板定义

**翘板**：用大单将跌停股拉起的行为

### 2. 翘板类型

| 类型 | 特征 | 风险 |
|------|------|------|
| 主力自救 | 主力被套，拉升出货 | 风险极高 |
| 情绪转折 | 情绪冰点转暖 | 机会较大 |
| 消息刺激 | 有利好消息 | 需要验证 |

### 3. 翘板操作策略

**适合翘板的情况**：
- 大盘情绪企稳
- 板块有修复预期
- 翘板资金实力强

**不适合翘板的情况**：
- 大盘继续下跌
- 板块继续调整
- 个股有利空

**翘板操作口诀**：
- 只翘龙头，不翘跟风
- 大盘配合才翘
- 翘板失败必须走
- 不幻想二次拉升

---

## 四、炸板处理

### 1. 炸板定义

**炸板**：涨停股打开涨停板

### 2. 炸板类型

| 类型 | 特征 | 应对 |
|------|------|------|
| 良性炸板 | 回封快，承接强 | 持有或低吸 |
| 恶性炸板 | 不回封，持续下跌 | 果断止损 |
| 尾盘炸板 | 临近收盘才炸 | 注意次日低开 |

### 3. 炸板应对

**良性炸板**：
- 观察是否快速回封
- 回封后可以持有
- 不回封要减仓

**恶性炸板**：
- 果断止损
- 不幻想回封
- 次日可能低开

**尾盘炸板**：
- 减仓为主
- 等待次日表现
- 不追加仓位

## 五、特停风险

### 1. 特停定义

**特停**：股票因涨幅过大被交易所临时停牌

### 2. 特停规则

| 市场 | 规则 |
|------|------|
| 沪市 | 连续3个交易日涨幅偏离值累计达20% |
| 深市 | 连续10个交易日内收盘价格涨幅偏离值累计100% |

### 3. 特停应对

**预防特停**：
- 关注连板高度
- 避开特停节点
- 控制仓位

**遇到特停**：
- 复牌后观察表现
- 龙头特停期间关注补涨
- 特停复牌后往往有分歧

**特停复牌操作**：
- 超预期表现：继续持有
- 低于预期：果断止盈
- 一字跌停：等待翘板

## 六、大盘极端行情

### 1. 大盘暴涨

**特征**：
- 全线上涨
- 涨停家数多
- 成交量放大

**应对**：
- 顺势加仓
- 持有不动
- 让利润奔跑

### 2. 大盘暴跌

**特征**：
- 全线下跌
- 跌停家数多
- 恐慌情绪蔓延

**应对**：
- 果断减仓
- 不抄底
- 等待企稳

### 3. 大盘熔断（历史）

**特征**：
- 暂停交易
- 流动性枯竭
- 恐慌踩踏

**应对**：
- 远离市场
- 等待恢复
- 不抄底

## 七、极端行情心态

### 1. 极端行情心理

| 心态 | 错误做法 | 正确做法 |
|------|---------|---------|
| 恐慌 | 盲目割肉 | 冷静分析 |
| 贪婪 | 追涨加仓 | 及时止盈 |
| 幻想 | 不止损死扛 | 果断止损 |

### 2. 极端行情原则

1. **活着最重要**：不爆仓是底线
2. **不抄底**：跌了还有更低
3. **不逃顶**：卖早了比卖晚了好
4. **控制仓位**：仓位轻心态好
5. **严格执行**：设好的止损必须执行

### 3. 极端行情后操作

**暴跌后**：
- 不急于抄底
- 等待情绪稳定
- 分批建仓

**暴涨后**：
- 不追高
- 逐步减仓
- 守住利润
