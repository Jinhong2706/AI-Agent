# 短线操作手法

## 概述

短线操作是A股市场最激动人心的交易方式。**买绿不买红，追涨不追高**。本章详细介绍打板、潜伏、接力、低吸等主流短线操作模式。

---

## 一、打板操作

### 什么是打板

在股票涨停的瞬间买入，期待次日溢价卖出。

### 打板类型

#### 1. 首板
- 股票首次涨停
- 确定性较低，需要观察
- 适合试错

#### 2. 连板
- 连续涨停的股票
- 妖股确认阶段
- 风险与收益并存

#### 3. 龙头板
- 板块内的领涨股
- 有带动效应
- 最值得参与

### 打板原则

| 原则 | 说明 |
|------|------|
| 不打普通股板 | 只打有明确逻辑和板块效应的板 |
| 打回封板 | 炸板后能快速回封的股票更强 |
| 不打缩量板 | 高位缩量板风险大，容易炸板 |
| 控制仓位 | 打板仓位不超过总仓位的1/3 |
| 避开特停节点 | 高位连板面临特停风险 |

### 打板时机

**最佳时机**：
- 早盘9:30-10:00之间
- 板块效应明显时
- 大盘环境配合

**不佳时机**：
- 尾盘涨停
- 大盘环境差
- 无板块效应

### 打板后处理

| 情况 | 应对 |
|------|------|
| 封板成功 | 持有到尾盘 |
| 炸板不回封 | 果断止损 |
| 次日高开 | 持有或分批卖出 |
| 次日低开 | 观察是否企稳 |

## 二、潜伏策略

### 什么是潜伏

在股票启动前提前买入，等待拉升后卖出。

### 潜伏逻辑

#### 1. 事件驱动潜伏
- 重大会议（两会、科创板等）
- 财报预期（业绩预增、高送转）
- 政策出台（行业扶持、政策利好）
- 产品发布（苹果发布会、新手机等）

#### 2. 技术形态潜伏
- 横盘缩量到极致
- 底部区域震荡
- 均线粘合
- 量能萎缩

#### 3. 估值修复潜伏
- 超跌蓝筹
- 错杀优质股
- 行业反转预期

### 潜伏要点

#### 选股标准
- 基本面不差
- 板块有预期
- 位置相对安全
- 流动性尚可

#### 买入方式
- 分批买入（3-4次）
- 不一次梭哈
- 控制总仓位

#### 持有耐心
- 逻辑兑现需要时间
- 不因短期波动动摇
- 设定止损位

#### 卖出时机
- 逻辑兑现当日
- 达到预期涨幅
- 出现破位信号

## 三、接力模式

### 什么是接力

在连板股打开空间后，寻找跟风股买入。

### 接力分类

#### 1. 龙头接力
- 追涨总龙头
- 风险最高，收益也最高
- 需要对情绪判断精准

#### 2. 1进2接力
- 首板股次日接力
- 成功率较高的模式
- 关键在于选股

#### 3. 跟风接力
- 龙头打开空间后
- 挖掘同板块跟风股
- 安全性相对较高

### 接力要点

#### 板块效应
- 龙头必须带动板块
- 跟风股必须与龙头有逻辑关联
- 板块内要有梯队

#### 介入时机
- 跟风股首板次日
- 不要追涨太高
- 位置是关键

#### 止损原则
- 龙头走弱及时出
- 跟风失败及时走
- 不幻想、不死扛

## 四、低吸策略

### 什么是低吸

在股票回调时买入，等待反弹卖出。

### 低吸时机

#### 1. 回调到均线支撑
- 5日线支撑
- 10日线支撑
- 20日线支撑

#### 2. 回调到前期低点
- 前期低点有支撑
- 双底结构
- 平台支撑

#### 3. 大幅杀跌后
- 恐慌盘出逃
- 超跌反弹机会
- 短线机会

### 低吸口诀
- **买绿不买红**：回调时买，不追涨
- **阴线买，阳线卖**：跌时买，涨时卖
- **分批低吸**：不要一次买满

## 五、趋势持股

### 什么是趋势持股

跟随股票趋势持有，直到趋势破坏。

### 趋势判断

#### 均线系统
- 5日线：短期趋势
- 10日线：中期趋势
- 20日线：波段趋势

#### 趋势特征
- 股价沿均线上涨
- 回调不破均线
- 量价配合

### 趋势操作

#### 持股原则
- 趋势不破不卖
- 跟随趋势享受利润
- 不要主观猜顶

#### 卖出信号
- 破5日线减仓
- 破10日线清仓
- 高位放量大阴线

---

## 六、仓位控制

### 仓位原则
- 分散持仓：单只不超过30%
- 动态调整：根据情绪加减仓
- 留有余地：不一次打满

### 仓位与情绪
| 情绪阶段 | 仓位建议 |
|---------|---------|
| 启动期 | 5-7层 |
| 高潮期 | 7-10层 |
| 退潮期 | 3-5层 |
| 冰点期 | 0-3层 |

---

## 七、操作心态

### 核心心态
1. **不急躁**：不要频繁交易
2. **不恐惧**：不要害怕错过
3. **不幻想**：不要死扛亏损
4. **不攀比**：不要羡慕别人的股票

### 常见错误
1. **追涨杀跌**：涨了才想买，跌了想卖
2. **频繁换股**：手里股票不涨，其他涨得好
3. **死扛亏损**：亏了不止损，等回本
4. **利润兑现太早**：赚一点就跑

### 正确心态
- 买前有计划
- 买后有耐心
- 错了及时改
- 对了要拿住
