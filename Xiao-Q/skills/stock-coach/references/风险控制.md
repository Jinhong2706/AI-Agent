# 风险控制

## 概述

风险控制是A股投资中最核心的技能之一。**宁可抛错，不能持股看着下跌不止盈**。市场充满不确定性，止损和止盈是保护本金、控制回撤的关键手段。

---

## 一、止损的时机和方法

### 1. 核心止损原则

- **KDJ三次死叉必须清仓**：当KDJ指标出现三次死叉时，准确率极高，必须果断清仓等待更低位置
- **跌破均线必须减仓**：当股价跌破重要均线（如15日线、60日线）时，要及时减仓
- **不与股票谈恋爱**：亏钱的股票不能死扛，要及时止损出局

### 2. 止损时机判断

| 情况 | 应对策略 |
|------|---------|
| 竞价低开太多 | 竞价就跑，降低仓位 |
| 直线跳水不反弹 | 当日假摔可观望，次日不反弹必须走 |
| 大盘出现死神图 | 迅速降低仓位到5-7成 |
| 追高买错 | 2-4个交易日内会有反抽板减亏机会 |
| 逻辑不兑现 | 及时止损，不幻想 |

### 3. 止损方法

- **竞价止损法**：竞价低开超过预期，果断竞价跑
- **均线止损法**：破15日线、60日线等重要均线必须减仓
- **成本线止损法**：利润回撤到成本线附近必须出局
- **时间止损法**：买入后长时间不涨，逻辑不兑现必须走

## 二、止盈的策略

### 1. 移动止盈法

**核心口诀**：
- 10个点利润：吐到7个点要无脑清仓
- 15个点利润：吐到11个点要无脑清仓
- **守住利润，守住利润，守住利润**

### 2. 分批止盈法

| 利润幅度 | 操作策略 |
|---------|---------|
| 10%+ | 先止盈一半，锁住利润 |
| 15%+ | 再止盈25% |
| 20%+ | 留少量仓位博取更大收益 |
| 高位放量大阴线 | 全部止盈 |

### 3. 止盈信号识别

- **高位放量滞涨**：当日涨幅大但成交量异常放大，可能是出货信号
- **连续加速上涨**：妖股连续一字板后开板，警惕主力出货
- **板块后排跟风股炸板**：龙头还在，但跟风股已经开始炸板
- **特停风险**：连板股面临特停风险时，提前止盈

---

## 三、被套后的应对策略

### 1. 被套原因分析

- 大盘系统性风险
- 追高买入位置不对
- 板块轮动节奏踏错
- 个股出现利空

### 2. 应对方法

| 情况 | 应对策略 |
|------|---------|
| 大盘只是假摔 | 可以持有等待反弹，但控制仓位 |
| 大盘真跌 | 果断止损，不要犹豫 |
| 一字板跌停 | 竞价挂单跌停价，等待翘板机会 |
| 被套在高位 | 等待反弹到均线附近减亏出局 |
| 逻辑还在 | 可以加仓降低成本，等待逻辑兑现 |

### 3. 注意事项

- **不补仓摊低成本**：跌了补仓是被自己账户证明完全错误的操作
- **不与股票谈恋爱**：亏钱的股票不能死扛
- **分析被套原因**：分清是系统性风险还是个股问题
- **设置止损线**：买入时就要想好止损位

---

## 四、减仓和清仓的判断标准

### 1. 减仓标准

- 大盘出现死神图信号
- 板块出现明显退潮迹象
- 个股出现高位放量大阴线
- 账户利润回撤超过预期
- 遇到重要压力位

### 2. 清仓标准

- KDJ三次死叉
- 跌破60日线（牛市生命线）
- 出现特停风险
- 情绪周期进入冰点期
- 遭遇黑天鹅事件
- 大盘进入风险期

### 3. 仓位控制原则

| 市场环境 | 建议仓位 |
|---------|---------|
| 牛市上升期 | 8-10层 |
| 震荡市 | 5-7层 |
| 情绪退潮期 | 3-5层 |
| 情绪冰点期 | 轻仓或空仓 |
| 大盘风险期 | 空仓观望 |

## 五、实战口诀

1. **宁可抛错，不能持股看着下跌不止盈**
2. **多想想你赚到的，少想象你少赚的**
3. **利润多的不要膨胀，利润少的不要急躁**
4. **分仓操作，降低单只股票的风险**
5. **买错不要紧，关键是及时止损**
6. **守住利润比追求利润更重要**

---

## 六、注意事项

1. **预设止损位**：买入时就设定好止损位，不要临时决定
2. **严格执行**：设定的止损止盈规则必须严格执行
3. **分批操作**：不要一次性满仓或清仓，分批操作更安全
4. **控制单票仓位**：单只股票仓位不超过总仓位的1/3
5. **顺势而为**：上升趋势持股，下跌趋势持币
6. **情绪管理**：不要因为亏损影响判断，也不要因为盈利而过度自信
