# 仓位管理

## 一、仓位管理原则

### 核心原则
1. **不重仓单只**：单只股票仓位不超过30%
2. **动态调整**：根据市场情绪调整仓位
3. **分批操作**：买入卖出分批进行
4. **留有余地**：不要一次打满仓

### 仓位与风险
| 仓位 | 风险等级 | 适用情况 |
|------|---------|---------|
| 10% | 极低 | 极度谨慎 |
| 30% | 低 | 退潮期/冰点期 |
| 50% | 中 | 启动期 |
| 70% | 高 | 高潮期 |
| 100% | 极高 | 极端行情 |

## 二、情绪与仓位

### 四阶段仓位配置

#### 启动期（建仓期）
```
建议仓位：5-7层
策略：积极试错，寻找方向
个股仓位：每只2-3层
持仓数量：2-3只
```

#### 高潮期（加速期）
```
建议仓位：7-10层
策略：重仓出击
个股仓位：每只3-4层
持仓数量：2-3只
```

#### 退潮期（分歧期）
```
建议仓位：3-5层
策略：谨慎操作
个股仓位：每只1-2层
持仓数量：2只以内
```

#### 冰点期（迷茫期）
```
建议仓位：0-3层
策略：观望为主
个股仓位：小仓位试错
持仓数量：1只或不持仓
```

## 三、分批建仓法

### 买入分批
```
第一次买入：30%（试探）
第二次买入：30%（确认）
第三次买入：40%（重仓）
```

### 卖出分批
```
第一次卖出：30%（收回成本）
第二次卖出：40%（锁定利润）
第三次卖出：30%（留仓博取更大利润）
```

### 适用场景
- 趋势股操作
- 抄底操作
- 不确定行情

## 四、止损止盈

### 止损原则
- **固定止损**：亏损5-8%必须止损
- **移动止损**：盈利后提高止损位
- **时间止损**：持有超过一定时间不涨
- **逻辑止损**：买入逻辑消失

### 止盈原则
- **分批止盈**：不要一次全部卖出
- **移动止盈**：跟随趋势不断提高止盈位
- **逻辑止盈**：逻辑兑现后止盈
- **高位止盈**：高位放量大阴线

### 盈亏比
- 理想盈亏比：3:1
- 即盈利应该是止损的3倍
- 这样胜率30%就能盈利

## 五、不同行情仓位

### 牛市行情
- 仓位可以偏重
- 顺势而为
- 回调加仓
- 让利润奔跑

### 熊市行情
- 仓位必须轻
- 控制风险
- 快进快出
- 不轻易抄底

### 震荡行情
- 仓位适中
- 高抛低吸
- 不追涨杀跌
- 控制频率

## 六、资金管理

### 总仓位控制
```
激进型：80-100%
稳健型：50-70%
保守型：30-50%
```

### 留仓资金
- 建议留20%资金应对突发
- 不要满仓操作
- 留有余地

### 加仓原则
- 只加盈利的仓位
- 不加亏损的仓位
- 加仓要轻

## 七、心态控制

### 仓位心态
1. **不眼红**：不羡慕别人满仓赚大钱
2. **不后悔**：不因卖早而后悔
3. **不侥幸**：不幻想跌了的能涨回来
4. **不贪婪**：不要想卖在最高点

### 正确心态
- 赚钱是市场强
- 亏钱是我自己逞强
- 活得久比赚得快重要
- 复利的力量

## 八、实战技巧

### 开仓技巧
- 大盘好时重仓
- 大盘差时轻仓
- 情绪高潮时减仓
- 情绪冰点时关注机会

### 持仓技巧
- 五日线持股
- 不轻易动
- 趋势不破不走
- 逻辑不破不卖

### 卖出技巧
- 分批卖出
- 不追求卖最高
- 破线必须走
- 记住落袋为安

## 九、常见错误

### 仓位错误
1. **一次满仓**：风险太大
2. **频繁满仓**：容易踩错节奏
3. **死守重仓**：亏损不断放大
4. **卖完踏空**：过早清仓

### 解决办法
1. 分批建仓
2. 控制单只仓位
3. 设置止损
4. 保持部分仓位

## 十、总结

### 仓位管理核心理念
1. **活着最重要**：不爆仓是底线
2. **顺势而为**：跟着市场走
3. **留有余地**：永远不要满仓
4. **复利思维**：稳定盈利比大赚重要

### 口诀
- **轻仓试错，重仓顺势**
- **赚钱加仓，亏钱减仓**
- **分批操作，不赌一把**
- **活着比赚钱重要**
