# 全网热搜早报技能

## 📦 技能说明

这是一个为 QQ 小Q AI助手设计的技能，能够自动聚合微博、知乎、B站三大平台的热搜，并生成每日早报。

## ✨ 功能特性

- 🔍 **多平台聚合**：自动抓取微博、知乎、B站实时热搜
- 📊 **智能排序**：按热度和讨论度自动筛选Top 5
- ⏰ **定时推送**：支持每天8点自动推送（需配合schedule-reminder技能）
- 🎙️ **语音播报**：可选TTS语音输出
- 🔍 **深度解读**：支持对单个热搜话题的详细展开
- 🎯 **话题过滤**：可按需排除特定类型话题（如娱乐新闻）

## 📖 使用方法

### 查看早报

只需对小Q说：
- "看早报"
- "今日热搜"
- "有什么热点"

### 设置定时推送

对小Q说：
- "每天早上8点给我推送早报"
- "定时推送热搜"

### 语音播报

- "用语音播报今天的早报"
- "听早报"

### 深度解读

- "详细说说第2条"
- "展开讲讲XXX话题"

## 🔧 依赖技能

- `web_search` - 联网搜索（内置）
- `tts` - 文字转语音（内置）
- `schedule-reminder` - 定时任务（可选，用于自动推送）

## ⚙️ 配置说明

### 定时任务配置

如需修改推送时间，可以在对话中告诉小Q：
- "改成每天7点推送早报"
- "取消早报推送"

### 话题过滤

- "看早报，不要娱乐新闻"
- "只看科技类热点"

## 📝 注意事项

1. **时效性**：热搜数据实时变化，每次都会重新搜索
2. **网络依赖**：需要联网才能获取热搜数据
3. **平台限制**：搜索结果质量依赖于 `web_search` 的返回
4. **使用频率**：建议每天最多查看2-3次，避免频繁调用

## 🐛 已知问题

- [ ] 部分平台搜索结果可能不稳定
- [ ] 跨平台去重算法需要持续优化
