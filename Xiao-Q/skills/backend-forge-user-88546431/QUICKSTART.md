# Backend Forge Quickstart

## 中型项目后端架构负责人

```text
bf- 我是前端负责人，需要独立承担一个中型 SaaS 项目的后端架构，请先带我做架构简报、ADR、数据/权限/运行架构和实现分片
```

```text
bf-a 以有经验前端工程师担任后端架构负责人的方式，给这个现有 FastAPI 项目梳理模块边界、数据模型、权限模型和分阶段交付计划
```

## 新项目共创

```text
bf- 新建一个 FastAPI 待办事项后端，包含用户、项目、任务，使用 PostgreSQL，先做本地可运行版本和测试闭环
```

```text
bf- 新建一个 Spring Boot 库存管理单服务，包含商品、库存流水、出入库接口，要求有迁移和接口测试
```

## 已有项目功能开发

```text
bf- 在现有 Spring Boot 项目里增加订单取消接口，需要校验订单状态、写操作日志，并补测试
```

```text
bf- 在 FastAPI 项目里给用户资料增加头像 URL 字段，包含迁移、接口更新和回归测试
```

## Bug/生产问题排查

```text
bf- 这个接口偶发重复扣库存，帮我先定位并补复现测试，再修复
```

```text
bf-fast 修复 users API 返回 500，错误日志在下面
```

## 测试补强

```text
bf- 给当前认证模块补测试，至少覆盖未登录、权限不足、越权和正常授权
```

## 性能/并发优化

```text
bf- 这个查询在数据量上来后很慢，帮我定位索引或查询问题，并给出可重复验证
```

## 全自动策略

```text
bf-a 给这个 FastAPI 项目增加健康检查接口和测试，按现有风格实现
```

`bf-a` 可以自动补安全默认细节，但遇到业务语义、数据删除、权限模型、生产配置、架构会签项时必须停下确认。
