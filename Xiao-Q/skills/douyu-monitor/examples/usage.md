# 使用示例

## 基础使用

### 示例1：添加主播监控
**用户输入：**
```
帮我监控张大仙
```

**Agent 响应：**
```
好的，正在查找主播...
✅ 已添加监控：张大仙（12345）
当前状态：🔴 直播中 - 今晚冲国服！
有开播动态会第一时间通知你 🔔
```

---

### 示例2：通过房间号添加
**用户输入：**
```
添加斗鱼主播 12345
```

**Agent 响应：**
```
✅ 已添加监控：房间号 12345
当前状态：⚪ 未开播
已加入监控列表，开播后会通知你 🔔
```

---

### 示例3：查询开播状态
**用户输入：**
```
张大仙开播了吗
```

**Agent 响应：**
```
📺 查询结果：
张大仙（12345）：🔴 直播中
标题：今晚冲国服！
分类：王者荣耀
观看地址：https://www.douyu.com/12345
```

---

### 示例4：查看监控列表
**用户输入：**
```
查看直播监控列表
```

**Agent 响应：**
```
📋 当前监控列表（共3位主播）：
1. 张大仙（12345）- 🔴 直播中
2. 梦泪（67890）- ⚪ 未开播
3. PDD（11111）- ⚪ 未开播
```

---

### 示例5：开播提醒（定时触发）
**触发条件：** 定时任务检测到主播开播

**Agent 推送消息：**
```
🔴 【开播提醒】
主播：张大仙
标题：今晚冲国服！
分类：王者荣耀
房间号：12345
观看地址：https://www.douyu.com/12345
开播时间：2026-06-20 20:00
```

---

### 示例6：取消监控

```

**Agent 响应：**
```

## 进阶使用

### 示例7：批量添加监控
**用户输入：**
```
帮我监控张大仙、梦泪、PDD
```

**Agent 响应：**
```
好的，正在添加...
✅ 已添加：张大仙（12345）
✅ 已添加：梦泪（67890）
✅ 已添加：PDD（11111）
共监控 3 位主播，开播动态会第一时间通知你 🔔
```

---

## 异常场景示例

### 示例10：主播不存在
**用户输入：**
```
监控主播 abcdef
```

**Agent 响应：**
```
❌ 未找到该主播，请确认：
• 房间号是否正确（纯数字）
• 主播昵称是否准确
• 该主播是否在斗鱼平台
```

---

### 示例11：监控列表为空
**用户输入：**
```
查看直播监控列表
```

**Agent 响应：**
```
📋 当前监控列表为空
先用"监控XX主播"添加主播吧 😊
```

---

### 示例12：取消不存在的监控
**用户输入：**
```
取消监控张三
```

**Agent 响应：**
```
❌ 监控列表中没有找到"张三"
当前监控的主播：张大仙、梦泪
```
