# 斗鱼主播开播提醒

## 技能简介

自动监控斗鱼主播的开播状态，一旦检测到开播立即通过QQ消息推送提醒，让你不再错过喜欢的主播直播 🎮

## 适用场景

- ✅ 想第一时间知道喜欢的主播开播了
- ✅ 同时关注多个斗鱼主播
- ✅ 不想一直刷新直播间页面
- ✅ 需要自动化的开播提醒

## 如何使用

### 添加主播监控
```
监控张大仙
添加斗鱼主播12345
帮我关注PDD
```

### 查询开播状态
```
张大仙开播了吗
梦泪在直播吗
```

### 查看监控列表
```
查看直播监控列表
我监控了哪些主播
```

### 取消监控
```
取消监控张大仙
不用监控PDD了
```

## 触发词参考

| 意图 | 触发词示例 |
|------|-----------|
| 添加监控 | 监控、添加、关注、订阅 |
| 查询状态 | 开播了吗、在直播吗、状态 |
| 查看列表 | 列表、哪些、查看、监控 |

## 输出内容

### 开播提醒
- 主播昵称
- 直播标题
- 直播分类
- 房间号
- 观看地址
- 开播时间

### 状态查询
- 当前是否在直播
- 直播标题
- 直播分类

## 安装说明

1. 将 `douyu-monitor` 文件夹放入技能目录
2. 重启 Agent 或重新加载技能列表
3. 直接对 Agent 说"监控XX主播"即可使用

## 注意事项

- ⚠️ 本技能仅支持斗鱼平台
- ⚠️ 需要定时任务支持才能实现自动提醒
- ⚠️ 监控数据存储在本地，清除数据需删除 `monitored_rooms.json`
- ⚠️ 斗鱼API可能有频率限制，请勿过快轮询

## 技术说明

- 使用斗鱼开放API获取房间状态
- 监控列表保存在 `monitored_rooms.json`
- 状态记录保存在 `last_status.json`
- 定时轮询间隔建议 ≥ 60 秒

## 常见问题

**Q：支持其他平台吗？**
A：当前仅支持斗鱼，后续可扩展虎牙、B站等

**Q：如何修改轮询间隔？**
A：修改定时任务的 cron 表达式即可

**Q：监控数据存在哪里？**
A：默认在当前工作目录的 `monitored_rooms.json`

**Q：能同时监控多少个主播？**
A：理论无上限，但建议 ≤ 20 个，避免API请求过快
