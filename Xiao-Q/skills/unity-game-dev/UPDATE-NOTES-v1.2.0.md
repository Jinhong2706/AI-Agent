# Unity Game Dev Skill v1.2.0 更新公示

## 更新摘要

基于用户 RACE 评价反馈，针对性优化技能的适用性、规范性与可靠性。模板数量保持 30+，新增 FAQ 故障排查页与运行时保护机制。

---

## 更新内容

### 新增
- **快速上手教程**：3 分钟做出可运行小游戏，附步骤截图级指引
- **触发关键词表**：30+ 条中英文触发词，AI 可主动判断介入时机
- **FAQ 故障排查页**（`references/faq-troubleshooting.md`）：8 大类 25+ 条常见问题，按场景分类，无需翻阅全文
- **运行时保护机制**：5 种防护模式（空值检查/单例安全/销毁判断/协程超时/异常兜底），生成代码默认包含

### 修复
- 技能描述统一为中文，修复 `contentZhAvailable=false` 问题
- 功能速查表分类优化（基础/高级移动/AI/UI/系统/架构 6 大类）
- 参考文件索引更新，新增 FAQ 文件优先级提示

## 适用人群

- Unity 零基础新手，想从零做出第一款游戏
- 有一定基础，需要快速查找某功能代码模板的开发者
