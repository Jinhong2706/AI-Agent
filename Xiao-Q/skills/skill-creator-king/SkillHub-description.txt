skill-creator-king 是从零创建、审查评分、迭代完善 AI Agent Skill 的元工具。支持 WorkBuddy / OpenClaw / Hermes 四平台，Phase 0-5 六阶段创建流程，14维质量审计，26 种反模式自动扫描，Token 预算估算，LLM 驱动修复+安全回滚，全生命周期覆盖。

核心能力：模板生成脚手架快速起步，validate.py + quality-audit.py 双引擎质量评分（131分制），engine.py 安全修复骨架（扫描→备份→验证→回滚），跨文件版本一致性检查，README/CHANGELOG 自动生成，轻量/完整双通道按需选择。零外部依赖，纯 Python 标准库即可运行全部脚本。
