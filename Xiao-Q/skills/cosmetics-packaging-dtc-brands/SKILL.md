# 海外化妆品：新锐 DTC 与融资品牌线（印刷包装潜客）

## 适用场景

- 目标为**海外**以 **DTC（独立站）或电商平台** 为主阵地的化妆品**新锐品牌**。
- 特征：SKU 上新快、季节与联名礼盒多，对**彩盒、礼盒、说明书**改版频繁，与印刷包装供应商匹配度高。
- 我方：**印刷包装**厂商（可有集团级客户服务经验，表述需与事实一致）。

## 公开联系邮箱与联系人（本 Skill 能做什么）

- **可以**：从品牌**官网 Contact、Wholesale/Stockists、Press** 等页面，以及**融资新闻稿**中媒体联络段落，摘录**公示邮箱**与**具名职务联系人**（须带来源链接）。
- **不可以**：将创始人私人社交账号当官方联系人；从数据经纪商或未授权导出中抄录个人邮箱；编造采购负责人邮箱。
- **结果预期**：大量 DTC 品牌仅提供表单或 `hello@` 类通用邮箱，表中如实填写并附 URL。

## 用户应提供的输入

- 目标**市场**（美国、英国、欧盟内国家、东南亚英语市场等）。
- **细分品类**（清洁美容、彩妆、护肤工具、香氛等）。
- 是否只看**近 N 年有融资或显著增长报道**的品牌（可选）。

## 研究流程（按顺序执行）

1. **定义「新锐」标准（与用户输入对齐）**
   - 例如：独立站 + 社交媒体官方账号 + 在售 SKU 列表可见；或 Crunchbase/PitchBook 等**公开**融资页可访问。
   - 将标准写进输出开头的「筛选假设」，便于复盘。

2. **名单来源优先级**
   - **融资与创业数据库**（公开页）：Crunchbase、PitchBook 摘要、各国科技媒体融资稿（公司名 + 官网交叉验证）。
   - **零售商「新品牌」栏目**：美妆连锁的 new brands、clean beauty 专区。
   - **众筹与首发报道**：Kickstarter/Indiegogo 美妆硬件或套装项目（注意区分一次性项目与持续经营品牌）。
   - **行业媒体榜单**：beauty industry 的「brands to watch」类年度文章（仅作线索，需官网核验）。

3. **包装需求信号（每条注明 URL）**
   - 独立站 **Shop** 是否有多 SKU、套装、节日礼盒、订阅盒。
   - **Our Story / Impact / Packaging** 是否提及环保包材、减塑、纸质外盒等。
   - **Press** 是否出现「联名包装」「限量设计」等关键词。

4. **体量与可持续性（公开信息）**
   - 记录：门店数（如有）、主要电商平台旗舰店、社媒粉丝量级区间（说明数据来源与日期）。
   - 不猜测未披露的营收；未公开则填「未披露」。

5. **公开联系方式（每条品牌做一次）**
   - 查阅独立站 **Contact、Wholesale、Press、Partners** 页面及融资稿联络段；只记录**页面上明文**邮箱。
   - **具名联系人**：仅新闻稿或官网明确写出姓名+职务时填入，并附链接。

6. **核验**
   - 每个品牌必须能打开**官网或官方旗舰店铺**并确认仍在售化妆品类商品。
   - 剔除仅代工无自有品牌、或网站失效的条目。

## 输出格式（表格字段固定）

| 字段 | 说明 |
|------|------|
| 品牌名 | 对外名称 |
| 国家/地区 | 公司注册地或主运营地 |
| 官网 / 主销售渠道 | URL |
| 品类 | |
| 新锐证据 | 融资报道链接、媒体报道、或「新品牌」栏目来源 |
| SKU/礼盒信号 | 简述 + 商品或集合页链接 |
| 包装叙事证据 | 如有，附链接 |
| 公开邮箱 | 多个用分号分隔；无则「未公示」或「仅联系表单」 |
| 邮箱与表单来源 URL | |
| 公开具名联系人 | 姓名 / 职务 / 来源链接；无则「未公示」 |
| 下一步建议 | 如：批发合作页、媒体合作入口 |

## 合规与边界

- 不使用未授权数据库内导出字段的复述；优先使用**可引用的公开网页**。
- 不批量爬取社交平台私信数据；不指导自动化注册或绕过平台限制。
- 营销触达需符合目标国对冷邮件与个人信息的规定（本 Skill 不提供法律意见）。

## 术语统一

- 使用「DTC」「独立站」「公开融资信息」「在售验证」等固定说法，与「企业线」「代工厂线」Skill 区分。
