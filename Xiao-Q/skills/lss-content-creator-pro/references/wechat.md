# 公众号写作 SOP

## 文章结构（严格遵守）

1. **标题**：悬念感/利益点/反常识，≤20字，手机屏可完整显示
2. **固定开头三行文案**（从 `references/ip-info.md` 读取，格式如下）：
   ```
   Hi 大家好，我是{{作者名}}
   {{身份介绍}}
   {{个人标签}} 🌿
   ```
   > ⚠️ 首次使用时，如果 ip-info.md 还是空模板（含 `[你的名字/笔名]`），必须先询问用户填写个人信息。
3. **开头主题配图**（⚠️ 固定规则，每篇必须）：三行文案下方紧跟一张与文章主题相关的配图，**无分割线**，直接 `![](url)`
   > 如果安装了 `image-gen-lss` skill 可自动生成；否则在文末「配图建议」中给出 prompt，用户自行生图后插入
4. **钩子句**（前3行必须勾住读者）→ 引出核心问题
5. **主体**：3-5个副标题，每节一个独立有力观点，200-400字
6. **冲突/反转**：全文至少一处「但是……」「你可能想错了……」的转折
7. **总结升华**：核心金句 + 开放性互动问题（不是「欢迎留言」这种废话）
8. **固定结尾**（排版规则如下，从 ip-info.md 读取具体文案）：

   **结尾排版结构（严格遵守）：**
   ```
   {{本篇收束金句}}

   欢迎在评论区聊聊，{{本篇互动提问}}

   ---

   END

   **我是{{作者名}}，**

   **{{身份介绍}}**

   {{结尾介绍完整版}}
   ```

   **排版逻辑**：
   - 核心自我介绍行**加粗**：突出个人品牌，是读者记住你的关键信息
   - 补充说明用 `>` 引用块：视觉弱化，作为补充而非主角
   - 如有二维码/联系方式沉底：行动引导的终点

## 爆款排版规范

### 节奏感
- 正文段落：3-4行一段，段段留白
- 金句单独成段，前后空行，用 `>` blockquote 突出
- 重点句 `**加粗**`：每段最多1处

### 小标题格式
- 统一用：`## 一、标题` 或 `## 🔑 标题`
- 小标题本身要有观点，不能只是分类词（❌「关于龙虾的三件事」→ ✅「大多数人养虾，养的其实是焦虑」）

### 列举
- 用 `-` 短 bullet，每条不超过2行
- 列举超过3条时，重新考虑是否可以合并或叙述

### 视觉分隔
- 每个主体节之间加 `---`
- 配图位置用 blockquote 标注

### 禁止项
- ❌ 连续5行以上无换行
- ❌ 表格（微信不渲染）
- ❌ 超过3级标题嵌套
- ❌ 堆砌 emoji（每节最多1个）
- ❌ 鸡汤结尾（「相信自己」「加油」类）

---

## 标题公式（必须生成3-5个候选，让用户选）

- 「数字型」：X个让你XXX的方法/认知
- 「反常识型」：我以为XXX，结果发现XXX
- 「身份代入型」：作为XXX，我必须说…
- 「悬念型」：关于XXX，有件事没人告诉你
- 「预测型」：未来X年，XXX会彻底改变
- 「对比型」：XXX的人 vs XXX的人，差距在哪里

---

## 爆款元素检查清单

- [ ] 开头3行内有钩子（悬念/反直觉/身份代入）
- [ ] 每个副标题都有独立观点（非流水账）
- [ ] 至少1个反直觉观点，且被充分展开（不只一句带过）
- [ ] 最强观点在文章前1/3或结尾，不在中间被埋
- [ ] 结尾有互动引导（让读者想评论/转发的那句话）
- [ ] 全文无超过5行连续段落
- [ ] 金句单独成段

## 配图建议

- 封面图：900x383，主题相关，有视觉冲击
- 正文插图：900x600，每1500字放一张
- Prompt 必须英文，风格词放最后，同篇文章风格统一
- 禁止在 Prompt 中要求图片含有文字
