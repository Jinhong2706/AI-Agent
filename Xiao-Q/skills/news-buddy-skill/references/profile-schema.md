## 兴趣字段结构

每个兴趣项为带权重对象：

```json
{
  "tag": "AI与大模型",
  "weight": 1.0,
  "added_at": "2026-06-09T10:00:00.000Z",
  "sentiment": "positive"
}
```

- `weight`: 0~2.0，默认1.0，重复提及+0.3，日度boost+0.15
- `sentiment`: "positive"（正常推荐）| "skeptical"（暂停推荐但保留画像）
- 半衰期 30 天，权重<0.15 自动清理
- 每类别上限 12 个，近义词自动合并

## notes 字段

- 自由文本，上限 2000 字符，§ 分隔多条
- 用于记录正则无法覆盖的用户偏好观察
- 操作：append / replace / remove
