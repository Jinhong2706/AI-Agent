# 使用示例

## 示例一：首次安装

**（skill 被安装，AI 立即执行三件事）**

**AI**：
```
嗨！AI新闻搭子已经就位 👋

为了给你推最相关的新闻，想简单了解一下你：
- 你是做什么工作的？（比如程序员、金融、教师、自由职业……）
- 坐标哪个城市？
- 平时比较关注哪些领域？（比如 AI、投资理财、教育、体育……）

随便说几句就行，说多说少都可以，我会自动理解的～
```

**用户**：我在深圳做前端开发，有房贷，关注 AI 和副业机会

**AI 处理流程**（对用户不可见）：
1. `--init-profile --input "我在深圳做前端开发，有房贷，关注 AI 和副业机会"`
2. 画像构建完成
3. 使用 automation_update 直接创建两个定时任务

**AI**：
```
收到！我已经了解你的情况了。已经帮你设置好了：

📰 每天早上 9 点 — 自动推送跟你相关的新闻
🔄 每天晚上 12 点 — 自动更新你的兴趣画像

明天早上 9 点你就会收到第一条推送了。
现在要先看看今天有什么跟你相关的新闻吗？
```

## 示例二：对话深入 + 画像迭代

**用户**：第 1 条展开说说？最近也在看 Rust 想转方向

**AI 信号检测**（对用户不可见）：
- "最近也在看 Rust 想转方向" → 高置信度信号（明确表达持续兴趣+行动意图）
- 触发画像更新：添加 "Rust编程" 到 hard_interests

**AI**：
Vite 7 这次最大的变化是……（展开深度解读）

你说在看 Rust 是吗？其实这个方向跟前端工具链结合越来越紧了，SWC、Turbopack 底层都是 Rust 写的。如果你已经有 JS/TS 基础，学 Rust 做工具链方向是条不错的路。

---

**用户**：区块链那条不用看了，我对这个不感兴趣

**AI 信号检测**（对用户不可见）：
- "不感兴趣" → 明确否定信号
- 触发画像更新：从 hard_interests 移除 "区块链"

**AI**：
好的，以后少推这类内容。

## 深度解读输出格式参考

```
📖 深度解读：{card_title}

📌 为什么跟你有关
{结合用户画像，解释这条新闻对用户个人的具体影响}

💡 表面之下
{大多数人没注意到的角度、矛盾点、深层因果链}

📊 事件全貌
{完整事件详述：关键数据、时间线、各方反应}

✅ 你可以做什么

🔹 本周行动：
{具体可执行的短期建议}

🔹 中长期思考：
{战略层面的思考方向}

🔗 原文：{url}
{如果是财经内容：⚠️ 以上仅为信息整理，不构成投资建议}
```
