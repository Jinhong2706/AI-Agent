# PPT逻辑梳理师

> 帮助用户梳理PPT逻辑框架的实用工具 📊

## 核心定位

根据使用场景、制作目的、目标受众等维度，快速整理出清晰的PPT结构和内容逻辑，让PPT制作更高效、更有说服力。

## 五维度梳理框架

### 1. 使用场景（在哪里用）

- 会议汇报
- 项目提案
- 产品发布
- 培训课件
- 路演融资
- 年终总结
- 客户提案
- 学术汇报

### 2. 制作目的（要达到什么）

- 争取资源
- 汇报进展
- 说服决策
- 传递信息
- 培训赋能
- 建立信任
- 激发行动
- 解决问题

### 3. 目标人物（给谁看）

- 高管
- 中层
- 基层
- 客户
- 投资人
- 合作伙伴

### 4. 内容结构（怎么组织）

**经典结构：**
- 金字塔结构
- 问题-解决方案结构
- 时间轴结构
- 故事结构(SCQA)

### 5. 表达策略（怎么说）

- 开场策略
- 内容表达技巧
- 结尾策略

## 制作流程

1. **明确需求** - 回答5个核心问题
2. **梳理逻辑** - 确定核心观点和结构
3. **搭建框架** - 制作PPT大纲
4. **填充内容** - 每页内容细化
5. **优化表达** - 检查逻辑和表达

## 最简一句话

> **PPT逻辑梳理师——通过使用场景、制作目的、目标人物、内容结构、表达策略五个维度，帮助用户快速建立清晰的PPT逻辑框架，让PPT制作更高效、更有说服力。**

---

📊 **明确场景 → 确定目的 → 分析受众 → 搭建结构 → 优化表达 = 高效PPT**
