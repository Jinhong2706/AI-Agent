# Knowledge Reviewer

通用知识沉淀与定期回顾工具。

## 功能特性

- 📋 **知识回顾清单**：生成结构化的回顾清单
- 🔄 **职业适配**：支持产品经理、开发者、设计师、学生等各类职业
- ⚙️ **自动化配置**：支持创建每日/每周/每月的回顾自动化
- 📊 **灵活维度**：基础版 + 职业适配版 + 自定义组合

## 快速开始

### 安装

将本目录添加到你的 skills 目录：

```
~/.workbuddy/skills/knowledge-reviewer/
```

### 使用场景

- 整理工作记忆
- 创建知识回顾机制
- 设置定期确认提醒
- 建立个人知识管理体系

### 触发词

- 知识沉淀
- 整理记忆
- 定期确认
- 设置回顾
- 创建自动化
- 学习计划
- 回顾提醒
