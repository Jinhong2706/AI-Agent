# 短视频爆款文案创作技能包

整合了曹俊在短视频文案创作中沉淀的所有方法论和执行标准，是新媒体运营文案创作的核心技能。

---

## 一、创作优先级规范（强制执行）

- **结构层**：爆款元素、标题、脚本结构框架 → 优先使用**薛辉新媒体运营方法论**
- **内容层**：
  - 文案开篇钩子：同时参考**埃德蒙开篇技巧（情绪钩子）** + **薛辉开篇技巧（开篇钩子模板）**
  - 文案正文、衔接技巧、价值升华、结尾技巧 → 优先选用**埃德蒙文案技巧核心手册**

## 二、薛辉新媒体运营方法论

### 2.1 八大爆款元素

选题必须融入至少1个爆款元素，确保自带流量属性：

| 元素 | 核心逻辑 | 举例 |
|------|----------|------|
| **成本** | 用户对价格/花费敏感 | "一年换一次膜亏了多少""这套设备省了3万" |
| **人群** | 圈定特定群体产生代入感 | "种葡萄的注意了""做水处理的老铁" |
| **猎奇** | 满足好奇心 | "25年不换工艺的秘密""为啥大棚都是蓝色的" |
| **反差** | 颠覆认知制造冲突 | "越贵的膜反而越省""打药多的棚反而膜坏得快" |
| **最差** | 展示最坏结果触发恐惧 | "膜选错一季白干""边膜先破的通病" |
| **头牌** | 展示最强/最好的一面 | "云南本地25年老厂""A级花翻了一倍" |
| **怀旧** | 唤起情感记忆 | 马总观后感、潮汕人下南洋 |
| **荷尔蒙** | 触发情绪反应 | "你信不信""想想都可怕" |

### 2.2 爆款标题公式

标题必须使用以下公式之一，禁止平淡无亮点的标题：

- **数字型**：用具体数字制造冲击（"3年不换膜""省了2000块"）
- **疑问型**：用提问引发好奇（"你算过亏了多少吗""你睡得着吗"）
- **对比型**：用对比制造反差（"越贵越省""别人A级你B级"）
- **颠覆型**：颠覆常识制造冲突（"不是肥不够是膜不对""最怕的不是贵是便宜"）

### 2.3 16种开篇钩子模板

开篇3秒必须使用钩子调动用户情绪，禁止平淡开头：

1. **痛点钩子**：直击用户最头疼的问题
2. **反问钩子**：用反问句引发思考
3. **利益钩子**：直接亮出利益点
4. **悬念钩子**：留下悬念引发好奇
5. **反差钩子**：展示颠覆认知的事实
6. **场景钩子**：带入具体场景引发共鸣
7. **数据钩子**：用数据制造冲击
8. **故事钩子**：用故事开头吸引注意力
9. **警告钩子**：发出警告触发恐惧
10. **好奇钩子**：激发好奇心
11. **共鸣钩子**：说出用户心声
12. **挑战钩子**：挑战用户认知
13. **秘密钩子**：暗示有秘密要揭露
14. **对比钩子**：用对比制造反差
15. **亲身经历钩子**：用第一人称经历增加可信度
16. **行业揭秘钩子**：揭开行业内幕

### 2.4 四类脚本结构（严格对应，禁止同质化）

**教知识型**：
开篇钩子 → 引出问题 → 要点讲解 → 价值升华 → 引导互动

**聊观点型**：
开篇钩子 → 观点阐述 → 案例支撑 → 价值升华 → 引导互动

**讲故事型**：
开篇钩子 → 背景铺垫 → 冲突展开 → 高潮转折 → 价值升华 → 引导互动

**晒过程型**：
开篇钩子 → 过程展示 → 亮点强调 → 信任背书 → 引导转化

## 三、埃德蒙文案技巧核心手册

### 3.1 情绪钩子（开篇）

- 开篇必须有情绪，不能平铺直叙
- 情绪类型：好奇、焦虑、共鸣、利益、恐惧、愤怒、惊喜
- 技巧：用一个具体的场景/细节/反常识事实触发情绪，而非抽象道理

### 3.2 正文技巧

- **有画面感**：用具体细节代替抽象描述（不说"质量好"，说"三年了还透亮"）
- **有情绪**：文案中要有情绪波动，不能像说明书
- **有记忆点**：每条文案至少有一个让人记住的细节或金句
- **口语化**：像说话不像写文章，"你琢磨琢磨""大伙儿""这事儿"

### 3.3 价值升华

- 聊观点/讲故事类脚本必须在正文结束、结尾互动前增加价值升华环节
- 升华方式：把具体信息提升到人生感悟/行业洞察/情感共鸣层面
- 示例：不说"选好膜省钱"，说"好膜不是用着贵，是用着省"
- 不喊口号，用克制的方式传递力量

### 3.4 结尾技巧

- 结尾必须有明确的行动引导，提升互动率
- 引导方式：提问、邀请评论、引导关注、引发讨论
- 避免硬广式结尾（"评论区留言""欢迎联系"等）
- 置顶评论配合主文案做互动引导

## 四、B端工厂开头文案公式

适用于源头工厂/设备类B端账号：

| 公式 | 开头句式 | 情绪触发 | 适用场景 |
|------|----------|----------|----------|
| **公式1** | "你信不信…" | 自信/挑战 | 源头工厂实力、设备效果（偏卖货导向账号） |
| **公式2** | "真没想到…" | 惊喜/反差 | 性价比、选型避坑（偏卖货导向账号） |
| **公式3** | "想想都可怕…" | 恐惧/后果 | 不选源头的代价、选错设备的后果（偏信任导向账号） |
| **公式4** | "千万不要…" | 警告/紧迫 | 避坑、防踩雷（偏信任导向账号） |

**应用规则**：
- 公式只用于**开头钩子**，正文保持B端专业内容，不能变成纯口播喊话
- 4个公式穿插使用，避免同质化，不要每条都套同一个公式
- 卖货导向账号偏公式1+2，信任导向账号偏公式3+4

## 五、内容创作底层哲学——好故事才是最大的流量密码

### 核心原则

1. **好故事，永远不愁没观众** — 技术是手段不是目的，故事才是灵魂
2. **力量来自真实，来自与观众生活的共振** — 真实细节 > 特效/包装
3. **观众能分清**哪些是看完就忘的刺激，哪些是久久难以平静的触动
4. **把目光从流量数据转向真实生活** — 不是追逐热点制造噱头，而是发现日常生活中朴素、动人、有力量的东西
5. **真正的对手不是技术，而是躲在技术背后的惰性与敷衍**

### 对短视频创作的6条具体指导

1. **禁止用公式代替创作**：爆款元素、钩子模板是工具不是目的，不能因为"公式管用"就堆砌公式，最终打动人的永远是真实的情感
2. **真实 > 编造**：创作者的真实感受比编造的故事更能引起共振
3. **与观众共情**：不是"我讲你听"的单向输出，是发现观众生活中的痛点/共鸣点，让他们觉得"这个人懂我"
4. **克制的叙事打动人**：最有力的瞬间往往是最克制的，比任何煽情都更戳人
5. **笨功夫击败流水线**：在一个AI批量生产的时代，用心的"笨"反而成为稀缺价值
6. **避免惰性创作**：每次创作都要问——这条内容里有我真实的思考吗？

### 源起
- 触发：《给阿嬷的情书》（潮汕方言电影，首日排片不足4%，口碑逆袭）+ 微短剧《ENEMY》（两个年轻人自编自导，抖音播放7亿+）
- 两部作品共同证明：没有流量明星、没有大制作，好故事本身就是最大的竞争力

## 六、爆款文案创作强制四步流程

### 第一步：前置校验
- 先明确内容所属的新媒体运营环节
- 确认内容红线、品牌人设、用户画像等基础约束
- 读取客户项目背景说明获取上下文

### 第二步：选题 & 标题创作
- 选题必须融入薛辉八大爆款元素，确保自带流量属性
- 标题必须使用薛辉爆款标题公式，禁止平淡无亮点的标题
- 按真实场景/痛点组织，禁止同一卖点来回倒腾（如V2大田农业的错误）

### 第三步：结构匹配（严格对应，禁止同质化）
- 根据内容类型选择对应脚本结构：教知识型/聊观点型/讲故事型/晒过程型
- 每条文案场景、痛点、角度必须独立，不同文案之间不能"换汤不换药"

### 第四步：内容优化
- 开篇必须使用钩子模板，第一秒调动用户情绪
- 正文使用埃德蒙文案技巧：有情绪、有画面感、有记忆点
- 聊观点/讲故事类必须有价值升华环节
- 结尾有明确互动引导 + 置顶评论配合

## 七、爆款文案质量验收标准（未达标禁止输出）

1. ✅ 标题有爆款属性，包含至少1个爆款元素，用户看到标题就有点击欲望
2. ✅ 开篇3秒能抓住注意力，有明确的情绪调动（好奇/焦虑/共鸣/利益等）
3. ✅ 结构完整，符合对应脚本框架要求
4. ✅ 有价值升华环节，内容不局限于信息本身，能传递观点或情感
5. ✅ 结尾有明确互动引导，提升完播/点赞/评论/转发数据
6. ✅ 同一批次文案之间场景、痛点、角度不重复，没有"换汤不换药"


## 九、内容红线

- 不贬低同行
- 不虚假宣传
- 不涉及敏感话题
- 产品数据需基于实际情况
- 客户之间内容隔离，禁止对比踩竞
- 真实 > 编造，不用AI编造代替用户真实感受
