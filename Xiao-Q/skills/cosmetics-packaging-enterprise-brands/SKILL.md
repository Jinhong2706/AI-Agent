# 海外化妆品：大型集团与成熟品牌线（印刷包装潜客）

## 适用场景

- 目标为**海外**法人实体，品类为化妆品（护肤、彩妆、香氛等）。
- 我方能力：**印刷包装**（含服务过欧莱雅等集团级客户的经验，可在话术中对标，不虚构案例细节）。
- 产出：**可核验**的公司与品牌清单，便于后续商务触达（触达方式与合规由业务自行执行）。

## 公开联系邮箱与联系人（本 Skill 能做什么）

- **可以**：从**客户主动公开**的网页或 PDF 中摘录**联系邮箱**与**具名联系人**，来源须可引用（例如官网 Contact、Press/Media、投资者关系 IR、法务/隐私页公示邮箱、新闻稿文末媒体联络人、展会官方展商页公示邮箱）。
- **不可以**：推断或编造邮箱与电话；从非公开数据库批量抄录个人邮箱；抓取需登录或明显违反平台条款的页面；将招聘页个人姓名当作采购联系人。
- **结果预期**：许多集团仅提供**联系表单**或总机，无对外采购邮箱，表中对应列填「未公示」并附上**联系入口 URL**。

## 用户应提供的输入

- 目标**国家或区域**（例如：西欧、北美、具体国家列表）。
- **品类侧重**（可选）：护肤 / 彩妆 / 香氛 / 全品类。
- **排除条件**（可选）：已合作竞品、无进口业务的纯本土小店等。

## 研究流程（按顺序执行）

1. **锁定集团与公司实体**
   - 从公开资料整理「母公司 / 上市主体 / 主要运营公司」名称与注册地；区分**品牌名**与**法律实体**。
   - 优先使用：集团官网投资者关系、年报、维基与百科类二次摘要仅作线索，**必须以官网或监管披露为准**。

2. **用高信号来源扩充名单**
   - **展会名录**：Cosmoprof、in-cosmetics 等公开 exhibitor / brand 目录（按区域筛选）。
   - **行业协会**：目标国化妆品相关协会的成员或公开名录。
   - **大型零售商品牌索引**：Sephora、Ulta、Douglas、Boots 等站内的品牌列表（用于发现品牌，再回链到官网）。

3. **包装与采购相关信号（每条必须写明证据来源）**
   - 官网 **Sustainability / Responsibility / Packaging** 页面是否提及减塑、FSC、单一材质、可回收等（说明客户对包装有叙事需求）。
   - **Press / News**：新品礼盒、限量系列、节日套装（暗示彩盒与礼盒需求）。
   - **Careers**：采购、包装工程师、可持续相关岗位（仅作**职能存在**线索，不记录具体员工姓名作为联系人）。

4. **公开联系方式（与上列并行，每条潜客做一次）**
   - 依次查看官网：**Contact、Press/Media、Investor Relations、Suppliers/Vendors**（若有）页面。
   - 仅记录页面上**明文写出**的邮箱；若仅有表单，记录页面 URL，邮箱列写「仅联系表单」。
   - **具名联系人**：仅当新闻稿或官网「媒体联络」「投资者联络」等段落**同时给出姓名与职务**时写入；须附来源链接。

5. **核验**
   - 每个条目至少具备：**品牌或公司官网 URL** + **与化妆品相关的公开证据**（产品线、渠道或监管侧公开信息之一）。
   - 剔除无法确认主体国别或无法确认业务性质的条目。

## 输出格式（表格字段固定）

对每条潜客输出一行，列名为：

| 字段 | 说明 |
|------|------|
| 集团/母公司 | 如有 |
| 品牌名 | 对外主品牌 |
| 国家/地区 | 运营或注册地（写清） |
| 公司官网 | 主站 URL |
| 品类 | 护肤/彩妆/香氛等 |
| 主要渠道信号 | 百货、连锁美妆、电商旗舰等（简述+来源） |
| 包装/可持续公开证据 | 摘要+页面或新闻链接 |
| 公开邮箱 | 多个用分号分隔；无则「未公示」或「仅联系表单」 |
| 邮箱与表单来源 URL | 摘抄或填写的页面链接 |
| 公开具名联系人 | 格式：姓名 / 职务 / 来源链接；无则「未公示」 |
| 下一步建议 | 如：供应商门户、IR 页、媒体对接入口 |

## 合规与边界

- 仅整理**公开网页与公开名录**信息；不指导绕过 robots/服务条款的大规模抓取。
- 面向欧盟/英国个人数据与营销时，提醒业务侧遵守 GDPR/PECR 等当地规则（本 Skill 不提供法律意见）。

## 术语统一

- 全程使用「潜客」「公开证据」「官网」等固定用词，避免混用「线索」「客户」造成已合作误解。
