## 商品详情页模板

```
【技能名称】
一句话介绍（不超过20字，像朋友圈标题一样抓眼球）

【详细介绍】
第1段：解决什么痛点（1句话）
第2段：怎么用（用户视角，1-2句）
第3段：为什么专业（背书，1句）
第4段：效果/价值（数字或场景化描述）

【使用示例】
用户输入：「帮我看看我的体质」
技能输出：「你是气阴两虚体质...」（结构化展示）

用户输入：「今天五行有什么要注意的」
技能输出：「今日木气旺盛，宜...忌...」

【定价策略】
建议价格 + 定价逻辑（限时/早鸟/原价）

【用户评价】（预设3条）
⭐⭐⭐⭐⭐ 「用了这个技能之后...」
⭐⭐⭐⭐⭐ 「比我之前找的...」
⭐⭐⭐⭐⭐ 「最厉害的是...」
```

## 定价心理学技巧

### 锚定效应
- 展示「原价199元 → 内测价49元」比直接「49元」更有吸引力
- 永远先展示高价，再展示折扣价

### 捆绑销售
- 「技能+私人咨询」打包，比单独卖技能客单价提升3倍
- 「年卡」比「月卡×12」看起来更划算

### 稀缺性
- 「内测名额仅限20个」「倒计时XX小时」
- 稀缺感提升转化率30-50%

### 免费试用
- 「免费诊断1次」作为钩子，转化订阅
- 赛华佗模式：免费看体质 → 9.9元订阅

## 变现模式对比

| 模式 | 适合技能类型 | 优点 | 缺点 |
|------|------------|------|------|
| 免费+订阅 | 诊断/分析/个性化 | 门槛低，积累用户 | 需要持续输出 |
| 一次性买断 | 工具/生成类 | 现金流好 | 用户用完就走 |
| 免费+增值 | 工具类 | 体验驱动 | 转化难 |
| 课程+社群 | 教育类 | 高客单价 | 制作成本高 |
| B2B授权 | 专业工具类 | 大客户付费 | 销售周期长 |

**推荐优先级：**
1. 诊断类技能 → 免费试用 + 9.9-49.9元/月订阅
2. 生成类技能 → 免费次数 + 付费解锁
3. 工具类技能 → 免费基础 + 私人定制付费
