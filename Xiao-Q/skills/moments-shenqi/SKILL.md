# Skill: 朋友圈神器

## Description

**一句话**：告诉我你想发什么，我帮你搞定文案、配图、排版——3秒出3套方案，越用越懂你。

**解决的痛点**：
- 拍了照片不知道配什么文案 → AI 根据场景 + 你的风格生成
- 文案写好了不知道配几张图 → 智能推荐配图数量和排列
- 想发高级感朋友圈但不会排版 → 一键排版优化
- 每次发圈都要翻小红书找灵感 → 内置热门文案库 + 实时搜索
- 别人写的文案好但不是我的风格 → AI 学习你的风格，生成像你写的文案

**量化价值**：
- 发一条朋友圈从「想30分钟」→「3秒出方案」，提效 **600倍**
- 一次学习，终身适配，越用越懂你的风格
- 覆盖12大场景 × 4种风格 = 48种文案组合，不重复

**适用场景**：
| 场景 | 用户 | 示例 |
|------|------|------|
| 个人社交 | 普通微信用户 | 旅行/美食/日常/自拍发圈 |
| 个人品牌 | 自由职业者/KOL | 打造有辨识度的朋友圈人设 |
| 私域营销 | 微商/社群运营 | 高转化的产品种草文案 |
| 职场社交 | 职场人 | 得体又有趣的工作生活分享 |

## Quick Start（30秒上手）

```
第一步：直接告诉我你想发什么
  → "帮我写一条去三亚旅行的朋友圈"
  → AI 自动生成3套方案

第二步（可选）：上传你以前的朋友圈截图
  → "/moments learn" + 贴截图
  → AI 学习你的风格，后续生成更像你

```

不需要任何安装、配置、API Key。开口就能用。

本 Skill 为纯文本知识型 Skill，**零配置**即可使用。

**依赖工具**（AI Agent 已内置，用户无需操作）：

## Commands

### 1. `/moments create <场景描述>` — 生成朋友圈方案

**输入**：场景描述（如"去西双版纳旅行""吃了一顿超棒的日料""深夜emo了"）
**输出**：3套候选方案（文案 + 配图 + 排版 + 定位建议）

**执行步骤**：

```
Step 1: 解析场景 → 匹配12大分类
        （旅行/美食/日常/自拍/emo/治愈/高级感/搞笑/节日/工作/运动/恋爱）
Step 2: 读取风格画像 {workspace}/.moments-creator/style_profile.json（若存在）
Step 3: 读取 references/xiaohongshu_moments_copywriting.md 中对应分类
Step 4: web_search "小红书 朋友圈文案 [场景关键词]" 补充最新热门
Step 5: 生成3套候选方案：
        - 方案A：贴合用户个人风格（若已学习）/ 默认极简风
        - 方案B：小红书当前热门风格
        - 方案C：高级感/差异化风格
Step 6: 每套方案输出：
        📝 文案正文（标注字数）
        📷 配图数量+布局+每张图内容建议
        🎨 色调/滤镜建议
        📍 定位建议
        ⚠️ 排版检查（折叠风险/emoji数量）
Step 7: 询问用户选择、调整或生成配图
```

**迭代优化**（AI 原生能力）：
用户选择方案后可以继续对话调整，AI 理解上下文自动迭代：
```
用户：方案A不错，但再文艺一点
AI：（理解"在方案A基础上增加文艺感"，保持极简结构但换更有意境的词）

用户：文案OK了，帮我生成配图
AI：（调用 image_gen 根据文案和色调建议生成配图）
```

**输入输出示例**：

```

   ① 傣族寺庙  ② 热带植物  ③ 傣族纹样细节

   ⑦ 拍立得    ⑧ 蝴蝶特写  ⑨ 日落
🎨 暖绿色调统一  📍 西双版纳傣族自治州
⚠️ 2字不折叠 ✅

━━━ 方案B · 小红书热门 ━━━
📝 文案：从北方的冬逃到南方的夏（14字）
📷 9张九宫格（暖橙色调）
📍 西双版纳傣族自治州
⚠️ 14字不折叠 ✅
```

**失败处理**：
- 场景模糊（如"发个朋友圈"）→ 追问"什么场景？旅行/美食/日常/心情？"
- web_search 失败 → 仅用本地参考库，不中断
- 风格画像不存在 → 提供多风格候选，提示可用 `/moments learn` 学习

### 2. `/moments learn` — 风格学习（增量学习）

**输入**：1-5张朋友圈截图 或 手动粘贴的历史文案
**输出**：风格画像报告 + 保存/更新画像文件

**核心机制 — 增量学习**：
- 第一次 learn → 创建风格画像
- 后续每次 learn → **追加样本，画像越来越精准**
- 分析维度随样本量增长自动深化

**执行步骤**：

```
Step 1: 接收截图或文案文本
Step 2: 图片 → OCR提取文案 + 分析配图（色调/数量/内容类型）
Step 3: 分析文案特征：
        - 字数分档（极简2-8 / 短句10-25 / 标准30-80 / 小作文80-150）
        - emoji 频率和偏好品种
        - 情感基调（治愈/幽默/文艺/高冷/日常/正能量）
        - 句式模式（排比/对比/极简/哲理）
        - 话题标签/定位习惯
Step 4: 分析配图特征：
        - 偏好张数（1/4/9）
        - 色调（暖/冷/黑白）
        - 内容类型（风景/人像/混合叙事）
        - 自拍比例
Step 5: 若已有画像 → 增量合并（新样本权重融合旧画像）
        若无画像 → 创建新画像
Step 6: 保存到 {workspace}/.moments-creator/style_profile.json
Step 7: 输出画像摘要 + 提示"再多学几条更精准"
```

**输入输出示例**：

```
用户：/moments learn
[上传2张新截图]

AI：
📊 风格画像更新！（累计学习 3 条朋友圈）

📝 文案风格：极简型（平均3字）— 置信度：高 ⬆️
😊 emoji：不使用 — 置信度：高 ⬆️
🎨 基调：松弛高冷 — 置信度：中 → 高
📷 配图：9张九宫格为主 — 置信度：高 ⬆️
🌈 色调：暖色调 — 置信度：高
📍 定位：总是开启
🏷️ 标签：不使用

📈 与上次对比：emoji"不使用"的判断更确定了（3/3条都没用emoji）
💡 建议再学习2-3条不同场景的朋友圈，让我更全面地了解你
```

**失败处理**：
- 图片模糊 → "截图不太清晰，能重新截一张吗？或者直接把文案文字贴给我"
- 未上传任何内容 → "请上传你以前的朋友圈截图，或直接粘贴你发过的文案"

### 3. `/moments ref <场景>` — 小红书热门参考

**输入**：场景关键词（如"旅行""美食""emo""高级感"）
**输出**：热门文案列表（按风格分组）+ 万能句式模板

**执行步骤**：

```

Step 2: 读取 references/xiaohongshu_moments_copywriting.md 对应分类
Step 3: web_search "小红书 朋友圈文案 [关键词]" 补充最新
Step 4: 按风格分组输出 + 附万能句式模板
```

---

### 4. `/moments layout` — 排版优化

**输入**：文案草稿 + 图片数量（可选）
**输出**：排版诊断 + 优化后文案

**执行步骤**：

```
Step 1: 字数统计 → 折叠风险（>200字=高风险，>150字=中风险）
Step 2: emoji 检查 → 数量/风格统一性
Step 3: 段落结构 → 是否需要空行/分段
Step 4: 配图数量 → 推荐1/4/6/9
Step 5: 输出诊断报告 + 优化版文案（标注改了什么）
```

---

### 5. `/moments pic <文案或场景>` — AI 生成配图（新增）

**输入**：已确定的文案或场景描述
**输出**：AI 生成的朋友圈配图

**执行步骤**：

```
Step 1: 解析文案/场景 → 确定图片主题
Step 2: 读取风格画像 → 确定色调偏好（暖色/冷色/统一滤镜风格）
Step 3: 根据推荐配图数量，构造 image_gen prompt：
        - 融入场景关键元素（地点、食物、建筑等）
        - 匹配色调偏好
        - 适配朋友圈展示比例（正方形1:1为佳）
Step 4: 调用 image_gen 生成图片
Step 5: 输出图片 + 建议排列顺序 + 色调微调建议
```

**输入输出示例**：

```
用户：/moments pic 西双版纳热带风景

AI：
🎨 根据你的暖绿色调偏好，生成以下配图：
[图片1] 热带雨林中的傣族金塔
[图片2] 荷花池特写
[图片3] 星光夜市夜景
[图片4] 傣族美食俯拍

📐 建议排成2×2田字格，色调已统一为暖绿
💡 如需九宫格全套（9张），说"再生成5张"
```

**失败处理**：
- image_gen 不可用 → "AI配图功能暂时不可用，建议先拍照或从相册选图，我帮你排版"

## AI-Native Features（技术巧思）

### 1. 增量风格学习
每次 `/moments learn` 不是覆盖而是追加。样本越多，画像维度越精细，置信度越高。AI 会主动告诉你"再学几条更准"。

### 2. 上下文感知迭代
用户说"再文艺一点""短一些""换个风格"时，AI 理解是在当前方案基础上调整，而非重新生成。支持多轮对话打磨直到满意。

### 3. 多模态协同
- 📸 截图输入 → OCR + 视觉分析 → 风格提取
- 📝 文本输入 → NLP 分析 → 风格分类
- 🎨 图片输出 → image_gen → AI生成配图
- 🔍 实时搜索 → web_search → 获取最新热门文案

### 4. 智能场景识别
用户不需要说"/moments create"，只要自然语言提到"朋友圈""发圈""不知道配什么文案"就自动触发。

### 5. 排版安全检查
每次生成都自动检查：字数折叠风险、emoji合理性、配图数量规整性，确保发出去不翻车。

## Constraints & Best Practices

1. **原创性**：参考小红书文案风格和句式进行改写组合，不直接抄袭
2. **配图规则**：严格 1/4/6/9 张，绝不推荐 2/3/5/7/8（微信排列不整齐）
3. **防折叠**：文案 ≤150字建议，>200字强制警告
4. **风格优先**：有画像 → 适配个人风格；无画像 → 多风格候选
5. **降级容错**：web_search/image_gen 失败时退化为本地能力，不中断服务
6. **隐私保护**：截图仅用于风格分析，不存储原图，只保存风格特征 JSON
