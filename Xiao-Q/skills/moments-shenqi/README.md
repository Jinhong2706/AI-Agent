# 朋友圈神器

> 告诉我你想发什么，3秒出3套方案。越用越懂你。

## 痛点

- 拍了照片不知道配什么文案
- 文案写好了不知道配几张图
- 想发高级感朋友圈但不会排版
- 每次发圈都要翻小红书找灵感
- 别人的文案好但不是自己的风格

## 解决方案

| 能力 | 说明 |
|------|------|
| 🎨 风格学习 | 上传朋友圈截图，AI 学习你的风格，越用越准 |
| 📖 热门参考 | 内置小红书热门「朋友圈文案」12大场景库 |
| 📐 图文排版 | 文案 + 配图数量 + 布局的完整方案 |
| 🖼️ AI配图 | 根据文案和风格偏好生成配图 |
| 🔄 迭代优化 | "再文艺一点""短一些"，多轮对话打磨 |

## 30秒上手

```
第一步：直接说 → "帮我写一条去三亚旅行的朋友圈"
第二步（可选）：上传截图 → "/moments learn" 学习你的风格
第三步：挑一个方案，发！
```

## 命令

| 命令 | 说明 |
|------|------|
| `/moments create <场景>` | 生成3套朋友圈方案 |
| `/moments learn` | 上传截图学习风格（支持多次增量学习） |
| `/moments ref <场景>` | 查看小红书热门朋友圈文案参考 |
| `/moments layout` | 排版优化诊断 |
| `/moments pic <描述>` | AI生成配图 |

## 技术亮点

- **增量风格学习**：每次 learn 追加样本，画像越来越精准
- **上下文迭代**：多轮对话打磨，"再文艺一点"立即调整
- **多模态协同**：截图OCR + 文案NLP + AI生图 + 实时搜索
- **排版安全检查**：自动检测折叠风险和配图数量规整性
