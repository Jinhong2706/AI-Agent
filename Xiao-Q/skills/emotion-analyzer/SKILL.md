# Emotion Analyzer v4.3 - 智能情绪分析

快速、准确的命令行情绪分析工具，支持多语言、网络用语、表情符号，**新增情感强度评分（1-10分）**，**修复英文强调词识别**。

## 快速开始

分析文本情绪：

```bash
python scripts/emotion_analyzer.py "你的文本"
```

示例：

```bash
python scripts/emotion_analyzer.py "今天真开心！"
python scripts/emotion_analyzer.py "fucking happy!"
python scripts/emotion_analyzer.py "This is damn good!"
```

## 功能特性

- ✅ **多语言支持**：中文、英文及混合文本
- ✅ **网络用语识别**：666、yyds、牛逼、SB 等
- ✅ **表情符号分析**：😊❤️😢💔 等
- ✅ **歧义词处理**：智能上下文感知（如"卧槽"根据语境判断）
- ✅ **英文强调词识别**：正确处理 "fucking happy" → 正面（v4.3 修复）
- ✅ **情感强度评分**：1-10分制，带可视化条形图和强度等级
- ✅ **详细输出**：各维度得分、匹配词汇、分析细节
- ✅ **高准确度**：置信度评分 + 智能分析算法

## 情绪类型

工具可识别以下情绪类型：

| 类型 | 说明 | 示例 |
|------|------|------|
| 正面 | 积极、愉快的情绪 | "开心"、"happy"、"fucking happy" |
| 负面 | 消极、不满的情绪 | "难过"、"shit"、"Fuck this!" |
| 中性 | 无明显情绪倾向 | "还行"、"ok" |
| 惊讶 | 意外、震惊的情绪 | "哇"、"竟然" |
| 疑问 | 询问、不确定的语气 | "吗？"、"why？" |

## 情感强度评分（1-10分）

v4.0 新增功能！根据以下因素综合计算：

- **词汇强度**：每个情绪词有预设强度（1-10）
- **强度修饰词**：非常、超级、有点等
- **英文强调词**：fucking、damn 等（正确识别为强调，v4.3 修复）
- **标点符号**：感叹号、问号数量
- **重复强调**：重复字符（如"！！"）
- **文本长度**：长短文本的调整

### 强度等级

| 分数 | 等级 | 说明 |
|------|------|------|
| 9-10 | 极强 🔥 | 情感非常强烈 |
| 7-8  | 强烈 💪 | 情感明确且强烈 |
| 5-6  | 中等 😐 | 情感适中 |
| 3-4  | 轻微 😕 | 情感较淡 |
| 1-2  | 微弱 🤔 | 情感很弱 |

## 输出说明

运行后会显示：

1. **情绪类型**：最终判断的情绪
2. **置信度**：0-1 之间的可信度评分
3. **情感强度**：1-10分 + 可视化条形图 + 强度等级
4. **各维度得分**：所有情绪维度的得分（带可视化条形图）
5. **匹配词汇**：命中的情绪词列表（标注强调词、歧义词）
6. **分析细节**：基础强度、强度修饰、否定检测、标点符号等

## 英文强调词处理（v4.3 修复）

工具现在能正确识别英文强调词：

- ✅ **"fucking happy"** → 正面（fucking 作为强调词）
- ✅ **"damn good"** → 正面（damn 作为强调词）
- ✅ **"Fuck this!"** → 负面（单独使用，表达负面情绪）
- ✅ **"This is shit"** → 负面（shit 是负面词）

判断逻辑：
1. 检测 "fucking/damn" 等词
2. 检查后面是否跟着正面词（happy/good等）
3. 如果是 → 识别为正面强调
4. 如果否 → 识别为负面表达

## 歧义词处理

工具会自动检测和处理歧义词（如"卧槽"、"我靠"等）：

- 根据上下文词汇判断（如"卧槽！这么厉害！"→ 正面）
- 根据标点符号辅助判断
- 输出中会标注"⚠️ 歧义词检测"提醒

## 技术实现

- **词典匹配**：预定义情绪词典（正面/负面/中性/惊讶/疑问），每个词有强度值
- **单词边界匹配**：使用 `\b` 边界，避免子串误匹配（v4.3 优化）
- **上下文分析**：检测歧义词和强调词周围的语境词汇
- **强度修饰**：识别"非常"、"有点"等修饰词及其倍率
- **否定处理**：处理"不"、"没"等否定词
- **标点分析**：感叹号、问号的情绪倾向和强度影响
- **重复字符**：检测强调性重复（如"666666"、"！！！"）
- **综合评分**：加权计算情感强度（1-10分）

## 注意事项

- 工具基于关键词匹配，对复杂语境的理解有限
- 非常短的文本（<5字符）可能判断为中性
- 无情绪词的重复文本会判定为中性
- 置信度低于 0.5 时结果仅供参考
