# 哼唱猜歌游戏

## 概述

这个技能提供**纯哼唱猜歌**游戏功能。系统内置流行+经典歌曲歌单，直接用语音哼唱旋律，用户根据旋律猜歌名和/或歌手。不提供歌词，完全靠音乐记忆力！

## 游戏流程

### 1. 开始游戏

当用户说"开始猜歌"、"玩猜歌游戏"、"哼唱猜歌"等，执行：

1. 从 `references/song_database.md` 随机选择一首歌
2. **直接用 TTS 哼唱旋律**（不显示歌词！）
3. 询问用户猜歌名和/或歌手

**示例：**
```
🎵 猜歌游戏开始！第1题

[语音哼唱：啦~啦~啦~啦~啦~，啦~啦~啦~啦~]

请猜：歌名是？歌手是？
```

### 2. 语音哼唱功能

**核心玩法：所有题目都用哼唱**

**哼唱方式：**
- 使用 `tts` 工具生成语音
- 哼唱内容：用"啦~"或"嗯~"等无意义音节
- 尽量模仿原曲的：
  - 音高变化
  - 节奏快慢
  - 旋律特征

**重要原则：**
- ❌ 绝不透露歌词
- ❌ 绝不透露歌名
- ✅ 只哼旋律，让别人听不出歌词

**调用示例：**
```python
# 轻快歌曲
tts("啦~啦~啦~啦~，啦~啦~啦~啦~啦~")

# 抒情歌曲
tts("啦~啦~啦~，啦~啦~啦~，啦~啦~啦~")

# 节奏感强的歌曲
tts("啦！啦！啦！啦~啦~，啦！啦！啦！啦~啦~")
```

### 3. 用户回答

- 如果用户答对歌名（或歌手，根据要求），回复正确反馈
- 如果答错，给予鼓励并提供正确答案
- 询问是否继续下一题

**示例：**
```
✅ 答对啦！
歌名：《七里香》
歌手：周杰伦

要继续猜下一首吗？
```

### 4. 提示功能

当用户输入"提示"、"不知道"等，提供以下提示：

**提示类型：**
- 🎤 **歌手提示**：显示歌手名字的首字母或字数（如"周X伦（3个字）"）
- 📝 **歌名提示**：显示歌名的长度或首字母（如"3个字，首字是'七'"）
- 🏷️ **风格提示**：提示歌曲风格（流行/摇滚/民谣/中国风等）
- 📅 **年代提示**：提示发行年代（如"2000年代"、"90年代经典"）
- 🔄 **重新哼唱**：再次用 TTS 哼唱旋律
- 💡 **难度降级**：如果实在猜不出，可以显示1-2句歌词（作为最后的提示）

**示例：**
```
用户："给我个提示"
你："💡 提示：这首歌是周X伦（3个字）唱的，2000年代作品"

用户："还是不知道，再唱一次"
你：[重新调用 tts 哼唱]
你："🎵 再听一次旋律！"

用户："投降了"
你："💡 最终提示：歌词开头是'窗外的麻雀...'"
```

### 5. 游戏设置

支持的可选设置：
- **曲目数量**：用户可以指定一轮猜几首歌（默认无限）
- **难度选择**：
  - 简单：哼唱完整的副歌部分（8-10秒）
  - 普通：哼唱副歌片段（4-6秒）
  - 困难：只哼唱前奏或旋律片段（2-3秒）
- **猜什么**：歌名、歌手、或两者都猜
- **提示次数**：限制提示次数增加挑战性

## 歌单管理

歌曲数据库位于 `references/song_database.md`，包含：
- 歌曲名称
- 歌手
- **哼唱脚本**（用于TTS的完整"啦~"版本）
- 歌曲风格/年代标签
- 难度分级（根据旋律辨识度）

歌单特点：
- 混合流行和经典歌曲
- 无上限，持续更新
- 涵盖华语、粤语等中文歌曲
- 优先选择旋律辨识度高的歌曲

**游戏建议：**
- 哼唱时尽量模仿原曲的节奏和情感 🎵
