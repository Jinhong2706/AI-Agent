# 歌曲数据库（纯哼唱版）

这个文件包含所有可用于**纯哼唱猜歌**的歌曲。

## 使用说明

- 每首歌包含：歌名、歌手、年代、风格、**哼唱脚本**（3个难度级别）
- **不提供歌词**，完全靠旋律猜歌
- 哼唱脚本：用"啦~"或"嗯~"等无意义音节，模仿原曲旋律
- 随机出题时，从这些歌曲中选择

## 如何添加新歌曲

1. 按照上面的格式添加新的歌曲条目
2. 确保包含：歌名、歌手、年代、风格、**哼唱脚本**（3个难度级别）
3. **哼唱脚本要求**：
   - 用"啦~"描述整段旋律
   - 标注时长（简单10秒、普通6秒、困难3秒）
   - 模仿原曲的节奏、音高、情感
   - 不能用任何歌词！只能用"啦~"或"嗯~"
4. **选择标准**：
   - 旋律辨识度高
   - 副歌部分有特色
   - 避免选择旋律太相似的歌曲

**哼唱脚本示例：**
```
[简单] "啦~啦~啦~啦~啦~，啦~啦~啦~啦~啦~啦~"（完整副歌，10秒）
[普通] "啦~啦~啦~啦~啦~，啦~啦~啦~啦~啦~"（副歌片段，6秒）
[困难] "啦~啦~啦~啦~啦~"（开头旋律，3秒）
```
