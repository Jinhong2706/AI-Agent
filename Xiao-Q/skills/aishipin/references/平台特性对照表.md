| 项目 | 说明 |
|------|------|
| 官网 | https://jimeng.jianying.com |
| 语言 | 中文优先 |
| 时长 | 3秒 - 3分钟 |
| 音频 | ✅ 自动生成（环境音+对白+BGM） |
| 价格 | 按积分计费 |

```
[视觉风格]，[时长]，

```
