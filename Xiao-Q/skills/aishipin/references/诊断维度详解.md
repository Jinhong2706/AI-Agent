# 剧本诊断维度详解

## 维度一：结构诊断

### 检查项
| 检查项 | 标准 | 常见问题 | 修复难度 |
|--------|------|----------|----------|
| 开头钩子 | 3秒内抓住注意力 | 开头拖沓、信息量不足 | 🟢低 |
| 中段冲突 | 有明确矛盾推进 | 平铺直叙、缺乏张力 | 🟡中 |
| 结尾反转 | 出人意料或印象深刻 | 仓促收尾、烂尾 | 🔴高 |
| 因果逻辑 | 前后呼应、合理 | 逻辑跳跃、前后矛盾 | 🟡中 |

### 开头钩子检查
**标准**：3秒内出现悬念、冲突或情绪点

**问题示例**：
- ❌ 第1镜展示环境，无人物，无冲突
- ❌ 开头5秒只有背景音乐

**修复方案**：
- 倒叙开头：先展示结果或高潮画面
- 动作开头：以一个动作或冲突开始
- 台词开头：以一句有悬念的台词开始

## 维度二：情绪诊断

### 检查项
| 检查项 | 标准 | 常见问题 |
|--------|------|----------|
| 情绪曲线 | 有起伏、有高潮 | 平淡、高潮太晚 |
| 情绪切换 | 自然过渡 | 突兀、断裂 |
| 情绪密度 | 15秒内至少1个情绪点 | 情绪空白段 |
| 收尾情绪 | 留有余味 | 情绪悬空 |

## 维度三：镜头诊断

### 检查项
| 检查项 | 标准 | 常见问题 |
|--------|------|----------|
| 景别变化 | 远全中近特有变化 | 单调（如连续特写） |
| 运镜节奏 | 有快有慢 | 全程固定或运镜过多 |
| 镜头时长 | 匹配内容节奏 | 太长或太短 |
| 转场设计 | 自然或创意 | 硬切或无设计 |

### 景别变化规则
**推荐变化**：
- 中 → 特 → 近 → 特 → 中（有节奏）
- 远 → 全 → 中 → 近 → 特（推镜头）

**避免**：
- 特 → 特 → 特 → 特（单调）
- 远 → 远 → 远 → 远（乏味）

### 运镜节奏规则
**15秒视频运镜建议**：
- 固定镜头：2-3个
- 运动镜头：2-3个
- 避免连续运镜

## 维度四：台词诊断

### 检查项
| 检查项 | 标准 | 常见问题 |
|--------|------|----------|
| 台词密度 | 15秒内不超过3句 | 信息过载 |
| 金句设计 | 有记忆点 | 平淡无奇 |
| 台词风格 | 符合角色人设 | OOC（角色崩坏） |
| 留白设计 | 适当沉默 | 全程填满 |

### 台词密度计算
**公式**：总字数 ÷ 总时长

**标准**：
- 15秒视频：≤50字
- 30秒视频：≤100字
- 1分钟视频：≤200字

**问题**：
- 密度过高：观众来不及消化
- 密度过低：信息量不足

### 金句设计原则
- 短：不超过15字
- 画面感：能想象画面
- 情绪点：有情绪共鸣
- 记忆点：朗朗上口

---

## 维度五：音效诊断

### 检查项
| 检查项 | 标准 | 常见问题 |
|--------|------|----------|
| BGM切换 | 配合情绪变化 | 全程同一BGM |
| 环境音 | 增强沉浸感 | 缺失或突兀 |
| 音效卡点 | 配合动作节奏 | 错位或过多 |
| 静音设计 | 制造反差 | 忽略静音的力量 |

### 音效设计原则
**环境音**：
- 咖啡店：爵士乐、杯碟声、交谈声
- 办公室：键盘声、电话声、空调声
- 户外：风声、鸟叫、车流声

**BGM切换时机**：
- 情绪转折点
- 场景转换点
- 高潮来临前

**静音的力量**：
- 紧张时刻：突然安静
- 震惊时刻：突然安静
- 对比强烈：先静后响

## 维度六：平台适配诊断

### 抖音检查
| 检查项 | 标准 |
|--------|------|
| 开头3秒 | 有钩子、有悬念 |
| 节奏 | 15秒内完成一个完整情绪 |
| 竖屏适配 | 主体居中，上方留白 |
| 字幕 | 关键信息有字幕 |

### 小红书检查
| 检查项 | 标准 |
|--------|------|
| 封面感 | 至少1镜有强视觉冲击 |
| 真实感 | 不要过度修饰 |
| 种草感 | 有产品或生活方式展示 |

### B站检查
| 检查项 | 标准 |
|--------|------|
| 信息密度 | 有干货或有趣 |
| 时长 | 1-3分钟为宜 |
| 互动点 | 有弹幕触发点 |

### 视频号检查
| 检查项 | 标准 |
|--------|------|
| 情感共鸣 | 有共情点 |
| 适老化 | 字体大、语速慢 |
| 正能量 | 积极向上 |

## 评分标准

### 各维度满分100分

| 分数段 | 等级 | 说明 |
|--------|------|------|
| 90-100 | S | 优秀，无明显问题 |
| 80-89 | A | 良好，小瑕疵 |
| 70-79 | B | 中等，有改进空间 |
| 60-69 | C | 及格，问题较多 |
| 60以下 | D | 不及格，需要大改 |

### 综合评分计算
```
综合评分 = 结构×20% + 情绪×20% + 镜头×15% 
         + 台词×15% + 音效×15% + 平台×15%
```
