# 爆款公式库详解

## 公式一：反转剧（抖音TOP1）

### 结构
```
铺垫(3-5s) → 期待建立(3-5s) → 反转(2-3s) → 金句(2-3s)
```

### 情绪流
好奇 → 期待 → 惊讶 → 会心一笑

### 镜头设计
| 镜号 | 时长 | 内容 | 作用 |
|------|------|------|------|
| 1 | 3-5s | 展示看似正常的场景 | 建立预期 |
| 2 | 3-5s | 角色做动作，暗示结果 | 强化预期 |
| 3 | 2-3s | 揭示不同真相 | 打破预期 |
| 4 | 2-3s | 点睛之笔 | 留下记忆 |

### 案例：职场反转
- 镜头1：男生紧张整理领带（预期：要去面试）
- 镜头2：走向西装男，恭敬打招呼（预期：面试官）
- 镜头3：西装男举起辞职信（反转：已被裁员）
- 镜头4：字幕"职场玄学"

### 注意事项
- ✅ 反转要在观众"确信"后出现
- ✅ 反转要合乎逻辑
- ❌ 不要反转太多次

## 公式二：种草文（小红书TOP1）

### 结构
```
痛点引入(2-3s) → 产品展示(3-5s) → 效果对比(3-5s) → 行动引导(2-3s)
```

### 情绪流
共鸣 → 好奇 → 心动 → 行动

### 镜头设计
| 镜号 | 时长 | 内容 | 作用 |
|------|------|------|------|
| 1 | 2-3s | 展示痛点 | 引发共鸣 |
| 2 | 3-5s | 产品特写 | 建立认知 |
| 3 | 3-5s | 使用前vs后 | 证明效果 |
| 4 | 2-3s | 价格+购买 | 促成转化 |

### 案例：口红种草
- 镜头1：素颜，嘴唇苍白，叹气
- 镜头2：口红特写，展示质地
- 镜头3：左右对比，涂后元气满满
- 镜头4：限时99，冲！

### 注意事项
- ✅ 痛点要真实
- ✅ 效果对比要真实
- ❌ 不要夸大效果

---

## 公式三：知识点（B站TOP1）

### 结构
```
提问引入(3-5s) → 核心解答(20-40s) → 案例佐证(10-20s) → 总结回顾(5-10s)
```

### 情绪流
好奇 → 获得感 → 信服 → 满足

### 镜头设计
| 镜号 | 时长 | 内容 | 作用 |
|------|------|------|------|
| 1 | 3-5s | 抛出问题 | 激发好奇 |
| 2 | 20-40s | 分点讲解 | 传递价值 |
| 3 | 10-20s | 案例/数据 | 增强可信 |
| 4 | 5-10s | 图文总结 | 加深记忆 |

### 案例：科普视频
- 镜头1：你知道为什么XX吗？
- 镜头2：3个知识点讲解
- 镜头3：真实案例佐证
- 镜头4：记住这3点

### 注意事项
- ✅ 知识点要准确
- ✅ 节奏不要太快
- ❌ 不要太学术化

---

## 公式四：情感共鸣（视频号TOP1）

### 结构
```
场景切入(3-5s) → 情绪渲染(10-15s) → 金句升华(3-5s) → 互动引导(2-3s)
```

### 情绪流
代入 → 共鸣 → 感动 → 参与

### 案例：情感短片
- 镜头1：深夜加班的场景
- 镜头2：情绪渲染，挣扎与坚持
- 镜头3："成年人的世界没有容易二字"
- 镜头4：点赞让更多人看到

---

## 公式五：产品展示（全平台通用）

### 结构
```
产品亮相(2-3s) → 功能展示(5-8s) → 细节特写(3-5s) → 品牌收尾(2-3s)
```

### 情绪流
惊艳 → 了解 → 心动 → 印象

### 案例：产品广告
- 镜头1：产品整体亮相
- 镜头2：核心功能演示
- 镜头3：材质/细节特写
- 镜头4：品牌logo+标语

---

| 情感故事 | 情感共鸣 | 视频号 |
| 广告展示 | 产品展示 | 全平台 |
| 互动活动 | 挑战赛 | 快手 |
