# 公众号内容合规指南

## 平台规则红线

### 绝对禁止内容
1. 政治敏感内容
2. 虚假信息、谣言
3. 抄袭、洗稿
4. 投资建议、荐股
5. 内幕信息传播
6. 炒作股价
7. 诱导交易
8. 违法广告

### 需谨慎处理的内容
1. 上市公司负面信息（需事实确凿、来源可靠）
2. 行业争议话题（需平衡多方观点）
3. 政策解读（需依据官方表述）
4. 人物报道（需避免名誉侵权）

## 财经内容特别规范

### 投资建议的红线

**禁止表述：**
- "建议买入XX股票"
- "目标价XX元"
- "强烈推荐"
- "必涨"
- "机构内幕消息"

**合规表述：**
- "市场观点认为..."
- "根据XX数据显示..."
- "分析师预计..."
- "值得关注的是..."
- "历史数据显示..."

### 数据引用规范

**必须标注来源：**
- 财务数据：根据公司财报/Wind数据
- 市场数据：根据交易所/行业组织
- 研究结论：根据XX机构研究报告

**引用格式示例：**
> 根据Wind数据显示，2025年A股市场...

> 财报显示，公司2024年营收同比增长...

### 争议性内容处理

1. **多方观点呈现**
   - 支持方观点 + 反对方观点
   - 不站队，不引导结论

2. **不确定性标注**
   - "市场对此存在分歧"
   - "各方观点不一"
   - "仍需观察"

3. **时间限定**
   - "截至发稿时"
   - "就目前公开信息"

## 版权合规

### 文字内容
- 不直接复制他人文章段落
- 引用需注明出处
- 数据可引用，表述需原创

### 图片使用
- 使用标注来源的公开图片
- 不使用他人原创图表
- 建议自行制作数据图表

## 风险规避清单

在文章发布前，对照以下清单自查：

- [ ] 所有数据是否有可靠来源？
- [ ] 是否存在可能被认定为投资建议的表述？
- [ ] 是否存在未经证实的传言？
- [ ] 争议性话题是否呈现多方观点？
- [ ] 是否存在抄袭嫌疑？
- [ ] 图片是否合规？
- [ ] 是否涉及未公开信息？

## 常见违规案例警示

| 违规类型 | 典型表现 | 后果 |
|---------|---------|-----|
| 荐股 | "明天买入必涨" | 封号、法律风险 |
| 传播谣言 | "XX公司暴雷传闻"（未证实） | 删除、封号、侵权 |
| 抄袭洗稿 | 大段复制他人文章 | 删除、封号、侵权 |
| 内幕信息 | "提前得知并购消息" | 法律风险 |

---

**核心原则：事实可验证、观点有依据、表述够谨慎**
