## 各观测点详细数据

### 妙峰山（门头沟，1291m）
- **特点**: 金顶停车场可直达观景台，路况好，适合新手
- **最佳观测点**: 金顶停车场 → 妙峰山金顶
- **预计到达时间**: 市区出发约 1.5-2h
- **云海类型**: 辐射雾为主（山谷风局地环流）
- **日出方向**: 正东偏北（冬季 110°，夏季 75°）

### 东灵山（门头沟，2303m）
- **特点**: 北京最高峰，云海层次最丰富
- **注意**: 登山强度较大，高海拔注意保暖
- **风险**: 夜间无路灯，需带手电
- **云海类型**: 辐射雾 + 迎风坡抬升雾

### 海坨山（延庆，2198m）
- **特点**: 2022年冬奥会主赛场，设施完善
- **最佳季节**: 10月-次年5月（冬季海坨戴雪）
- **云海类型**: 逆温层雾（冬季最强）
- **日出方向**: 正东偏北（冬季适合）

### 雾灵山（密云，2116m）
- **风险**: 降水概率偏高（阵雨），不推荐作为首选
- **适合**: 有经验的山地摄影师
- **云海类型**: 迎风坡雾（东南季风水汽）

### 百花山（门头沟，1991m）
- **特点**: 植被茂密，水汽充足，云海持续时间长
- **注意**: 需徒步约2h到达山顶
- **云海类型**: 辐射雾 + 植物蒸腾增湿

### 云蒙山（怀柔，1414m）
- **特点**: 密林峡谷，瀑布云常见，视觉震撼
- **注意**: 山路较陡，建议结伴
- **云海类型**: 峡谷抬升雾（局地对流）

### 黄草梁（门头沟，1700m）
- **位置**: 门头沟斋堂镇，与妙峰山/百花山同属西山系
- **特点**: 官方景区营地，可观星河+云海双景双绝，近年热门
- **日出方向**: 正东（与海坨山方向互补）
- **预计到达时间**: 市区出发约 2.5-3h
- **云海类型**: 辐射雾（高山草甸，无树木遮挡，雾散快）

### 长峪城（昌平，1400m）
- **位置**: 昌平流村镇，紧邻海坨山北麓
- **特点**: 海坨山云海北侧参照点，东北-西南沟谷拦截东南水汽
- **日出方向**: 正东偏北
- **预计到达时间**: 市区出发约 2h
- **云海类型**: 峡谷局地环流雾（与海坨山同期概率接近）

### 东指壶（平谷，1234m）
- **位置**: 平谷区，紧邻河北省
- **特点**: 平谷最高峰，靠近多个小型水库，湿度高，小众秘境
- **日出方向**: 正东
- **预计到达时间**: 市区出发约 2-2.5h
- **云海类型**: 辐射雾 + 局地水汽雾
- **注意**: 路况一般，建议SUV

### 燕羽山（延庆，1180m）
- **位置**: 延庆区，独立性较强
- **特点**: 延庆腹地，可作为海坨山云海的辅助观景点
- **日出方向**: 正东偏南
- **预计到达时间**: 市区出发约 2h
- **云海类型**: 迎风坡抬升雾

### 箭扣长城（延庆，~900m）
- **位置**: 延庆区渤海镇，箭扣自然风景区
- **特点**: 箭扣长城最著名段（天梯-将军守关-北京结），有长城穿云经典照
- **日出方向**: 正东偏北
- **预计到达时间**: 市区出发约 2.5-3h
- **云海类型**: 逆温层雾（长城在雾上方，云从城墙穿过）
- **注意**: 长城段徒步强度大，无路灯，需手电，优先结伴

### 金山岭长城（滦平/密云，~1100m）
- **位置**: 河北省滦平县巴克什营镇，紧邻密云
- **特点**: 金山岭长城保存最完好，敌楼最密（72座），有长城穿云盛景
- **日出方向**: 正东偏北
- **预计到达时间**: 市区出发约 2.5h（高速直达景区门口）
- **云海类型**: 逆温层雾（长城在雾上方）
- **注意**: 景区有索道，建议徒步上山段（约1.5h）

### 北灵山（门头沟，~1898m）
- **位置**: 门头沟斋堂镇北侧（灵山系北麓）
- **特点**: 与东灵山同属北京西山系，北麓沟谷拦截东南水汽
- **海拔**: 约 1898m（与百花山相邻但更偏北）
- **日出方向**: 正东
- **预计到达时间**: 市区出发约 2.5h
- **云海类型**: 辐射雾 + 迎风坡雾

### 西南灵山（门头沟，~1800m）
- **位置**: 门头沟区与河北省交界处，灵山西坡
- **特点**: 与东灵山同属西山系，但为西坡视角，观测方向不同
- **海拔**: 约 1800m
- **日出方向**: 正东偏南
- **预计到达时间**: 市区出发约 2.5-3h
- **云海类型**: 迎风坡雾

## 云海公式（核心原理）

```
云海概率 = f(辐射冷却, 稳定层结, 风力条件, 水汽, 前置降水)
```

**辐射冷却**（权重 30%）：
- 条件：晴夜 + 强辐射 + 低湿度
- 量化：地面温度降幅 = 日最高温 - 凌晨最低温（≥ 10°C 为佳）

**稳定层结 / 逆温层**（权重 25%）：
- 条件：逆温层存在，高度 800-1500m
- 量化：气压 > 1015 hPa 且温度递减率 < 6.5°C/km

**风力条件**（权重 20%）：
- 理想：2-8 km/h（微风），> 15 km/h 雾散，< 2 km/h 雾浓但不稳

**水汽供给**（权重 15%）：
- 关键：凌晨湿度峰值 > 90%，或 T-Td < 2°C

**前置降水**（权重 10%）：
- 理想：前1-5天有 5-20mm 降水（土壤湿润），当天无降水

## 出行装备建议

| 装备 | 必需 | 说明 |
|------|:----:|------|
| 登山鞋 | ✅ | 山路湿滑 |
| 保暖衣物 | ✅ | 海拔每升1000m降温6.5°C |
| 头灯 | ✅ | 夜路上山必需 |
| 充电宝 | ✅ | 低温下手机掉电极快 |
| 雨衣 | ⚠️ | 备选，山地气候多变 |
| 无人机 | ⚠️ | 航拍云海效果极佳 |
| 单反+三脚架 | ⚠️ | 长曝光拍雾海流动 |
| 地图/离线导航 | ⚠️ | 部分路线无信号 |
