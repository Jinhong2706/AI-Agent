## 零、云海形成机制分类（5种主要类型）

> **核心概念**：云海本质是**在山体高度形成的层云**，其形成机制决定了云海的厚度、持续时间、视觉效果。

### 0.1 五种形成机制总览

| 类型 | 形成机制 | 典型条件 | 最佳观测时段 | 视觉特征 | 典型地点 |
|------|---------|---------|-------------|---------|---------|
| **辐射雾型云海** | 夜间地面辐射冷却，近地面空气降温至露点，水汽凝结成雾 | 晴朗无云夜 + 微风<3m/s + 高湿度>80% + 逆温层 | 日出前后 05:00-07:00 | 雾层低浅（50-300m），日出后逐渐消散，层次分明 | 海坨山、雾灵山、百花山 |
| **平流雾型云海** | 暖湿空气流经冷地面/水面，冷却凝结成雾 | 暖湿气流 + 冷下垫面 + 风速3-8m/s | 全天可持续，夜间增强 | 雾层厚度大（500-1500m），范围广，持续时间长 | 白石山、白草畔 |
| **上坡雾（地形雾）型云海** | 暖湿空气沿山坡上升，绝热冷却凝结成雾 | 迎风坡 + 上升气流 + 高湿度 | 风向稳定时段 | 雾随地形流动，动态感强，云雾从身旁流过 | 雾灵山、百花山、云蒙山 |
| **蒸发雾（蒸汽雾）型云海** | 冷空气流经暖水面，水面蒸发的水汽遇冷凝结 | 秋冬季节 + 冷空气 + 暖湖面/水库 | 清晨或夜间 | 雾层薄，呈蒸汽状，贴近水面 | 长峪城（近水库）、东指壶 |
| **锋面雾型云海** | 锋面过境时，暖雨滴落入冷空气中蒸发成雾 | 锋面过境前后 + 降水 + 温度骤变 | 锋面过境时 | 雾层不稳定，伴有降水，范围大但持续时间短 | 各峰均可出现 |

### 0.2 详细诊断标准

#### ① 辐射雾型云海

**物理机制：**
```
夜间晴空 → 地面辐射冷却（Q↓）→ 近地气温下降（T↓）
→ T降至Td以下 → 水汽凝结 → 形成逆温层 → 雾层稳定
日出后 → 太阳辐射加热（Q↑）→ 逆温层破坏 → 雾消散
```

**判定条件：**
| 参数 | 条件 | 说明 |
|------|------|------|
| 夜间云量 | < 30% | 晴空利于辐射冷却 |
| 风速 | < 5 km/h | 微风，雾层稳定 |
| 湿度 | > 85% | 高湿度 |
| T - Td | < 2°C | 接近饱和 |
| 逆温层 | 存在 | 温度随高度增加 |
| LCL | < 300m | 极低雾底 |

**典型气象要素配置：**
```
夜间 02:00-05:00: T↓, RH↑, wind↓, cloud_cover=0
黎明 05:00-06:00: T最低, RH最高(~100%), LCL最低(~50-100m)
日出 06:00-08:00: T↑, RH↓, 雾层抬升
```

**视觉特征：**
- 雾底极低（50-200m），层次分明
- 日出后逐渐消散，观赏窗口短（1-2小时）
- 最佳拍摄时间：日出前后30分钟

---

#### ② 平流雾型云海

**物理机制：**
```
暖湿空气（T高, RH高）→ 平流至冷下垫面（T低）
→ 底层空气冷却 → T降至Td → 凝结成雾
持续暖湿平流 → 雾层维持或加厚
```

**判定条件：**
| 参数 | 条件 | 说明 |
|------|------|------|
| 风向 | 来自暖湿区域 | 如东南风来自海洋 |
| 风速 | 3-10 km/h | 适中，维持平流 |
| 海陆温差 | 海温 > 气温 | 典型平流雾条件 |
| 稳定度 | 稳定层结 | 利于雾层维持 |
| 持续时间 | 可持续数天 | 与辐射雾不同 |

**典型气象要素配置：**
```
风向: SE-E（来自暖湿区）
风速: 5-8 km/h（稳定平流）
温度: 较高（暖平流）
湿度: > 90%
雾层厚度: 500-1500m（较厚）
持续时间: 全天可持续
```

**视觉特征：**
- 雾层厚实，覆盖范围广
- 持续时间长，可维持数天
- 视觉稳定，适合全天观赏

---

#### ③ 上坡雾（地形雾）型云海

**物理机制：**
```
暖湿空气 → 遇到山坡 → 被迫抬升 → 绝热冷却（T↓≈0.6°C/100m）
→ T降至Td → 凝结成雾 → 沿山坡流动
```

**判定条件：**
| 参数 | 条件 | 说明 |
|------|------|------|
| 风向 | 垂直于山脊 | 迎风坡 |
| 风速 | > 5 km/h | 足够的上升动力 |
| 地形坡度 | > 10° | 明显抬升 |
| 湿度 | > 80% | 充足水汽 |
| 凝结高度 | < 山顶海拔 | 可形成雾 |

**各峰迎风坡判定：**
| 山峰 | SE风 | S风 | E风 | NW风 |
|------|:----:|:----:|:----:|:----:|
| 雾灵山 | ✅ | ✅ | ✅ | ❌ |
| 百花山 | ✅ | ✅ | ❌ | ❌ |
| 云蒙山 | ✅ | ✅ | ✅ | ❌ |
| 白石山 | ✅ | ✅ | ❌ | ❌ |
| 白草畔 | ✅ | ✅ | ✅ | ❌ |

**视觉特征：**
- 云雾沿山坡流动，动态感极强
- 雾从身旁流过，如置身仙境
- 迎风坡最佳，背风坡干燥

---

#### ④ 蒸发雾（蒸汽雾）型云海

**物理机制：**
```
冷空气（T低, Td低）→ 流经暖水面（T水高）
→ 水面蒸发 → 水汽进入冷空气 → T升至Td以上 → 凝结成雾
```

**判定条件：**
| 参数 | 条件 | 说明 |
|------|------|------|
| 季节 | 秋冬 | 冷空气活跃 |
| 水气温差 | 水-气 > 10°C | 强蒸发 |
| 风速 | < 5 km/h | 雾层稳定 |
| 水面温度 | > 气温 | 暖水面 |

**典型地点：**
- 官厅水库周边（长峪城附近）
- 密云水库周边（雾灵山方向）
- 金海湖周边（东指壶方向）

**视觉特征：**
- 雾层薄，呈蒸汽状
- 贴近水面，高度有限
- 冬季清晨最典型

---

#### ⑤ 锋面雾型云海

**物理机制：**
```
锋面过境 → 暖雨滴落入冷空气 → 蒸发成水汽
→ 冷空气中水汽过饱和 → 凝结成雾
或: 锋后冷空气流经暖湿地面 → 蒸发成雾
```

**判定条件：**
| 参数 | 条件 | 说明 |
|------|------|------|
| 锋面位置 | 过境前后 | 气象预报 |
| 降水 | 有 | 雨滴蒸发 |
| 温度变化 | 骤变 | 锋面特征 |
| 风向变化 | 明显 | 锋面过境 |
| 持续时间 | 短（数小时） | 随锋面移动 |

**风险提示：**
- ⚠️ 锋面雾伴有降水，能见度极低，山区行车危险
- ⚠️ 雾层不稳定，观赏效果可能不佳
- ⚠️ 需密切关注天气预报

**视觉特征：**
- 雾层不稳定，伴有降水
- 范围大但持续时间短
- 光线暗淡，拍摄效果差

---

### 0.3 云海类型诊断流程

```
Step 1: 判断湿度条件
  → RH < 70%: 无雾条件，云海概率低
  → RH > 85%: 进入 Step 2

Step 2: 判断夜间云量
  → cloud_cover < 30%: 辐射雾型可能性高
  → cloud_cover > 70%: 进入 Step 3

Step 3: 判断风向风速
  → 风向来自暖湿区 + 风速 3-10 km/h: 平流雾型
  → 风向垂直山脊 + 风速 > 5 km/h: 上坡雾型
  → 风速 < 3 km/h + 近水面: 蒸发雾型

Step 4: 判断天气系统
  → 有锋面过境: 锋面雾型

Step 5: 综合诊断
  → 可能多型并存（如辐射雾+上坡雾）
  → 主导类型由最强机制决定
```

## 一、雾的垂直分层模型

理解云海的关键：云海不是雾，而是**在山体高度形成的层云**。

### 三层雾结构

```
海拔
  ▲
3000m │  高空云层（层云/层积云）— 不算云海
     │  ───────────────────────
2500m │  上坡雾顶（云海顶界）
     │  ─────────────────────── ← 云底估算线
2000m │  云海层（最佳观测区）  ← 海坨山2198m在这里
     │  ───────────────────────
1500m │  平流雾/辐射雾层
     │
1000m │  谷雾/辐射雾底（妙峰山1291m接近此层）
     │
 500m │  近地面辐射雾
     │  ═══════════════════════
  0m │  地面
```

**云底高度估算公式（修正版）：**
```
# 实测露点（推荐）
LCL_ag(m) = 125 × (T − dew_point_2m)    ← 地面以上的云底高度
雾顶(m MSL) = 气象站海拔(m) + LCL_ag(m) + 雾厚(m, 保守取100m)

# 备选（无实测露点时）
CloudBase(m) = 125 × (T_max − T_dew)
```
- T_max = 日最高气温（°C）
- T_dew = 露点温度（°C）
- **⚠️ 严禁假设 T_dew**：2026-04-16 事故记录表明，假设 Td=7°C 导致妙峰山云底估算错误高达 1875m，正确值仅 680m

**⚠️ 雾顶是每日动态计算的，以下为参考示例而非标准值。**

> **2026-04-20 修正**：雾顶由 `fog_top_msl = station_elev + LCL_ag + 100m` 动态计算，每天都不同。
> 以下为 2026-04-17 特定天气条件下的实测示例，仅作定性参考。

## 二、雾型交互矩阵（组合效应）

多种雾型同时存在时，会**叠加增强或相互削弱**。

### 常见组合

| 组合 | 效果 | 概率修正 | 视觉质量 |
|------|------|:--------:|---------|
| 上坡雾 + 辐射雾 | 上坡雾叠加在山谷辐射雾之上 | **+8** | ★★★★★ 最佳 |
| 上坡雾 + 平流雾 | 大范围稳定云海，厚度增加 | **+6** | ★★★★☆ 范围大 |
| 辐射雾 + 谷雾 | 填充山谷，上方山尖清晰 | **+4** | ★★★★ 层次分明 |
| 平流雾 + 锋面雾 | 阴天层云，无云海 | **-5** | ★☆☆☆ 暗淡 |
| 上坡雾单独 | 纯净壮丽 | **+5** | ★★★★★ |
| 辐射雾单独 | 消散快 | **+0** | ★★★ 短时美 |

### 组合评分公式

```
组合修正分 = Σ(各雾型基础分) + 交互增强分 - 冲突扣分

交互增强分：
  上坡雾+辐射雾 → +3（垂直叠加，增强效果）
  上坡雾+平流雾 → +2（水平扩展，稳定）
  辐射雾+谷雾   → +2（填谷效果）

冲突扣分：
  锋面雾参与   → -4（层云遮蔽）
  蒸发雾出现   → -1（低效）
```

## 四、雾底高度动态估算

### 实时估算方法

从 Open-Meteo 获取 `temperature_2m` 和露点数据（需另行计算或从空气质量API获取）：

```python
# 方法1：简化估算（只需温度和露点）
def estimate_cloud_base(max_temp, dew_point):
    """
    云底高度 = 125 × 温度露点差
    TDD = Temperature-Dew Point Depression
    """
    tdd = max_temp - dew_point
    cloud_base = 125 * tdd
    return cloud_base

# 方法2：更精确（考虑气压）
def estimate_cloud_base_precise(max_temp, dew_point, pressure=1013):
    """
    更精确估算，考虑气压修正
    pressure: 海平面气压 (hPa)
    """
    tdd = max_temp - dew_point
    # 气压修正：高压系统云底偏低，低压偏高
    pressure_factor = (pressure - 1013) * 10 / 100  # 简化
    cloud_base = 125 * tdd + pressure_factor * 50
    return max(cloud_base, 100)  # 最低100m

# 示例：海坨山周五（最高26°C，露点约10°C，气压1012hPa）
t_max = 26
t_dew = 10
p = 1012
base = estimate_cloud_base_precise(t_max, t_dew, p)
peak = 2198  # 海坨山
print(f"估算云底: {base:.0f}m")           # → 约 2012m
```

## 六、雾型诊断算法（优化版）

### Step 1 — 提取关键气象指标

```python
def get_fog_diagnosis_data(hourly_data):
    """从逐时数据中提取雾型诊断所需指标"""
    # 凌晨时段（02:00-05:00）取平均值
    night_slice = hourly_data["02:00":"05:00"]

    wind_speed = avg(night_slice["wind_speed"])        # 凌晨平均风速 km/h
    wind_dir = dominant_wind_dir(night_slice["wind_direction"])  # 主导风向
    humidity_peak = max(night_slice["humidity"])      # 凌晨湿度峰值
    cloud_night = avg(night_slice["cloud_cover"])     # 夜间平均云量
    precip = sum(night_slice["precipitation"])        # 凌晨降水累计
    pressure_trend = pressure_now - pressure_6h_ago    # 气压趋势

    # 云底估算
    t_max = daily_max_temp      # 日最高温
    t_dew = compute_dew_point(hourly_data)  # 露点温度
    cloud_base = 125 * (t_max - t_dew)

    return {
        "wind_speed": wind_speed,
        "wind_dir": wind_dir,
        "humidity_peak": humidity_peak,
        "cloud_night": cloud_night,
        "precip": precip,
        "pressure_trend": pressure_trend,
        "cloud_base": cloud_base,
        "is_south_wind": 90 <= wind_dir <= 270,
    }
```

### Step 2 — 各雾型概率计算

```python
def calc_fog_type_probs(d):
    """
    计算各雾型概率（返回 {雾型: 概率} dict）
    d = get_fog_diagnostics_data() 输出
    """
    probs = {}

    # === 锋面雾 ===
    if d["precip"] > 1.0 and d["pressure_trend"] < -3:
        probs["锋面雾"] = 90
    elif d["precip"] > 0.3:
        probs["锋面雾"] = 50
    else:
        probs["锋面雾"] = 5

    # === 上坡雾（核心云海雾型）===
    if 3 <= d["wind_speed"] <= 20 and d["humidity_peak"] >= 75:
        score = 40
        if d["is_south_wind"]:       score += 25   # 迎风坡最强
        if d["cloud_night"] <= 40:  score += 20   # 晴夜加成
        if 5 <= d["wind_speed"] <= 12: score += 15  # 最理想风速区间
        if d["humidity_peak"] >= 85: score += 10   # 极高湿度
        if d["pressure_trend"] > 0: score += 5    # 气压上升（高压逼近）
        probs["上坡雾"] = min(score, 95)

    # === 平流雾 ===
    if 8 <= d["wind_speed"] <= 25 and d["humidity_peak"] >= 75:
        score = 35
        if d["is_south_wind"]:        score += 25   # 暖湿气流
        if d["cloud_night"] >= 40:    score += 20   # 多云（辐射弱）
        if d["humidity_peak"] >= 85:  score += 15   # 极高湿度
        probs["平流雾"] = min(score, 90)

    # === 辐射雾 ===
    if d["wind_speed"] <= 8 and d["cloud_night"] <= 40:
        score = 30
        if d["wind_speed"] <= 3:       score += 25   # 极静风
        if d["cloud_night"] <= 20:    score += 20   # 完全晴夜
        if d["humidity_peak"] >= 80:  score += 15   # 高湿度
        if d["pressure_trend"] > 0:    score += 10   # 高压控制
        probs["辐射雾"] = min(score, 90)

    # === 谷雾 ===
    if d["wind_speed"] <= 5 and d["cloud_night"] <= 30:
        probs["谷雾"] = 50 + (30 if d["wind_speed"] <= 2 else 0)

    return probs
```

### Step 3 — 雾型组合与云底综合评分

```python
def calc_fog_cloud_sea_score(fog_probs, cloud_base, peak_elevation):
    """
    计算雾型对云海形成的综合影响分
    fog_probs: {雾型: 概率}
    cloud_base: 估算云底高度 (m)
    peak_elevation: 山峰海拔 (m)
    """
    # 1. 云底位置评分（最重要）
    diff = peak_elevation - cloud_base
    if diff > 500:     base_score = 10
    elif diff > 200:   base_score = 8
    elif diff > 0:     base_score = 5
    elif diff > -300:  base_score = 3
    else:              base_score = 0

    # 2. 雾型组合效应
    fog_effect = {
        "上坡雾":   {"base": 8.3, "bonus_with": {"辐射雾": 8, "平流雾": 6, "谷雾": 4}},
        "平流雾":   {"base": 7.0, "bonus_with": {"上坡雾": 6, "辐射雾": 3}},
        "辐射雾":   {"base": 6.3, "bonus_with": {"谷雾": 7, "上坡雾": 8}},
        "谷雾":     {"base": 6.8, "bonus_with": {"辐射雾": 7}},
        "锋面雾":   {"base": 3.8, "bonus_with": {}},
        "混合雾":   {"base": 5.0, "bonus_with": {}},
    }

    # 取主导雾型
    dominant = max(fog_probs, key=fog_probs.get)
    info = fog_effect.get(dominant, fog_effect["混合雾"])
    fog_score = info["base"]

    # 组合加分
    for other_type, other_prob in fog_probs.items():
        if other_type != dominant and other_prob > 30:
            bonus = info["bonus_with"].get(other_type, 0)
            fog_score += bonus * (other_prob / 100)

    # 3. 综合
    final_score = base_score * 0.5 + fog_score * 0.5
    return round(final_score, 1), dominant, fog_probs
```

### Step 4 — 最终概率映射

```python
def score_to_prob(score):
    """将综合评分映射到云海概率"""
    if score >= 9.0:   return (75, 85)  # 极大概率
    elif score >= 8.0: return (65, 75)
    elif score >= 7.0: return (55, 65)
    elif score >= 6.0: return (45, 55)
    elif score >= 5.0: return (30, 45)
    elif score >= 4.0: return (20, 30)
    else:              return (10, 20)
```
