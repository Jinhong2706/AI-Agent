# 合同审查助手

这是一个最小可安装 Skill 包示例，用于演示合同审查类技能的基础结构、YAML front matter、触发条件、执行流程、风险分级、异常处理与交付契约。

## 包含文件

- `SKILL.md`：技能主文件，包含可解析 YAML front matter 与中文技能说明。
- `package.json`：最小包元数据。
- `README.md`：安装包说明。

## 能力范围

- 合同初步审查：适用于合同、协议、补充协议、订单条款、合作备忘录等文本。
- 风险识别：识别条款缺失、权责不平衡、付款与交付风险、违约责任、解除、保密、知识产权、争议解决等问题。
- 风险分级：按高、中、低风险输出结构化风险清单。
- 修订建议：提供可执行的修改方向，必要时给出参考措辞。
- 异常处理：覆盖材料缺失、法域不明、重大专业交易、疑似违法规避、隐私与商业秘密等情况。
- 交付契约：输出一句话结论、审查范围与限制、风险清单、重点修订建议、待确认问题和下一步建议。

## 使用提醒

本示例仅用于技能包生成、安装和审查流程测试，不构成正式法律意见。实际用于业务场景时，应结合公司制度、交易背景、适用法域与专业法务或律师意见进行复核。
