# 数据分析 Skill 示例包

这是一个最小可安装 Skill 包示例，用于演示数据分析类技能的注册信息、触发条件、数据读取清洗、分析流程、可视化、异常处理与交付契约。

## 文件结构

- `SKILL.md`：技能主体说明，包含可解析 YAML front matter 和中文执行规范。
- `package.json`：最小包元数据，声明包名、版本、文件清单与关键词。
- `README.md`：安装包说明和使用建议。

## 使用建议

将本目录作为独立技能包复制或发布到目标 SkillHub/技能加载目录后，由运行环境读取 `SKILL.md` 的 front matter 与正文规则。

## 能力边界

本包仅提供最小可安装骨架，不内置固定分析脚本。实际执行时应根据用户提供的数据文件、指标口径和交付要求动态生成读取、清洗、分析和可视化步骤。
