# cloud-doc-editing

这是一个最小可安装的云文档编辑类 Skill 示例包，用于演示如何安全处理 WPS、飞书等云文档的读取、编辑、权限、版本与回写确认边界。

## 包内容

- `SKILL.md`：技能主体说明，含可解析 YAML front matter。
- `package.json`：最小包元数据与基础校验脚本。
- `README.md`：安装与使用说明。

## 安装使用

将本目录作为一个 Skill 包导入或复制到目标 SkillHub/技能目录中。导入前建议执行：

```bash
npm run check
```

该检查只验证 `SKILL.md` 是否存在 YAML front matter 起止边界，不替代完整 YAML 解析与业务测试。

## 适用场景

- 云文档内容摘要、结构化提取、审阅建议。
- 在用户明确授权后，对云文档进行小范围编辑。
- 对 WPS、飞书等在线文档进行版本、权限、协作状态与回写确认检查。

## 安全边界

- 不绕过权限，不处理未授权链接或凭据。
- 不默认覆盖云端原文；回写前必须确认目标、范围和风险。
- 对多人协作、版本变化、评论修订、公式宏等高风险内容必须先提示用户确认。
- 无法确认回写成功时，只能说明状态不确定，并建议用户在云端人工核验。
