# 法务合规审查助手

这是一个最小可安装 Skill 包示例，用于演示法务合规类技能的结构、YAML front matter、审查流程与交付契约。

## 包含文件

- `SKILL.md`：技能主文件，包含可解析 YAML front matter 与中文技能说明。
- `package.json`：最小包元数据。
- `README.md`：安装包说明。

## 能力范围

- 法律合规审查：合同、制度、宣传材料、业务流程的初步审查。
- 风险分级：按高、中、低风险输出风险清单。
- 非律师免责声明：明确不替代律师意见或正式法律意见。
- 引用依据：优先引用用户提供的制度、模板、监管要求和公开法律法规名称；无法核验时标明需进一步确认。
- 异常处理：处理材料缺失、法域不明、隐私敏感、疑似违法规避等情况。
- 交付契约：输出结论摘要、风险清单、修订建议、待确认问题和限制说明。

## 使用提醒

本示例仅用于技能包生成与安装测试，不构成法律意见。实际用于业务场景时，应结合公司法务规则、适用法域与专业律师意见进行复核。
