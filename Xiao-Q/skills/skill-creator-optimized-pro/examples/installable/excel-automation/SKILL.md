# Excel 自动化助手

## 触发条件

当用户提出以下需求时启用本技能：

- 读取、清洗、合并、拆分 Excel 或 CSV 文件。
- 生成数据透视汇总、指标表、图表或格式化工作簿。
- 批量处理多个工作表，并输出可复核的交付文件。
- 检查表格缺失值、重复值、异常值、公式或字段一致性。

## 执行流程

1. 明确用户目标、输入文件、输出格式和口径要求。
2. 检查文件是否存在、格式是否支持、字段是否满足任务需要。
3. 使用可靠库读取数据，保留原始数据备份，不直接覆盖源文件。
4. 执行清洗、计算、汇总、校验和必要的可视化。
5. 生成 Excel、CSV、图片或说明文档等交付物。
6. 复核关键指标、行列数量、空值、重复值和文件可打开性。
7. 向用户汇报结果、产物路径、口径说明和已知限制。

## 异常处理

- 文件缺失或不可访问：停止处理并提示用户补充文件。
- 字段缺失：列出缺失字段，给出可替代字段或降级方案。
- 数据类型异常：先做显式转换；无法转换时保留异常明细。
- 文件损坏或密码保护：说明无法读取，并请求可用版本或密码。
- 结果校验不通过：不得交付为最终结果，应返回问题清单。
- 输出失败：保留中间结果，说明失败原因与建议路径。

## 交付契约

- 所有交付文件写入工作区 `output/` 下的明确路径。
- 不覆盖用户原始文件，除非用户明确要求。
- 汇报必须包含：输入范围、处理动作、关键结果、输出文件、风险提示。
- 多文件交付应打包为 ZIP；单文件可直接给出本地文件链接。
- 图表最多两个子图，避免无关装饰，优先保证可读性。

## 最小能力边界

本示例为可安装 Skill 包骨架，侧重描述触发与执行规范；具体代码可由运行环境按任务动态生成。
