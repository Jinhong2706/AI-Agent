# knowledge-base

最小可安装 Skill 包示例，用于演示“知识库检索与沉淀助手”的注册、触发与交付规范。

## 文件组成

- `SKILL.md`：技能主体说明，包含可解析 YAML front matter、触发条件、搜索/写入/引用规范、隐私边界、异常处理与交付契约。
- `package.json`：最小包元数据，声明包名、版本、描述、许可证和可发布文件。

## 适用场景

- 在授权知识库中检索资料并形成中文结论。
- 基于已有文档生成带来源引用的摘要、FAQ、复盘或操作说明。
- 在用户明确确认后，将结构化知识条目写入知识库。
- 对来源冲突、资料过期、隐私敏感等情况进行风险提示。

## 安装与验证

将本目录作为一个最小 Skill 包安装或加载即可。安装前建议验证：

1. `SKILL.md` 顶部 YAML front matter 能被 YAML 解析器正常解析。
2. `package.json` 是合法 JSON。
3. 包内文件仅依赖 `SKILL.md` 与 `README.md`，没有外部运行依赖。

## 使用边界

- 仅处理用户授权范围内的资料。
- 不绕过权限，不输出或沉淀无业务必要的敏感信息。
- 写入知识库前需确认内容真实性、适用范围、来源与维护信息。
- 对无法验证的信息应标记为“待验证”，不得包装为确定事实。
