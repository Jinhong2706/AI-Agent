# Office Dress Code Guide — 办公室穿搭指南

Use when the user mentions 穿搭、职场穿衣、办公室着装、dress code、商务着装、穿什么合适、场合穿搭, or asks for workplace outfit advice. NOT for 时尚博主内容、奢侈品推荐、购物清单、婚礼礼服.

## 描述

根据不同职场场合（客户会议/团建/面试/日常办公），结合季节、行业和个人风格，推荐得体的职场穿搭方案，避免"穿错衣"的尴尬，在专业形象和个人风格之间找到平衡。

## 重要限制（请提前告知用户）

- **需要场景信息**：不同场合着装要求天差地别，需明确场景
- **个人偏好为主**：推荐仅供参考，最终以个人舒适度和公司文化为准
- **不推荐具体品牌/单品**：提供风格方向而非购物链接
- **文化差异**：不同行业/地区/公司着装文化不同，需了解背景

## 能力

- 根据场合推荐穿搭方案（正式/商务休闲/休闲）
- 提供季节性搭配建议
- 分析不同行业的着装文化
- 给出避雷指南（什么不能穿）
- 提供基础衣橱构建建议
- 针对特殊场合（面试/年会/出差）的专属方案

## 执行步骤

### Step 1: 明确场景
1. 确认具体场合和目的
2. 了解行业和公司文化
3. 确认性别/季节/天气

### Step 2: 推荐方案
1. 确定着装等级（正式/半正式/休闲）
2. 搭配上下装+鞋+配饰
3. 提供2-3个方案选择

### Step 3: 避雷提示
1. 指出该场合的穿搭禁忌
2. 提供"安全牌"兜底方案
3. 补充细节建议（颜色/面料/配饰）

## 输出原则

1. **场合适配**：穿搭建议与具体场合高度匹配
2. **实用为主**：推荐容易买到和搭配的基础款
3. **避雷优先**：先说不能穿什么，再说可以穿什么
4. **多方案选择**：提供2-3个不同风格选项
5. **包容多元**：尊重个人风格，不强加单一审美标准

## 常见问题（FAQ）

**Q: 互联网公司有dress code吗？**
A: 通常没有严格规定，但面试/见客户时建议smart casual。日常T恤牛仔即可。

**Q: 什么是business casual？**
A: 比正装休闲一级：男生衬衫+休闲裤+乐福鞋；女生针织衫+烟管裤+平底鞋。

**Q: 面试穿正装会不会太over？**
A: 取决于行业。金融/咨询/法律面试正装合适，互联网/创意行业可以smart casual。

**Q: 团建穿太随意会不会不好？**
A: 看团建类型。户外运动穿运动装，聚餐类保持整洁即可，避免太邋遢。

**Q: 出差一周怎么少带多搭？**
A: 基础色系（黑白灰藏蓝）可以互相搭配，5天只带8件就够。

**Q: 年会穿什么？**
A: 比日常隆重一级即可。不需要红毯礼服，但可以有细节亮点。

## 最佳实践

1. **了解公司文化**：观察同事和领导的着装，取中间偏上水平
2. **基础衣橱先行**：投资5-8件基础款百搭单品
3. **合身最重要**：再好的衣服不合身都减分，必要时改裁
4. **提前准备**：重要场合前一天确认穿搭，避免早上手忙脚乱
5. **一个亮点**：整体低调+一个点睛之笔（领带/丝巾/胸针），既安全又有个性

| 工装/制服选择 | 属于采购范畴 | 咨询制服供应商 |

### 常见误用

- **误用 1**：不说场合就问穿什么 → 着装离开场合无法建议
- **误用 2**：期望推荐具体购物链接 → 提供方向而非导购
- **误用 3**：完全照搬不考虑个人风格 → 建议在推荐基础上保留个性
