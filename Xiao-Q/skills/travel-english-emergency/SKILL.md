# Travel English Emergency — 旅游英语急救

Use when the user mentions 旅游英语、出国英语、travel English、出国旅游对话, or asks for English phrases for traveling abroad. NOT for 系统英语学习、商务英语、雅思备考、翻译服务.

## 描述

按旅游场景（点餐/问路/住酒店/就医/购物/紧急求助）提供即学即用的英语表达，附中文谐音和发音提示，解决"出国旅游语言不通、关键时刻说不出口"的问题。

## 重要限制（请提前告知用户）

- **应急级别**：提供的是即用型短句，非系统化英语教学
- **发音近似**：谐音标注只能近似，建议配合翻译APP使用
- **地区差异**：英美澳等国表达习惯可能不同
- **复杂场景有限**：涉及法律/医疗等复杂场景建议求助使馆或专业翻译

## 常见问题（FAQ）

**Q: 完全不会英语能出国吗？**
A: 可以。有翻译APP+基础手势+关键词卡基本够用。但掌握20句核心表达会让旅程顺畅很多。

**Q: 谐音标注准确吗？**
A: 是近似的，不完全准确。但足够让对方听懂你的意思。

**Q: 最重要的几句是什么？**
A: Hello / Thank you / Excuse me / How much? / Where is...? / Help! 这6句覆盖90%紧急场景。

**Q: 不会英语买东西怎么办？**
A: 指+计算器显示数字+翻译APP。大部分店员都习惯了。

**Q: 打车怎么沟通目的地？**
A: 直接给司机看地图APP上的目的地地址，最简单有效。

## 最佳实践

1. **出发前准备**：针对行程场景（机场/酒店/餐厅）各学5句
2. **离线翻译包**：下载Google翻译的离线包，没网也能用
3. **关键词截图**：把关键表达截图存手机相册，随时翻看
4. **大胆开口**：发音不标准没关系，配合手势对方能理解
5. **学会微笑**：礼貌和微笑是全球通用语言

- **误用 3**：用于商务/学术场合 → 旅游英语风格较随意
