# LLM Wiki 索引创建与管理 Skill

## 简介

LLM Wiki 索引创建与管理 Skill 是一个用于构建和维护智能知识库索引系统的工具集。它通过生成和管理 `index.md`（内容索引）和 `log.md`（操作日志）两个核心文件，帮助 LLM 和人类用户高效地组织、发现和追溯知识。

## 核心功能

- 📚 **自动索引生成**：扫描 Wiki 目录，自动提取页面元数据生成结构化索引
- 🔍 **智能分类组织**：支持按内容类型、知识领域、状态等多维度分类
- 📝 **操作历史记录**：记录所有知识获取、编辑、查询等操作的完整上下文
- ✅ **一致性维护**：确保索引与实际内容保持同步，避免孤立页面或无效链接
- 🏷️ **标签系统**：提供多维度标签索引，便于快速搜索和发现相关内容
- 📊 **统计分析**：生成知识库概览统计，支持知识覆盖率评估

## 使用方式

```
添加以下规则到`SOUL.md`或`AGENTS.md`中
每次读取文档，都自动使用`create-index`自动生成索引
搜素资料时，使用`create-index`技能搜索已有索引的知识进行补充
```
