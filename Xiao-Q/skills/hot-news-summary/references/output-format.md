## 点评风格模式

每条热点采用**三层洞察模型**输出，并支持用户选择点评风格：

### 三层洞察结构

| 层级 | 含义 | 字数 | 示例 |
|------|------|------|------|
| **What** | 事件本质（一句话定义） | 15-25字 | 某科技公司发布自研芯片，性能对标国际旗舰 |
| **So What** | 为什么重要（影响链） | 30-50字 | 标志国产芯片从设计到量产闭环突破，可能重塑手机供应链格局，降低对外依赖 |
| **Now What** | 后续关注什么（预判） | 20-35字 | 关注下游厂商采用进度及美国出口管制政策的可能调整 |

### 四种点评风格

用户可指定风格，未指定则默认使用「严肃分析」：

| 风格 | 调性 | What 风格 | So What 风格 | Now What 风格 |
|------|------|-----------|-------------|--------------|
| **严肃分析** | 客观、专业、克制 | 陈述事实，不加修饰 | 分析因果链条与结构性影响 | 指出后续关键变量 |
| **毒舌吐槽** | 犀利、反讽、解构 | 一针见血指出荒诞 | 揭示表面之下的真实逻辑 | 冷幽默预判走向 |
| **投资视角** | 机会导向、数据驱动 | 点明商业本质 | 分析产业链受益方与估值影响 | 提示关注相关标的与时间节点 |
| **吃瓜八卦** | 轻松、八卦、猎奇 | 爆料式引入 | 八卦内幕与人物关系 | 悬念式预告后续剧情 |

## 群聊格式

### 适用场景
- 微信群、钉钉群、飞书群等即时通讯工具
- 需要快速浏览、简洁传达的场景
- 移动端阅读

### 格式模板

```
🔥 今日热点摘要 {日期} | {风格标签}

━━━━━━━━━━━━━━━━━━

📌 {热点标题1}
📊 共振指数：{跨平台数}平台 | 综合评分：{得分}
💡 What：{一句话本质}
🔍 So What：{影响链}
🔜 Now What：{后续关注}
🔗 {链接}

━━━━━━━━━━━━━━━━━━

📌 {热点标题2}
...

（共10条）

━━━━━━━━━━━━━━━━━━
📈 数据来源：微博/知乎/B站/抖音/百度/36氪/少数派/贴吧（{实际成功数}平台）
⏰ 更新时间：{时间}
```

### 完整示例（严肃分析风格）

```
🔥 今日热点摘要 2025-05-14 | 严肃分析

━━━━━━━━━━━━━━━━━━

📌 某科技公司发布自研芯片
📊 共振指数：5平台 | 综合评分：89.2
💡 What：国产芯片首次实现设计-量产全链路突破
🔍 So What：降低手机产业链对外依赖，可能重塑全球芯片供应格局
🔜 Now What：关注下游厂商采用进度及出口管制政策调整
🔗 https://s.weibo.com/...

━━━━━━━━━━━━━━━━━━

📌 2025年经济形势讨论
📊 共振指数：4平台 | 综合评分：82.5
💡 What：消费复苏与就业压力并存成为核心议题
🔍 So What：反映政策端稳增长与调结构的两难抉择
🔜 Now What：关注后续政策信号及PMI数据走势
🔗 https://www.zhihu.com/...

━━━━━━━━━━━━━━━━━━
📈 数据来源：微博/知乎/B站/抖音/百度/36氪/少数派/贴吧（8平台）
⏰ 更新时间：2025-05-14 14:30
```

### 格式要点
- 标题使用火焰emoji + 风格标签，一眼识别
- 每条热点展示共振指数和综合评分，体现筛选逻辑
- 三层洞察用不同emoji引导，层次分明
- 分割线区分每条热点，移动端阅读体验好
- 总长度控制在1500字以内
- 结尾标注实际成功抓取的平台数

## 可视化看板格式

### 适用场景
- 需要一图掌握全貌的场景
- 会议演示、汇报材料
- 大屏监控、信息周知

### 格式模板

生成以下ECharts图表组合：

1. **热度TOP10柱状图**
   - X轴：热点标题（缩略）
   - Y轴：综合评分
   - 颜色按话题领域区分
   - 标注共振指数

2. **平台贡献饼图**
   - 展示各平台入选条数占比
   - 标注平台状态（正常/降级/失败）

3. **话题领域雷达图**
   - 各领域入选数量
   - 直观显示今日热点的领域偏重

4. **跨平台共振气泡图**
   - X轴：热度标准化分
   - Y轴：共振平台数
   - 气泡大小：综合评分
   - 一眼识别"高热度+高共振"的双高热点

### 实现方式
使用 ECharts skill 生成图表，数据来源于 `hot_lists.json` 中的评分结果。

## 语音播报格式

### 适用场景
- 通勤、运动等无法看屏的场景
- 配合 TTS 生成2-3分钟音频
- 播客/电台式体验

### 格式模板

```
【开场白】
大家好，今天是{日期}，以下是今日全网热点速览。

【正文】每条热点约15秒
第{序号}条，{标题}。
核心信息：{What}。
为什么重要：{So What}。
后续值得关注：{Now What}。

【结尾】
以上就是今天的全部热点。数据来自{平台数}大平台，{降级提示}。
我们明天见。
```

### 播报节奏
- 开场白：10秒
- 每条热点：12-15秒（What 5秒 + So What 6秒 + Now What 4秒）
- 结尾：10秒
- 总时长：约2分30秒 - 3分钟

## 格式选择建议

| 场景 | 推荐格式 | 推荐风格 |
|------|---------|---------|
| 微信/钉钉群快速分享 | 群聊格式 | 严肃分析 |
| 飞书/Slack团队资讯 | 群聊格式 | 严肃分析/投资视角 |
| Notion/Obsidian归档 | 知识库格式 | 严肃分析 |
| 汇报演示/PPT素材 | 可视化看板 | 严肃分析 |
| 通勤听热点 | 语音播报 | 吃瓜八卦/毒舌吐槽 |
| 投资决策参考 | 知识库格式 | 投资视角 |
| 朋友闲聊分享 | 群聊格式 | 毒舌吐槽/吃瓜八卦 |

### 混合使用建议
若用户未指定格式和风格：
1. 默认输出**群聊格式 + 严肃分析风格**
2. 在输出末尾提示：「如需详细版本，请说明"知识库格式"；如换口味，试试"毒舌吐槽"/"投资视角"/"吃瓜八卦"风格」
3. 如用户仅指定风格不指定格式，按默认格式+指定风格输出
4. 如用户仅指定格式不指定风格，按指定格式+默认风格输出
