# 多渠道内容分发优化器 — 产品文档

## 一、产品概述

### 1.1 产品定位

多渠道内容分发优化器是一款面向内容创作者和运营团队的 AI Agent 技能工具，旨在解决「一篇核心内容需要适配多个社交媒体平台」这一高频痛点。通过自动化的内容分析、平台适配和格式转换，将传统需要数小时的手工操作压缩至分钟级别。

### 1.2 目标用户

| 用户类型 | 使用场景 | 核心需求 |
|---------|---------|---------|
| 内容创作者 | 个人自媒体多平台运营 | 一稿多投，节省创作时间 |
| 内容运营团队 | 企业品牌多渠道传播 | 统一内容质量，提升分发效率 |
| MCN机构 | 矩阵账号管理 | 批量内容适配，规模化运营 |
| 市场营销部门 | 品牌推广活动 | 快速生成多平台营销素材 |

### 1.3 核心价值

- **效率提升**：将单篇内容的多平台适配时间从 2-4 小时压缩至 10-15 分钟
- **质量保障**：基于各平台算法偏好的科学适配，而非经验驱动的人工调整
- **格式规范**：严格遵守各平台格式要求，避免因格式问题导致的限流或审核失败
- **策略一致**：确保同一内容在不同平台传递一致的核心信息

## 四、技术特性

### 4.1 内容分析能力

| 能力 | 描述 | 技术实现 |
|------|------|---------|
| 内容类型识别 | 自动判断科普/观点/教程/评测/故事 | 关键词频率分析 |
| 语气风格检测 | 识别专业/轻松/正式/通俗风格 | 特征词匹配 |
| 数据元素检测 | 识别文本中的数据、百分比、统计信息 | 正则表达式匹配 |
| 核心信息提取 | 提取3-7个核心论点或信息节点 | 多策略提取算法 |
| 适配评分 | 评估内容适配多平台的适合度 | 多因子加权评分 |

### 4.2 平台适配能力

| 适配维度 | 描述 |
|---------|------|
| 字数适配 | 根据目标平台字数范围自动压缩或扩展 |
| 结构重组 | 按平台偏好重新组织内容结构 |
| 标题生成 | 按平台标题风格生成2-3个备选标题 |
| 标签策略 | 按平台规则生成标签组合 |
| 配图规划 | 根据内容主题生成配图数量、尺寸、风格建议 |

### 4.3 验证检查能力

| 检查项 | 描述 |
|--------|------|
| 平台覆盖率 | 确保覆盖至少4个目标平台 |
| 字数合规 | 各版本字数在目标平台允许范围内 |
| 标签合规 | 标签数量和格式符合平台要求 |
| 标题质量 | 标题具有平台特征和吸引力 |
| 敏感词检测 | 标记可能的违规词汇 |
| 内容保真 | 核心信息在各版本中保留 |

## 五、使用指南

### 5.1 基本使用流程

1. 准备一篇核心内容（文章/脚本/图文）
2. 输入触发词（如「多平台分发」「内容适配」）+ 提供内容
3. AI Agent 自动执行五步工作流
4. 获取结构化分发报告
5. 按建议发布到各平台

### 5.2 输入方式

- **直接粘贴**：将文章文本直接输入对话
- **文件路径**：提供 Markdown/TXT 文件路径
- **URL 链接**：提供文章 URL（自动抓取内容）

### 5.3 输出说明

输出为结构化 Markdown 报告，包含：
- 各平台完整适配版本
- 标题备选列表（每平台3个）
- 配图建议清单
- 发布时间建议
- 质量评分卡
- 发布排期建议

## 七、常见问题

**Q: 适配后的内容会改变原意吗？**
A: 不会。内容适配遵循「保真原则」，核心观点和关键数据在所有平台版本中保持一致，仅调整表述方式和叙事结构。

**Q: 支持哪些内容语言？**
A: 当前版本主要支持中文内容。英文内容的适配支持将在后续版本中添加。

**Q: 可以只选择部分平台适配吗？**
A: 可以。在输入内容时，可以指定目标平台列表，仅生成选中平台的适配版本。

**Q: 适配建议是否可以直接发布？**
A: 建议在发布前人工审核，特别是数据准确性和品牌调性方面。AI生成的适配版本是高质量的初稿，人工审核可以进一步提升质量。
