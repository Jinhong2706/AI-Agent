# 多平台内容分发标准参考手册

> 版本：1.0.0 | 更新日期：2026-05-27
> 本手册为多渠道内容分发优化器的核心参考资料，涵盖六大中文主流社交媒体平台的内容格式规范。

---

## 一、微信公众号

### 1.1 基本规范

| 项目 | 规范要求 |
|------|---------|
| 文章字数 | 建议 1500-3000 字，最低不少于 800 字 |
| 标题长度 | 18-30 字，不超过 64 字（含标点） |
| 封面图尺寸 | 900×383px（首条）/ 200×200px（次条） |
| 正文配图 | 建议 3-8 张，宽度 900px，JPG/PNG 格式 |
| 排版建议 | 段落不超过 200 字，行间距 1.5-2.0 倍 |

### 1.2 内容风格

- **定位**：深度阅读场景，用户期望获取有价值的信息
- **结构要求**：引言（背景铺垫）→ 分论点展开（3-5个小标题）→ 总结升华
- **语言风格**：专业但易懂，可适当使用比喻和案例
- **SEO 优化**：标题含核心关键词，正文前 300 字含 2-3 个长尾关键词
- **互动设计**：文末设置互动话题，引导留言讨论

### 1.3 标题公式

- 数字型：「5 个关键趋势告诉你 XXX 的未来」
- 疑问型：「为什么 XXX 越来越重要？」
- 对比型：「普通人 vs 专业人士：XXX 的认知差距」
- 干货型：「一文讲透 XXX：从入门到精通」

### 1.4 最佳发布时间

| 时段 | 流量等级 | 适用内容 |
|------|---------|---------|
| 07:30-08:30 | ⭐⭐⭐ | 资讯速递、行业早报 |
| 12:00-13:00 | ⭐⭐⭐⭐ | 轻松阅读、知识分享 |
| 17:30-18:30 | ⭐⭐⭐ | 行业分析、深度解读 |
| 20:00-22:00 | ⭐⭐⭐⭐⭐ | 深度长文、专题策划 |

### 1.5 算法偏好

- 完读率是核心权重指标（用户阅读到文章底部）
- 分享率影响推荐量（社交裂变传播）
- 原创标记的内容有额外流量扶持
- 标题点击率影响初始推荐量

## 二、小红书

### 2.1 基本规范

| 项目 | 规范要求 |
|------|---------|
| 笔记字数 | 建议 300-800 字，最高 1000 字 |
| 标题长度 | 15-25 字，必须吸引眼球 |
| 首图尺寸 | 1080×1440px（3:4 竖版），JPG 格式 |
| 图片数量 | 建议 5-9 张，最少 1 张 |
| 标签数量 | 5-8 个标签，含热门标签+垂直标签 |
| Emoji 使用 | 建议每段 1-2 个，增强视觉效果 |

### 2.2 内容风格

- **定位**：种草社区，用户追求实用、真实、好看的内容
- **结构要求**：吸睛标题 → 要点列表（编号+Emoji）→ 总结 + 互动引导
- **语言风格**：第一人称视角、口语化、接地气
- **视觉要求**：首图是核心竞争力，需精心设计
- **互动设计**：文末引导点赞收藏，评论区主动回复

### 2.3 标题公式

- 符号型：「🔥 必看！XXX 干货合集」
- 情绪型：「救命！这个 XXX 太好用了」
- 数字型：「5 个 XXX 技巧，建议收藏！」
- 身份型：「作为 XXX 从业者，我想说…」

### 2.4 标签策略

| 标签类型 | 占比 | 示例 |
|---------|------|------|
| 热门标签 | 20-30% | #干货分享 #涨知识 #好物推荐 |
| 垂直标签 | 50-60% | #科技数码 #AI工具 #效率提升 |
| 长尾标签 | 10-20% | #2026年AI发展趋势 |

### 2.5 最佳发布时间

| 时段 | 流量等级 | 适用内容 |
|------|---------|---------|
| 12:00-13:00 | ⭐⭐⭐⭐ | 午休碎片阅读 |
| 18:00-20:00 | ⭐⭐⭐⭐⭐ | 下班后浏览高峰 |
| 21:00-23:00 | ⭐⭐⭐⭐⭐ | 深夜种草场景 |

### 2.6 算法偏好

- 首图点击率（CES 评分核心指标）
- 收藏率权重高于点赞率
- 笔记发布后 2 小时内互动数据决定推荐量
- 搜索流量占比高，关键词布局很重要

## 三、抖音

### 3.1 基本规范

| 项目 | 规范要求 |
|------|---------|
| 视频时长 | 建议 30秒-3分钟，短视频 15-60秒效果最佳 |
| 脚本字数 | 100-300 字（对应 30秒-2分钟口播） |
| 画面比例 | 9:16 竖版，1080×1920px |
| 标题/文案 | 30 字以内，配合 3-5 个话题标签 |
| BGM | 必须搭配背景音乐，推荐使用平台热门音乐 |

### 3.2 内容风格

- **定位**：短视频娱乐+知识平台，用户注意力窗口约 3 秒
- **结构要求**：3秒钩子 → 核心观点展开 → 反转/惊喜 → 行动号召
- **语言风格**：口语化、快节奏、情绪饱满
- **互动设计**：视频结尾引导点赞评论，评论区置顶互动问题

### 3.3 钩子公式（前 3 秒）

- 悬念型：「你知道 XXX 的真相是什么吗？」
- 反常识型：「所有人都以为 XXX，但其实…」
- 利益型：「学会这个 XXX，效率提升 10 倍」
- 共鸣型：「如果你也有 XXX 的困惑，一定要看完」

### 3.4 最佳发布时间

| 时段 | 流量等级 | 适用内容 |
|------|---------|---------|
| 07:00-09:00 | ⭐⭐⭐ | 早通勤碎片时间 |
| 12:00-13:00 | ⭐⭐⭐⭐ | 午休时间 |
| 19:00-21:00 | ⭐⭐⭐⭐⭐ | 晚间娱乐高峰 |
| 21:00-23:00 | ⭐⭐⭐⭐ | 睡前刷手机 |

### 3.5 算法偏好

- 完播率是最核心指标（用户看完视频的比例）
- 前 3 秒留存率决定初始推荐池
- 互动率（点赞+评论+转发+收藏）综合评估
- 趋势话题流量加成明显

## 四、B站

### 4.1 基本规范

| 项目 | 规范要求 |
|------|---------|
| 视频时长 | 建议 5-15 分钟，分P内容每P 8-12 分钟 |
| 脚本字数 | 800-2000 字（对应 5-10 分钟口播） |
| 画面比例 | 16:9 横版，1920×1080px |
| 封面图 | 1920×1080px 或 1146×717px |
| 标签数量 | 5-10 个标签 |
| 分区选择 | 根据内容选择对应分区（科技/知识/生活等） |

### 4.2 内容风格

- **定位**：中长视频知识平台，用户追求深度和趣味性并重
- **结构要求**：开场互动（抛问题/做预告）→ 知识点逐个展开 → 总结 + 互动引导
- **语言风格**：专业但不枯燥，适当加入梗和网络用语
- **弹幕设计**：在知识点衔接处设置「弹幕触发点」（如提问、感叹）

### 4.3 标题公式

- 括号型：「XXX 的真相（看完震惊了）」
- 系列型：「【XXX系列】第三期：深入解析 XXX」
- 对话型：「聊一聊 XXX，这件事没那么简单」
- 悬念型：「这个 XXX，90% 的人都理解错了」

### 4.4 最佳发布时间

| 时段 | 流量等级 | 适用内容 |
|------|---------|---------|
| 17:00-19:00 | ⭐⭐⭐⭐ | 放学下班后 |
| 20:00-23:00 | ⭐⭐⭐⭐⭐ | 晚间黄金时段 |
| 周末全天 | ⭐⭐⭐⭐ | 长视频观看高峰 |

### 4.5 算法偏好

- 投币率和收藏率是核心推荐指标
- 弹幕密度影响推荐（互动活跃度）
- 完播率权重低于互动率（与抖音不同）
- 一键三连（点赞+投币+收藏）综合评估
- 系列内容有持续推荐加权

## 五、知乎

### 5.1 基本规范

| 项目 | 规范要求 |
|------|---------|
| 回答字数 | 建议 2000-5000 字，深度回答 3000 字以上 |
| 文章字数 | 2000-8000 字 |
| 标题/开头 | 15-30 字，直接切入问题核心 |
| 图片 | 数据图表、引用截图为主，辅助论据 |
| 引用标注 | 数据来源必须标注，引用格式规范 |
| 话题选择 | 1-3 个相关话题 |

### 5.2 内容风格

- **定位**：高质量问答社区，用户追求专业、有深度的内容
- **结构要求**：背景引入（定义问题）→ 问题分析（分层论述）→ 论据展开（数据+案例）→ 结论建议
- **语言风格**：理性客观、逻辑清晰、论据充分
- **引用规范**：数据标注来源，引用学术文献使用标准格式

### 5.3 开头公式

- 直接型：「关于 XXX，我的看法是…」
- 经验型：「作为 XXX 领域 5 年从业者，我来分享一下…」
- 数据型：「根据最新数据，XXX 在过去一年增长了 XX%…」
- 反驳型：「这个问题下很多回答存在误区，我来纠正一下…」

### 5.4 最佳发布时间

| 时段 | 流量等级 | 适用内容 |
|------|---------|---------|
| 08:00-10:00 | ⭐⭐⭐⭐ | 上班前碎片阅读 |
| 12:00-14:00 | ⭐⭐⭐⭐ | 午休深度阅读 |
| 20:00-23:00 | ⭐⭐⭐⭐⭐ | 晚间深度阅读高峰 |

### 5.5 算法偏好

- 赞同数和感谢数是核心指标
- 内容质量分（盐值体系）影响推荐
- 专业认证用户有额外权重
- 长回答（3000字+）在搜索中排名更高
- 专栏文章有独立的推荐机制

## 七、跨平台分发注意事项

### 7.1 发布顺序建议

1. **知乎/B站**（深度内容优先，建立专业背书）
2. **微信公众号**（长图文首发，沉淀私域流量）
3. **小红书/抖音**（精简版分发，扩大触达范围）
4. **微博**（提炼观点，蹭热点传播）

### 7.2 间隔时间建议

- 同一内容在不同平台发布，建议间隔 **2-4 小时**
- 系列内容在各平台保持 **1-2 天** 的间隔节奏
- 热点时效性内容建议在 **24 小时内** 全平台覆盖

### 7.3 通用敏感词规避

以下词汇在各平台均需谨慎使用，可能触发审核或限流：

- 营销类：加微信、私聊、转账、红包、免费领取
- 夸大类：第一、唯一、绝对、100%、保证
- 医疗类：治愈、根治、药到病除、特效
- 金融类：稳赚、翻倍、零风险、保本

### 7.4 内容质量基准

| 维度 | 最低标准 | 优秀标准 |
|------|---------|---------|
| 原创性 | 核心观点独立 | 表述方式完全原创 |
| 信息密度 | 每段至少1个信息点 | 每段2-3个信息点 |
| 逻辑连贯 | 段落之间有逻辑关联 | 全文形成论证闭环 |
| 可读性 | 无错别字，语句通顺 | 排版美观，节奏感强 |
| 互动设计 | 有基本互动引导 | 针对平台定制互动策略 |
