# Utility Bill Estimator — 水电费估算

Use when the user mentions 水电费、电费估算、用电量、utility bill、electricity estimate、water bill, or asks to estimate utility costs. NOT for 缴费、节能改造、电器维修.

## 描述

根据用户提供的历史用量数据和家电使用习惯，预测当月水电燃气费用，分析用量趋势和异常波动，提供针对性节能建议，帮助用户合理控制家庭能源开支，避免月底账单超预期。

## 重要限制（请提前告知用户）

- **需要历史数据**：需要提供往期用量或费用作为计算基础
- **估算有误差**：预测基于历史规律，实际费用以账单为准（误差约10-20%）
- **不能直接缴费**：只做估算和分析，缴费需在对应平台自行操作
- **电价因地区而异**：不同地区、不同阶梯电价差异大，请提供所在地信息


## 能力

- 基于历史数据预测当月水电费
- 分析季节性用量变化趋势
- 计算阶梯电价/水价
- 估算主要家电的耗电量
- 识别异常用量波动
- 提供节能省钱建议

## 执行步骤

### Step 1: 收集数据
1. 获取近3-6个月用量/费用数据
2. 了解家庭人数和主要家电
3. 确认所在地区和电价政策
4. 了解本月特殊因素（空调/电暖/新增电器）

### Step 2: 估算预测
1. 计算历史月均用量
2. 考虑季节因素调整
3. 加入特殊因素影响
4. 计算阶梯费用

### Step 3: 分析建议
1. 对比历史趋势
2. 分析用量构成
3. 识别节能空间
4. 给出具体省钱建议

## 常见问题（FAQ）

**Q: 一度电多少钱？**
A: 因地区和阶梯不同，一般0.5-0.9元/度。第一档最便宜，超量进入第二/三档会贵。

**Q: 空调一天耗多少电？**
A: 1.5匹空调制冷大约1-1.5度/小时，开8小时约8-12度。变频空调会省一些。

**Q: 怎么知道哪个电器费电？**
A: 看电器铭牌上的功率(W)，功率÷1000×使用小时数=度数。如2000W电暖器用4小时=8度。

**Q: 为什么夏天电费特别高？**
A: 主要是空调。空调用电通常占夏季总电费的60-70%。

**Q: 阶梯电价怎么算？**
A: 年用电量分三档，分别有不同单价。超出第一档后整体电费会明显增加。

**Q: 有没有简单的省电方法？**
A: 最有效：空调设26-28度、冰箱适当调高温度、随手关灯、待机电器拔插头。

## 最佳实践

1. **记录每月数据**：养成每月记录用量的习惯，建立趋势基线
2. **关注阶梯档位**：接近档位临界值时适当控制用量
3. **季节性预警**：夏冬用电高峰提前做好心理准备
4. **定期检查**：异常波动可能是电器故障或漏水漏电
5. **节能投资**：高耗电电器考虑更换为一级能效

## 不适用场景

### 常见误用

- **误用 1**：只提供一个月数据就要精准预测 → 至少需要3个月数据才有参考价值
- **误用 2**：把估算当作实际账单 → 估算仅供参考，缴费以实际为准
- **误用 3**：忽略季节因素 → 夏冬和春秋的差异可能达2-3倍
