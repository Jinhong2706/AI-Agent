# Windows 11 快捷键大全

## 系统级快捷键

### 开始菜单与任务栏
- `Win` - 打开/关闭开始菜单
- `Win + X` - 打开高级用户菜单（右键开始菜单）
- `Win + T` - 在任务栏程序间切换
- `Win + 数字键(1-9)` - 启动任务栏对应位置的程序
- `Win + Alt + 数字键` - 打开任务栏程序的跳转列表
- `Win + B` - 聚焦到系统托盘
- `Win + D` - 显示桌面
- `Win + M` - 最小化所有窗口
- `Win + Shift + M` - 还原最小化的窗口
- `Win + Home` - 最小化/还原除当前窗口外的所有窗口

### 窗口管理（Snap Layouts）
- `Win + Z` - 打开 Snap Layouts 布局选择器
- `Win + 方向键` - 窗口贴边排列
  - `Win + ←` - 窗口贴左
  - `Win + →` - 窗口贴右
  - `Win + ↑` - 窗口最大化/贴顶
  - `Win + ↓` - 窗口还原/最小化
- `Win + Shift + 方向键` - 将窗口移动到其他显示器

### 虚拟桌面
- `Win + Tab` - 打开任务视图（虚拟桌面+时间线）
- `Win + Ctrl + D` - 创建新虚拟桌面
- `Win + Ctrl + ←/→` - 切换虚拟桌面
- `Win + Ctrl + F4` - 关闭当前虚拟桌面
- `Win + Ctrl + 数字键` - 切换到指定虚拟桌面

### 系统设置与搜索
- `Win + S` 或 `Win` 后直接输入 - 打开搜索
- `Win + I` - 打开设置应用
- `Win + A` - 打开快速设置面板
- `Win + N` - 打开通知中心和日历
- `Win + K` - 打开投影/连接面板
- `Win + P` - 投影模式切换（仅第二屏/复制/扩展）
- `Win + W` - 打开小组件面板
- `Win + C` - 打开 Microsoft Teams（若已安装）
- `Win + H` - 打开语音输入（听写）

### 截图与录屏
- `Win + Shift + S` - 打开截图工具（区域截图）
- `Win + PrtScn` - 全屏截图并保存到图片文件夹
- `Win + Alt + R` - 开始/停止游戏栏录屏
- `Win + Alt + G` - 录制过去30秒的游戏片段
- `Win + Alt + PrtScn` - 游戏栏截图

### 辅助功能
- `Win + U` - 打开辅助功能设置
- `Win + +` - 打开放大镜并放大
- `Win + -` - 放大镜缩小
- `Win + Esc` - 关闭放大镜
- `Win + Ctrl + Enter` - 开启/关闭讲述人
- `Win + Enter` - 开启/关闭讲述人（旧版）

### 文件资源管理器
- `Win + E` - 打开文件资源管理器
- `Win + R` - 打开运行对话框
- `Ctrl + Shift + Esc` - 打开任务管理器
- `Win + Pause/Break` - 打开系统属性

### 表情符号与符号
- `Win + .` 或 `Win + ;` - 打开表情符号面板
- `Win + V` - 打开剪贴板历史（需先启用）

## 高级快捷键

### 命令行相关
- `Win + X` 后按 `A` - 以管理员身份打开 PowerShell
- `Win + X` 后按 `I` - 打开 PowerShell（普通权限）
- `Win + X` 后按 `T` - 打开终端（Windows Terminal）

### 微软应用
- `Win + Shift + S` - 截图工具
- `Win + G` - 打开 Xbox Game Bar
- `Win + Alt + Print Screen` - 游戏栏截图

## 鼠标与触控板手势

### 任务栏手势
- **中键点击任务栏图标** - 打开新实例
- **Shift + 点击任务栏图标** - 以管理员身份打开
- **右键点击任务栏图标** - 显示跳转列表

### 触控板手势（笔记本电脑）
- **三指点击** - 打开搜索
- **三指上滑** - 打开任务视图
- **三指下滑** - 显示桌面
- **三指左右滑** - 切换虚拟桌面
- **四指左右滑** - 切换虚拟桌面（部分设备）

## 隐藏的快捷键

### 文件资源管理器内
- `Alt + D` - 聚焦地址栏
- `Ctrl + E` 或 `Ctrl + F` - 聚焦搜索框
- `F2` - 重命名选中文件
- `F5` - 刷新
- `Alt + Enter` - 打开属性
- `Shift + Delete` - 永久删除（不进回收站）
- `Alt + P` - 显示/隐藏预览窗格

### 浏览器中（Edge/Chrome）
- `Ctrl + T` - 新标签页
- `Ctrl + Shift + T` - 恢复关闭的标签页
- `Ctrl + W` - 关闭当前标签页
- `Ctrl + Tab` - 下一个标签页
- `Ctrl + Shift + Tab` - 上一个标签页
- `Ctrl + 数字键(1-8)` - 切换到对应位置的标签页
- `Ctrl + 9` - 切换到最后一个标签页

## 效率提升组合

### 快速启动应用
创建自定义快捷键：
1. 右键应用快捷方式 → 属性
2. 快捷键栏设置组合键
3. 推荐使用 `Ctrl + Alt + 字母`

### 虚拟桌面工作流
- 桌面1：工作应用（浏览器、Office）
- 桌面2：通讯工具（微信、QQ）
- 桌面3：娱乐（视频、音乐）
- 使用 `Win + Ctrl + ←/→` 快速切换

### Snap Layouts 高级用法
1. 将窗口拖到屏幕顶部 → 显示布局选项
2. 将窗口拖到侧边 → 自动贴边并提示另一侧窗口
3. 使用 `Win + Z` 在已有窗口上选择布局

## 实用提示

- **Win 键是核心**：大多数 Windows 11 快捷键都以 Win 键开头
- **组合键顺序**：先按修饰键（Win/Ctrl/Alt/Shift），再按功能键
- **右键菜单**：按住 Shift 右键点击文件，显示扩展菜单（经典菜单）
- **快速设置**：`Win + A` 可以快速调整亮度、音量、WiFi、蓝牙等
