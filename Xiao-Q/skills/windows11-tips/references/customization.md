# Windows 11 个性化与外观定制

## 任务栏定制

### 1. 任务栏对齐方式
设置 → 个性化 → 任务栏 → 任务栏行为
- **对齐方式**：左对齐 或 居中（默认）
- 推荐：习惯 Windows 10 的用户选择左对齐

### 2. 隐藏/显示任务栏项目
设置 → 个性化 → 任务栏
- 隐藏的图标菜单
- 任务栏角标
- 搜索、任务视图、小组件、聊天按钮

### 3. 任务栏自动隐藏
设置 → 个性化 → 任务栏 → 任务栏行为
- 勾选"自动隐藏任务栏"
- 鼠标移到底部自动显示

### 4. 多显示器任务栏
设置 → 个性化 → 任务栏 → 任务栏行为
- 在所有显示器上显示任务栏
- 显示所有任务栏的任务窗口
- 合并任务栏按钮

## 开始菜单定制

### 1. 固定常用应用
- 右键应用 → 固定到开始
- 拖拽调整磁贴位置和大小
- 创建文件夹：拖拽一个磁贴到另一个上

### 2. 布局管理
- "推荐"部分：显示最近使用和推荐文件
- 可以右键取消固定或清除所有推荐
- 调整"固定"和"推荐"区域大小

### 3. 开始菜单颜色
设置 → 个性化 → 颜色
- 选择模式：浅色/深色/自定义
- 选择主题色：勾选"开始和任务栏"
- 透明效果：开启 Mica 效果

## 主题与颜色

### 1. 应用 Windows 主题
设置 → 个性化 → 主题
- 内置主题：Windows（默认）、深色、红色等
- 在线主题：浏览主题（需联网）
- 自定义主题：保存当前设置为新主题

### 2. 颜色配置
设置 → 个性化 → 颜色
- **浅色模式**：传统白色界面
- **深色模式**：护眼黑色界面
- **自定义模式**：Windows 界面和系统应用分开设置

### 3. Mica 效果
- 开启"透明效果"获得 Mica 材质
- 开始菜单、任务栏、设置应用均有半透明效果
- 需要显卡支持（大多数现代显卡都支持）

### 4. 强调色
- 手动选择：从色板选择喜欢的颜色
- 自动取色：根据壁纸自动生成强调色
- 应用到：开始菜单、任务栏、窗口边框、标题栏

## 壁纸与锁屏

### 1. 桌面壁纸
设置 → 个性化 → 背景
- 图片：选择单张图片
- 纯色：选择纯色背景
- 幻灯片放映：自动轮播多张图片
- 选择契合度：填充/适应/拉伸/平铺/居中/跨区

### 2. 锁屏界面
设置 → 个性化 → 锁屏界面
- 背景：Windows 聚焦（自动换壁纸）/图片/幻灯片
- 在锁屏上显示状态：邮件、日历、天气等
- 在登录屏幕上显示锁屏背景

### 3. 触控键盘个性化
设置 → 个性化 → 触控键盘
- 背景：图片/纯色
- 按键动画：开启/关闭
- 按键音：调整音量或关闭

## 字体与图标

### 1. 系统字体调整
设置 → 个性化 → 字体
- 安装新字体：拖拽字体文件到页面
- 调整字体大小：100%-225%
- 推荐字体：微软雅黑、Segoe UI Variable

### 2. 图标大小与间距
设置 → 辅助功能 → 文本大小
- 调整文本缩放：100%-225%
- 单独调整：标题栏、菜单、消息框等

### 3. 桌面图标管理
设置 → 个性化 → 主题 → 桌面图标设置
- 显示：计算机、回收站、用户的文件、控制面板、网络
- 更改图标：选择自定义图标
- 清理桌面：右键 → 查看 → 自动排列图标

## 右键菜单定制

### 1. 恢复经典右键菜单
参见 [hidden-features.md](hidden-features.md) 注册表修改方法

### 2. 添加右键菜单选项
使用工具：
- **Context Menu Editor**：图形化编辑右键菜单
- **Easy Context Menu**：功能强大的菜单管理工具
- **手动编辑注册表**：（高级用户）

### 3. 常用右键增强
- 右键 → 新建 → 文档类型定制
- 发送到菜单：自定义发送目标文件夹
- Shift + 右键：显示扩展菜单（包含更多选项）

## 动画与效果

### 1. 动画效果调整
设置 → 辅助功能 → 视觉效果
- 动画效果：开启/关闭窗口动画
- 透明效果：Mica 材质开关
- 自动隐藏滚动条：始终显示/自动隐藏

### 2. 性能与美观平衡
设置 → 系统 → 关于 → 高级系统设置
→ 性能设置 → 视觉效果
- 最佳外观：全部开启（美观但耗资源）
- 最佳性能：全部关闭（快速但简陋）
- 自定义：选择性开启（推荐）

### 3. 鼠标指针定制
设置 → 蓝牙和其他设备 → 鼠标 → 其他鼠标选项
→ 指针选项卡
- 方案：Windows 默认/Windows 反转/其他
- 自定义每个指针状态
- 启用指针阴影

## 声音与通知

### 1. 系统声音定制
设置 → 系统 → 声音 → 更多声音设置
→ 声音选项卡
- 方案：Windows 默认/无声/其他
- 自定义每个事件的提示音
- 下载更多声音方案

### 2. 通知样式
设置 → 系统 → 通知
- 通知横幅：开启/关闭
- 通知声音：开启/关闭
- 在操作中心显示通知：开启/关闭
- 优先级：高/普通/低

### 3. 焦点辅助
设置 → 系统 → 专注助手
- 自动规则：全屏/游戏/演示时
- 优先级列表：允许特定联系人通知
- 闹钟始终突破：重要闹钟不被屏蔽

## 多任务处理外观

### 1. Snap Layouts 外观
设置 → 系统 → 多任务处理
- Snap Layouts：开启/关闭
- 显示 Snap Layouts：当鼠标悬停在窗口最大化按钮上
- 显示对齐的组：在任务栏显示已对齐窗口组

### 2. 虚拟桌面定制
- 每个桌面可设置不同壁纸（需第三方工具）
- 任务视图背景：跟随系统主题
- 桌面名称：点击桌面名称可重命名

## 高级定制工具

### 1. PowerToys
微软官方工具集：
- FancyZones：高级窗口布局管理器
- File Explorer Preview Pane：文件预览增强
- Keyboard Manager：快捷键重映射
- PowerRename：批量文件重命名

### 2. Start11（付费）
- 完全自定义开始菜单
- 恢复经典样式
- 任务栏增强功能

### 3. ExplorerPatcher（免费开源）
- 恢复 Windows 10 任务栏
- 恢复经典右键菜单
- 调整开始菜单样式

### 4. Rainmeter（桌面美化）
- 桌面小部件：时钟、天气、系统监控
- 皮肤丰富：数千种免费皮肤
- 高度自定义：编辑 .ini 文件

## 快速切换个性化方案

### 创建个性化预设
1. 设置好所有个性化选项
2. 设置 → 个性化 → 主题
3. 点击"保存主题"
4. 命名并保存
5. 可在不同方案间快速切换

### 导出主题
- 右键已保存主题 → 导出
- 生成 .deskthemepack 文件
- 可分享给其他 Windows 11 用户

## 提示

🎨 **个性化建议：**
- 深色模式 + 蓝色强调色 = 专业感
- 浅色模式 + 红色强调色 = 活力感
- 透明效果 + 动态壁纸 = 未来感

⚡ **性能考虑：**
- 过多动画和透明效果会消耗更多资源
- 老旧电脑建议关闭部分视觉效果
- 使用纯色背景比动态壁纸更省电

🔧 **备份个性化设置：**
设置 → 账户 → Windows 备份 → 记住我的偏好
- 同步主题到所有设备
- 需要登录微软账户
