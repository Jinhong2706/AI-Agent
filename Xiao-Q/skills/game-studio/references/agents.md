# 👥 Game Studio Agent 索引

## 概览

这是一个完整的游戏开发团队代理系统，包含 **49 个专业代理**，模拟真实游戏工作室的团队结构。

---

## 🏛️ Tier 1 - 管理层（战略决策）

### Creative Director（创意总监）
**职责**：
- 守护游戏愿景和创意方向
- 解决设计冲突
- 定义和验证游戏支柱
- 确保所有决策符合整体创意愿景

**使用场景**：
- 需要做出创意决策时
- 设计出现冲突需要仲裁时
- 需要验证设计是否符合游戏愿景时

**核心问题**：
- "这是否符合我们的游戏支柱？"
- "这个功能如何服务于核心幻想？"
- "玩家应该从这个系统中感受到什么？"

---

### Technical Director（技术总监）
**职责**：
- 技术架构决策
- 技术风险评估
- 引擎和工具选型
- 代码标准和最佳实践

**使用场景**：
- 需要做出技术架构决策时
- 评估技术可行性时
- 遇到复杂技术问题时
- 需要代码审查时

**核心问题**：
- "这个技术选择有什么风险？"
- "如何确保系统可扩展？"
- "有哪些技术债务需要注意？"

---

### Producer（制作人）
**职责**：
- 项目协调和进度管理
- 资源分配和优先级排序
- 风险管理
- 团队协作流程

**使用场景**：
- 项目规划和冲刺计划时
- 需要协调多个系统开发时
- 评估项目进度时
- 风险管理时

**核心问题**：
- "这个功能的优先级是什么？"
- "我们应该在什么顺序开发这些系统？"
- "有什么风险可能影响项目进度？"

## 🎨 Tier 2 - 设计层（专业设计）

### Game Designer（游戏设计师）
**职责**：
- 核心循环设计
- 系统机制设计
- 玩家进度设计
- 游戏规则设计

**使用场景**：
- 设计新功能或系统时
- 定义游戏规则时
- 设计玩家进度系统时
- 平衡性调整时

**专业领域**：
- MDA Framework（机械-动态-美学框架）
- Flow State Design（心流状态设计）
- Self-Determination Theory（自我决定理论）
- Bartle Player Types（玩家类型）

---
