# 改写词库：原文 → 推文

## 核心原则

推文口播是「讲」出来的，不是「写」出来的。一切词汇选择以「听感」为准。

---

## 书面语 → 口语

| 原文（书面） | 推文（口语） |
|------------|------------|
| 不多时 | 没过一会儿 |
| 夕阳西下 | 太阳快落山了 |
| 开口道 | 说 / 开口说 |
| 恍然大悟 | 一拍脑门：我知道了！|
| 心中暗想 | 删掉，换动作 |
| 娓娓道来 | 一一道来 / 慢慢说 |
| 冷若冰霜 | 冷着一张脸 / 看都没看我一眼 |
| 万籁俱寂 | 安静得可怕 / 一根针掉地上都能听见 |

---

## 抽象描述 → 具体画面

| 原文（抽象） | 推文（具象） |
|------------|------------|
| 他心情复杂 | 他嘴角抽了一下 / 他看了我一眼又别过头 |
| 气氛很紧张 | 没人说话，连呼吸声都能听见 |
| 她很高傲 | 下巴抬得老高，正眼都不看我 |
| 他很生气 | 一巴掌拍在桌上，茶杯都跳起来了 |
| 她很难过 | 咬着嘴唇，眼泪打转 |
| 他很惊讶 | 眼珠子都快瞪出来了 |
| 她很害羞 | 头快低到胸口了 |
| 他很得意 | 嘴角翘上天了 |
| 场面很震撼 | 所有人跪了一地 / 没人敢动 |
| 他很疲惫 | 眼下一片青黑，走路打飘 |

---

## 第三人称 → 第一人称

| 原文（第三人称） | 推文（第一人称） |
|----------------|----------------|
| 赵风开口道 | 我开口 |
| 她看向赵风 | 她看向我 |
| 众人听见赵风的话，纷纷点头 | 我一说完，全场都在点头 |
| 程昱站在赵风身后 | 仲德站在我身后 |
| 张飞对赵风说 | 张飞对我说 / 翼德凑过来跟我说 |

---

## 称谓与语气词（增加代入感）

| 场景 | 可用表达 |
|------|---------|
| 震惊/愣住 | 这不是……/ 什么？/ 我傻了 |
| 确认/醒悟 | 还真是…… / 原来如此 / 我说呢 |
| 愤怒/激动 | 妈的！/ 好家伙！/ 靠！ |
| 无奈/苦笑 | 行吧 / 算了 / 我还能说什么 |
| 得意/自信 | 那可不 / 还用说？ / 我就知道 |
| 紧张/不安 | 完了 / 不好 / 不对劲 |

---

## 连接词与过渡

| 原文连接方式 | 推文连接方式 |
|------------|------------|
| 此后/紧接着 | 然后 / 接着 |
| 与此同时 | 就在这时候 / 另一边 |
| 忽然/突然 | 忽然 / 一下子 / 猛地 |
| 然而/但是 | 可 / 可是 / 但我…… |
| 最终/终于 | 最后 / 总算 |
| 片刻之后 | 过了一会儿 / 没多久 |

---

## 夸饰词库（增强画面冲击力）

| 普通表达 | 推文夸饰 |
|---------|---------|
| 很多 | 黑压压一片 / 密密麻麻 / 全是 |
| 很大 | 大得吓人 / 一眼望不到头 |
| 很痛 | 疼得要命 / 像被劈开一样 |
| 很快 | 一眨眼的工夫 / 嗖的一下 |
| 很安静 | 安静得像死了一样 / 连根针都听得见 |
| 气势很强 | 隔着三里地都能感觉到 / 腿都软了 |
