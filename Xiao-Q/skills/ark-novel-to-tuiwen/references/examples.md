# 完整范例库

## 范例1：三国穿越 — 知识碾压（高光密集型，约2200字）

### 原文概要
主角穿越成三国时期的县令，华佗来拜访，主角用现代医学知识震惊华佗。同时当地有瘟疫，主角组织防疫，被百姓称为神医。

---

```
━━━━━━━━━━━━━━━━━━━━━━
【标题】
我给华佗讲细菌，他当场傻眼了

━━━━━━━━━━━━━━━━━━━━━━
【人物卡】
① 我：穿越县令，懂现代医学
② 华佗：神医老者，背着药箱
③ 管家：县衙管事，一惊一乍

━━━━━━━━━━━━━━━━━━━━━━
【场景卡】
① 县衙正厅：白天，阳光照进厅堂，案桌上有炭笔和竹简
② 感染区：城外村落，村民发烧咳嗽，空气中弥漫草药味
③ 茅草屋：主角的临时医院，木板上躺着病人，酒精气味刺鼻

━━━━━━━━━━━━━━━━━━━━━━
【完整口播叙事】

管家冲进来喊：老爷，华佗求见！

我一口茶差点喷出来。华佗？那个华佗？我穿越到三国已经三天了，当了这破县的小县令，整天断些鸡毛蒜皮的案子。现在华佗来找我，我能不激动吗？

我三步并两步冲到门口，看见一个鹤发童颜的老者背着药箱，站在台阶下。我二话不说，鞠了个九十度的躬。华佗吓得连连后退，赶紧扶我：侯爷，小老儿受不起！

我把他请进正厅。他打开药箱，掏出一摞竹简，说这是他毕生所学，想请我指教。

我翻开一看——全是放血疗法、阴阳五行、五禽戏。我不是说这些东西不行，但在这个时代，这些已经是天花板了。

我说：华佗先生，你知道细菌吗？

他愣住。我拿起炭笔，在桌上画了一个圈。

我说：就是这种东西，小到肉眼看不见，但它钻进你的伤口，伤口就会化脓。钻进你的肺，你就会发烧咳嗽。

华佗盯着那个圈，眼睛越来越大。他的手在发抖。

过了老半天，他声音都变了：侯爷……你是如何得知的？

我说：因为我来自两千年后。

华佗沉默了很长时间。然后他慢慢站起来，对着我跪了下去。

他说：请侯爷教老夫，什么是细菌。

就在这时候，外面突然跑来一个衙役，连滚带爬地滚进厅里。

他喊：老爷！不好了！城外三个村子，一夜之间全病了！发烧、咳嗽、上吐下泻！

我心里咯噔一下。瘟疫。

我说：华佗先生，你的第一课到了。跟我来。

我们带着十几个衙役赶到城外。那个村子惨不忍睹。路边躺着人，脸色蜡黄，嘴角吐着白沫。有人抱着孩子哭，有人跪在路边磕头求老天爷。

华佗第一反应是号脉。我拦住他。

我说：没用。这是传染病，号脉号不出来。

我让衙役把所有病人集中到村口的空地上。然后我从药箱里翻出几坛烈酒——这玩意在三国叫"清酒"，度数不高，聊胜于无。

我说：从现在开始，所有人碰病人之前，先用酒洗手。病人用过的东西全烧掉。病人的排泄物挖深坑埋了。谁不照做，下一个倒下的就是他。

衙役们面面相觑，华佗也愣在原地。

我管不了那么多，一头扎进病人堆里。高烧的喂退烧草药，脱水的灌盐糖水。没有消炎药，我只能靠酒消毒，靠病人的命硬撑着。

三天三夜，我没合眼。

第四天早上，第一个退烧的病人睁开眼睛，看见我坐在旁边，眼泪就下来了。

他叫了我一声：神……神医。

然后一个接一个的病人开始退烧。华佗站在旁边，看着我的手。

他说：侯爷，你手上起了泡，沾了酒也不喊疼吗？

我低头一看，十个指头全起了泡。消毒的时候不觉得，现在才火辣辣地疼。

华佗的眼眶红了。

他说：老夫行医四十年，从未见过这样的治法。侯爷，你不是大夫，但你是救人的菩萨。

我笑了笑，说：菩萨不敢当，但有些东西，比阴阳五行好用。

就在我以为疫情要结束的时候，一个衙役跑过来，手里抓着一封信。

他说：老爷，京里来的急报。

我打开一看，手就抖了。

曹操亲笔。

（约2200字）

━━━━━━━━━━━━━━━━━━━━━━
【结尾话术】
曹操在信里写了四个字……你猜是什么？下集马上揭晓！
```

## 范例2：都市逆袭 — 隐藏首富（高光密集型，约2000字）

### 原文概要
主角是隐藏身份的千亿首富，去女朋友家提亲被岳母嫌穷，当场亮出一份80亿的土地合同打脸岳母和全家人。

---

```
━━━━━━━━━━━━━━━━━━━━━━
【标题】
丈母娘催我打彩礼，我转头拿了份合同

━━━━━━━━━━━━━━━━━━━━━━
【人物卡】
① 我：千亿首富，伪装成普通打工族
② 丈母娘：势利眼，催着要50万彩礼
③ 小桃：我女朋友，不知道我有钱
④ 陈秘书：我的私人秘书

━━━━━━━━━━━━━━━━━━━━━━
【场景卡】
① 老丈人家客厅：老式居民楼，家具旧，茶几上摆着水果和茶水
② 我家书房：整面墙是城市地图，桌上摆着一摞合同
③ 老丈人家门口：楼道灯坏了，我拎着两袋水果站在门口

━━━━━━━━━━━━━━━━━━━━━━
【完整口播叙事】

我第一次去她家。

拎着两袋水果，站在她家门口，门上的漆掉了一半。小桃拉着我的手说：别紧张，我爸妈很好说话的。

门开了。她妈上下打量了我一眼，目光停在水果袋上，嘴角动了一下。

我也没多想，进去了。

饭桌上她妈一直问：小李，你在哪上班啊？一个月多少钱啊？买房了没有？

我说在创业公司做运营，月薪八千，还没买房。

她妈筷子一放。

八、千？

那顿饭吃得很安静。小桃在桌子底下捏我的手，我笑了笑表示没事。

吃完饭她妈把我拉到一边。

她说：小李啊，我跟你说实话。我们家小桃条件不错，追她的人多了去了。你要娶她也可以——彩礼五十万，婚房一套，代步车一台。三个月之内凑齐。

我说：阿姨，彩礼五十万是不是有点……

她说：五十万多吗？城里的李家少爷，一开口就是两百万。我要你五十万，已经看小桃面子了。

我点点头，说行，我回去准备一下。

出了门，小桃追出来。她眼眶红了，说对不起，她妈就是这样的人。

我说没事，五十万嘛，我能凑。

她哭得更厉害了。

我回到家，推开书房。陈秘书站在门口，手里拿着平板。

我说：城西那块地的合同呢？

陈秘书愣了一下：李总，那块地……八十个亿。

我说：我知道。拿来。

我把合同翻到最后一页，签了字。然后拍照发给了丈母娘。

不到五分钟，电话响了。

她妈的声音在发抖：小……小李，这合同是你的？

我说：是啊阿姨，我刚刚签的。彩礼钱没问题，五十万对吧，我让财务明天打给你。

电话那头沉默了。

我听见她妈在喘气，喘了好几声。

她说：不不不……小李你不用打，彩礼我们再商量……

我说：不用商量了。合同您看到了就行。对了阿姨，楼下那辆黑色轿车是我的，之前怕太高调没开过去。明天我开过来接小桃。

挂完电话，我给小桃发了条消息。

我说：宝贝，你妈同意我们的事了。

小桃回了我三个问号。

我说：没什么，就给她看了看我的合同。

小桃沉默了好一会儿。

然后她发来一行字。

她说：所以你骗了我三年？

我看着屏幕，手停在半空中。

（约2000字）

━━━━━━━━━━━━━━━━━━━━━━
【结尾话术】
小桃说她要来我家找我算账……我把保险柜打开了，里面还有九份合同。下集看小桃看到保险柜的反应！
```

## 范例3：穿越女频 — 庶女逆袭（中等高光密度，约1800字）

### 原文概要
女主穿越成古代不受宠的庶女，被嫡母安排嫁给瘸腿将军。女主不哭不闹，反而利用现代商业思维盘活将军府的产业，让所有人刮目相看。

---

```
━━━━━━━━━━━━━━━━━━━━━━
【标题】
嫁过去第三天，瘸腿将军要休了我

━━━━━━━━━━━━━━━━━━━━━━
【人物卡】
① 我：穿越庶女，懂商业和管理
② 将军夫君：瘸腿大将军，冷漠，觉得娶了废物庶女
③ 嫡母：原主嫡母，巴不得把我嫁出去

━━━━━━━━━━━━━━━━━━━━━━
【场景卡】
① 将军府书房：深夜，满桌子账本，烛火跳动，我趴在桌上睡着了
② 将军府库房：白天，灰尘弥漫，我扯开油布露出十几个大箱子
③ 皇宫宴席：灯火辉煌，文武百官，将军坐在轮椅上，我站在他身后

━━━━━━━━━━━━━━━━━━━━━━
【完整口播叙事】

花轿抬进将军府那天，我是一路哭着进去的。

不是吓哭的，是兴奋哭的。我终于离开那个嫡母了。

将军夫君坐在轮椅上，黑着一张脸。他连我的红盖头都没掀，就让嬷嬷把我领到偏院去了。

嬷嬷小声跟我说：将军说了，让夫人安生住着，没事别去前院。

翻译一下：他不想要我。

行吧。

第二天我去请安，他正在书房看书。我站在门口，他头都没抬。

他说：你回去吧，我这里不需要你。

我说：行，那你能不能把库房钥匙给我？

他终于抬头了。

他说：你要钥匙做什么？

我说：你府里入不敷出三年了，库房里堆着一堆废物，账上的现银不够发这个月的俸禄。你要是不让我管，下个月你那些兵就得饿肚子。

他盯着我看了十秒钟。

然后把钥匙扔了过来。

我接了。

库房门一开，我差点被灰呛死。里面堆着大大小小十几个箱子，全蒙着油布。我一把扯开。

最上面那个箱子里，铺了满满一层——人参。

不是普通人参，是长白山的野山参。一根一根，根须完整，品相绝佳。

我手在抖。

这些在将军府堆了三年的"废品"，放到京城药铺，一根能卖二两银子。这一整箱，至少值一千两。

我连夜重新理了所有账本。欠款、库存、月俸、人情往来。在Excel里我是财务总监，在这里我用算盘三天算完了三年的烂账。

第四天早上，我把一本新账放在他面前。

我说：下个月你能多出一千二百两。

他翻了翻账本，问我：怎么做到的？

我说：你的人参放在库房喂老鼠，我拿去卖给药铺了。还有你名下三处铺子，地段都不错但全亏钱，我重新定了价，换了掌柜。最后一件事——你还欠兵部一笔军饷没结，我帮你要回来了。

他合上账本。

然后他笑了。

我嫁进来六天，第一次见他笑。

他说：你比我军师还能算。

就在我以为一切顺利的时候，宫里来了一道旨——皇上要办家宴，点名要将军携夫人出席。

瘸子夫君看看我，表情变了。

他说：那群人，恨不得我死在轮椅上。

我说：那正好，让他们看看，你想死都没那么容易。

（约1800字）

━━━━━━━━━━━━━━━━━━━━━━
【结尾话术】
皇宫家宴上，太子妃当着全场面羞辱我……你猜我怎么回她的？下集告诉你！
```

## 范例4：玄幻修真 — 废材觉醒（单一爆点型，约1500字）

```
━━━━━━━━━━━━━━━━━━━━━━
【标题】
退婚那天，九道天雷劈开了我的灵脉

━━━━━━━━━━━━━━━━━━━━━━
【人物卡】
① 我：修真废材，实为万古第一血脉
② 苏婉清：前未婚妻，青鸾宗圣女
③ 苏家老祖：金丹修士，高高在上

━━━━━━━━━━━━━━━━━━━━━━
【场景卡】
① 苏家大殿：白玉地板，高阶修士云集，我站在大殿中央
② 后山禁地：荒山野岭，乌云翻滚，闪电劈开天幕
③ 苏家山门：宗门牌坊下，所有人跪了一地

━━━━━━━━━━━━━━━━━━━━━━
【完整口播叙事】

婚书被扔到我脸上。

苏婉清站在大殿之上，当着所有人的面说：你一个废灵脉，也配娶我？

满堂修士没一个说话。苏家老祖坐在主位，闭着眼睛喝茶。

我弯下腰，把婚书捡起来。折好，揣进怀里。

然后我转身走出了苏家大门。

没人拦我。

我走了十里山路，走到后山那片荒坡。从小别人练功我都躲在这，因为在这没人看得见我。

天上开始打雷。

第一道雷劈下来，我抬头看，金光灌顶。

像被雷劈中了头顶。疼得我跪在地上喘不过气。但很奇怪，骨头里钻出一种痒，像有什么东西在往外长。

第二道雷。第三道。第四道。

耳朵里全是轰鸣声，但是我能听见自己的骨头在响。不是断裂的声音，是生长的声音。

第九道雷劈完，世界安静了。

我从土坑里爬起来，低头看自己的手——金色的纹路从指尖往上爬，一直爬到肩膀。不是画的，是从皮肤下面透出来的光。

脑子里"叮"的一声。

一个声音响起来：万古第一血脉，觉醒完成。修为：炼气→筑基→金丹→元婴。

四个大境界，九道雷，三秒钟。

元婴。苏家老祖修炼了两百年，还在金丹期卡着。

我站起身，脚下三尺的泥土全部裂开。树上掉下来一片叶子，还没落地就被我的灵力烧成了灰。

我走回苏家。

苏婉清还在大殿里。看见我进来的瞬间，她的脸刷地白了。

她退了一步，撞翻了茶案。

我走到大殿中央，把婚书掏出来，撕成两半，扔到地上。

我说：现在是谁不配？

苏家老祖猛地睁开眼。他的茶杯掉在地上，摔成了碎片。

他看了我三秒钟。

然后他站了起来。

然后他跪了下去。

（约1500字）

━━━━━━━━━━━━━━━━━━━━━━
【结尾话术】
苏家老祖跪下去的那一刻，天上又响了一声雷。这次不是劈我——是劈向了青鸾宗的宗门大殿。下集更劲爆！
```

## 范例速查

| 高光密度 | 适用场景 | 情绪曲线 | 范例 |
|--------|---------|---------|-----|
| 稀疏 | 1个核心高光+大量过渡，短平快 | 钩子→背景→一波高潮→悬念 | 范例5 |
| 中等密度 | 2-3个连续高光，多角色登场 | 钩子→背景→逐层推进→高潮→悬念 | 范例3 |
| 高光密集 | 连续爆点+多条线索+大型冲突 | 完整5段式充分展开 | 范例1、2、4 |

> 输出长度由高光决定，不是由章节数决定。范例标注的字数仅供参考，不作为输出标准。

## 范例5：权谋宫斗 — 高光稀疏型（约600字）

> 这一章原文有大量朝会仪式描写和人物内心戏，真正值得推的只有一个"圣旨反转"爆点。

```
━━━━━━━━━━━━━━━━━━━━━━
【标题】
圣旨念到一半，大臣全跪了

━━━━━━━━━━━━━━━━━━━━━━
【人物卡】
① 我：不受宠的皇七子，装怂保命
② 大太监：宣旨太监，趾高气扬

① 王府正厅：午后，阳光斜照进厅堂，满地跪着的人

━━━━━━━━━━━━━━━━━━━━━━
【完整口播叙事】

那天下午，宫里来了个太监。

他站在我家正厅，手里举着圣旨，下巴抬得老高。

我跪在地上听。

前面一大段都是套话，我懒得听。直到念到中间——

圣旨说，皇七子品行端正，赐婚太傅之女。

我傻了。

太傅之女？那不是太子妃的内定人选吗？父皇把她赐给我——这不是赐婚，这是往我身上扣了一顶帽子。

那太监念完，把圣旨递过来，笑眯眯地说：恭喜王爷。

我看着那卷黄绸，手没伸出去。

满厅的人还跪着，没人敢抬头。

太监的笑容僵住了：王爷？

我还是没接。

他急了：王爷，这可是圣旨！

我抬头看他，笑了笑。

我说：你回去告诉父皇，这旨——我不接。

我不接圣旨的消息传到宫里……太子连夜进了宫。下集看他怎么对付我！
```
