# 情绪曲线模型库

## 使用方法

判读原文体裁 → 匹配对应模型 → 按模型节奏分配叙事篇幅。

> 占比是**节奏感参考**，不是字数预算。比如"背景占20%"意思是别铺垫过头，不是说必须写200字。

---

## 模型1：穿越开场型

**适用体裁**：穿越/穿书/重生

**核心情绪线**：
信息炸弹（钩子）→ 代入感建立（背景）→ 碾压快感（冲突）→ 巅峰高潮 → 更大悬念

**节奏要点**：
| 阶段 | 占比 | 核心任务 |
|------|------|---------|
| 钩子 | 10% | 现代知识vs古代认知的碰撞画面，一句话炸 |
| 背景 | 20% | 我是谁、穿成了谁、有什么金手指、什么处境 |
| 冲突 | 45% | 用金手指碾压古代规则，至少2-3个碾压场景逐层升级 |
| 高潮 | 15% | 碾压的高光画面，让观众爽到起鸡皮疙瘩 |
| 悬念 | 10% | 忽然出现的更大挑战，等级和之前完全不在一个量级 |

**关键技巧**：
- 钩子打「信息差」：现代知道但古代不知道的东西
- 爽感来自于「所有人觉得不可能→但主角做到了」
- 每层碾压结束后，观众的爽感不要马上凉，立即接下一层

## 模型2：打脸爽文型

**适用体裁**：装逼打脸、逆袭、隐藏大佬

**核心情绪线**：
期待（钩子）→ 被轻视感（背景）→ 冲突累积 → 爆发打脸 → 碾压余韵

**节奏要点**：
| 阶段 | 占比 | 核心任务 |
|------|------|---------|
| 钩子 | 8% | 身份反差：看似普通人，其实是大佬 |
| 背景 | 15% | 渲染被看不起的氛围，让观众共情"被轻视感" |
| 冲突 | 50% | 先抑后扬：被踩→再被踩→忽然亮牌→逐层碾压 |
| 高潮 | 17% | 身份揭晓的高光画面，全场跪 |
| 悬念 | 10% | 更强的人出现了，或者身份暴露带来的连锁反应 |

**关键技巧**：
- 前半段"抑"要真实：观众代入被看不起的感觉
- 后半段"扬"要逐层升级：小打脸→中打脸→大打脸→核爆
- 打脸方式多样化：身份、财富、能力、人脉

## 模型3：悬念压抑型

**适用体裁**：悬疑、揭秘、虐文、情感冲突

**核心情绪线**：
压抑钩子 → 层层逼近真相 → 压抑到最重 → 爆发/反转 → 余韵不散

**节奏要点**：
| 阶段 | 占比 | 核心任务 |
|------|------|---------|
| 钩子 | 10% | 一个反常/压抑的画面或对话，让人不安 |
| 背景 | 25% | 铺垫压抑氛围，埋线索，观众和主角一起困惑 |
| 冲突 | 40% | 逐层接近真相，每层更压抑，每层更多线索 |
| 高潮 | 15% | 真相揭露，反转炸裂，情绪在这一刻完全释放 |
| 悬念 | 10% | 反转后的连锁反应，下一个更大的谜团 |

**关键技巧**：
- 节奏可以比爽文慢，但信息密度不能低
- 用环境和动作传递压抑感（天色、温度、身体反应）
- 反转让观众觉得「之前所有细节都串联起来了」

## 模型4：收获满足型

**适用体裁**：系统文、金手指文、种田经营、收集养成

**核心情绪线**：
期待（钩子）→ 第一个奖励 → 更大的奖励 → 巅峰满足 → 新目标出现

**节奏要点**：
| 阶段 | 占比 | 核心任务 |
|------|------|---------|
| 钩子 | 8% | 系统激活/金手指到手的瞬间 |
| 背景 | 15% | 为什么需要这个系统、之前多惨 |
| 冲突 | 50% | 奖励逐层递进：小→中→大→逆天，每层验证奖励效果 |
| 高潮 | 17% | 最强的那个奖励揭晓，画面要炸、要炫 |
| 悬念 | 10% | 下一条任务/下一个奖励是什么 |

**关键技巧**：
- 「叮！」要保留仪式感，但不要每句都叮
- 奖励揭晓→能力验证→众人反应，三步一个循环
- 最炸的奖励放在高潮段，之前的都是铺垫

## 模型5：多爽点串联型

**适用体裁**：快节奏文、战神归来、无敌流

**核心情绪线**：
钩子 → 爽点A → 爽点B → 爽点C → 爽点D → 悬念

与模型2（打脸）的区别：不需要被轻视→反击的"抑扬"结构，全程昂扬，爽点直接碾过去。

**节奏要点**：
| 阶段 | 占比 | 核心任务 |
|------|------|---------|
| 钩子 | 10% | 第一波碾压画面，直接炸开 |
| 爽点A | 20% | 第二波碾压，比第一波更大 |
| 爽点B | 20% | 第三波碾压，还在升级 |
| 爽点C | 20% | 第四波碾压，观众已经爽麻了 |
| 高潮 | 20% | 最炸的一波，全篇顶点 |
| 悬念 | 10% | 更大的对手/挑战浮现 |

- 每个爽点干净利落，不留喘息
- 爽点之间用1-2句过渡，不能断节奏
- 全文节奏像过山车，不到结尾不下来

## 快速匹配表

| 原文特征 | 推荐模型 | 示例钩子 |
|---------|---------|---------|
| 主角带现代知识/记忆穿越 | 模型1 | 「我给华佗讲细菌，他当场傻眼了」 |
| 主角隐藏身份被看不起 | 模型2 | 「丈母娘催我打彩礼，我转头拿了份合同」 |
| 真相层层揭露，情绪压抑 | 模型3 | 「她问我愿不愿意带她走，我沉默了」 |
| 系统/金手指/奖励型 | 模型4 | 「我送了三年外卖，系统终于激活了」 |
| 主角战力/能力碾压一切 | 模型5 | 「十万大军围城，我一人走了出去」 |
