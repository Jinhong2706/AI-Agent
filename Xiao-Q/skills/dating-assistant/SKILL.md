# 相亲聊天助手 (Dating Chat Assistant)

## 技能目的

帮助用户在相亲场景的在线聊天中，给出恰当、自然、能让氛围愉快顺畅的回复建议。核心是**情商**——让对方感觉被理解和欣赏，同时展现自己的魅力和诚意。

## 触发条件

当用户表达以下意图时触发本技能：

1. **直接请求帮助回复**
   - "帮我回复相亲对象"
   - "帮我看看怎么回"
   - "这样回行吗"
   - "怎么回比较好"

2. **描述相亲对象的消息**
   - "相亲对象说：XXX"
   - "他/她刚才说..."
   - "她问了我一个问题..."

3. **描述聊天困境**
   - "不知道怎么回"
   - "聊不下去了"
   - "感觉有点尴尬"
   - "怎么接这个话"

4. **要求生成回复选项**
   - "给我几个回复选项"
   - "有没有更好的说法"
   - "怎么夸她/他"

**注意**：如果用户只是在讨论相亲话题但没有请求帮助回复，不要触发本技能。

## 执行逻辑

### 步骤 1：理解场景

分析用户提供的信息，识别：

- **聊天阶段**：
  - 破冰期：刚加上微信，前几句话
  - 了解期：互相介绍基本信息（工作、兴趣、生活）
  - 深入期：开始有更私人的话题交流
  - 暧昧期：语气更亲昵，可能涉及约会暗示
  - 邀约期：讨论见面安排

- **对方语气**：
  - 热情友好：感叹号多、主动提问、分享欲强
  - 礼貌平淡：中规中矩，回复较简短
  - 试探谨慎：问话多，主动分享少
  - 调侃活泼：表情包多、语气轻松
  - 高冷疏远：回复极短或不主动开启话题

- **对方消息的核心信息**：她在问什么？分享什么？暗示什么？

### 步骤 2：回复原则

遵循以下原则生成回复：

1. **呼应对方内容**
   - 接住对方的话题，不要突然换话题
   - 对她分享的内容给予回应（共鸣或好奇追问）

2. **展现兴趣但不跪舔**
   - 适度表达欣赏和好奇
   - 避免过度热情或过于冷淡
   - 保持平等的交流姿态

3. **轻松自然，避免尴尬**
   - 用日常口语，千万别背台词感
   - 适当幽默调侃，但别过度
   - 可以用表情包辅助

4. **制造下一次聊天机会**
   - 留下话题延续的空间
   - 或者自然引向线下见面

### 步骤 3：生成回复

提供 **3条回复选项**，覆盖不同风格：

| 风格 | 特点 | 适用场景 |
|------|------|----------|
| **稳重温柔型** | 真诚、温和、有温度 | 对方偏保守/认真 |
| **活泼俏皮型** | 轻松、有趣、带点小调侃 | 对方偏活泼/已破冰 |
| **简洁干练型** | 简短有力，不啰嗦 | 对方回复简短/节奏快 |

### 步骤 4：输出格式

```
📍 场景分析
- 聊天阶段：了解期
- 对方语气：热情友好
- 她在问/说：询问你的工作

💬 回复建议

【稳重温柔型】
"我在互联网公司做产品，其实挺有意思的，每天都在想怎么让用户用得更顺手。你呢，平时工作忙不忙？"

【活泼俏皮型】
"产品经理！就是那种天天开会、画图、被程序员吐槽的角色[捂脸] 你呢，是做什么的呀？"

【简洁干练型】
"做互联网产品的，你呢？"

---
✨ 小贴士：[补充一条情商建议，如"如果她分享了什么，可以顺势夸她"等]
```

## 回复参考库

### 常见场景的经典回复模板

**对方问工作**
```
"做[行业]的，具体是[岗位]，平时主要[工作内容简述]。你呢？"
```
如果对方先说工作，可以：
- 找到共鸣点："我也是经常加班的类型..."
- 好奇追问："听起来挺有意思的，平时主要忙什么？"

**对方分享爱好/兴趣**
- 表达欣赏："听起来很有生活品味"
- 好奇追问细节："平时在哪里玩这个？"
- 关联自己："我之前也尝试过..."

**对方问"你在干嘛"**
- 不要只说"没干嘛"
- 描述+反问："刚下班在回家路上，你呢？"

**对方发了表情包/调侃**
- 接住调侃，适当回击
- 配合表情包回复更自然

**对方问"你为什么相亲"**
- 真诚但轻松："家里人催得紧[捂脸]，自己也觉得遇到合适的人挺重要的"
- 或者："平时工作忙，社交圈小，就想着试试这个方式"

**对方问"你觉得我怎么样"**
- 真诚+具体细节："印象很好，感觉你是个很有想法的人，而且聊天很舒服"
- 不要过度夸外表，显得轻浮

**对方很久没回复后的开场**
- 轻松自然，不提等多久
- 找个新话题开启："刚看到一个[相关话题]，想起你之前说..."

**邀约被婉拒**
- 表示理解，不追问原因
- 保持友好："没关系，最近确实都挺忙的，下次再约~"

## 禁忌提醒

⚠️ **绝对避免**：
- 不要只回"嗯"、"哦"、"好的"（除非对方也这样）
- 不要连续提问像查户口（连续问3个问题）
- 不要过早涉及敏感话题（收入、婚育计划等）
- 不要过度吹嘘自己
- 不要评价对方外貌（尤其第一阶段）
- 不要在对方没回应时连续发消息

⚠️ **谨慎使用**：
- 表情包：确认对方也在用再大量使用
- 调侃/自嘲：先看对方接受度
- 直接夸外表：适度即可，不要油腻

## 特殊情况处理

**对方回复很简短（"嗯"、"哦"）**
→ 可能累了/在忙，也可能是话题不感兴趣
→ 回复要简短，留空间
→ 尝试换个话题或主动结束这轮聊天

**对方明显敷衍/冷淡**
→ 不要过度热情追问
→ 保持友好但降低回复频率
→ 换个更有趣的话题重新吸引

**不知道聊什么冷场了**
→ 问一个开放式问题
→ 分享一个自己的生活/见闻
→ 用表情包/有趣的话题破冰

**想结束聊天但不想显得冷淡**
→ "今天聊得很开心，我先忙了，回聊~"
→ "有点困了，明天再聊？晚安[月亮]"
