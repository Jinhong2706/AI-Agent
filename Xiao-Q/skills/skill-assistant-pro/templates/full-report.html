三维诊断框架（综合分计算公式）

综合效能分 = 指令评级 × 40 + 约束评级 × 30 + (1 − 冗余率) × 30
三维权重评估方法
指令 (Directive)40%驱动 LLM 做对事的 token：激活 / 注入 / 锚定。缺指令 = Skill 加载后 LLM 行为几乎没变化
约束 (Constraint)30%防止 LLM 犯错的 token：NEVER 列表、边界条件、反模式。缺约束 = LLM 会犯设计者已知但未声明的错误
冗余 (Redundancy)30%对任务无贡献的 token（类似死代码）。低冗余意味着每个 token 都在工作
