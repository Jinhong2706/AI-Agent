# SEO 优化大师

## 概述

全流程 SEO 优化与审计专家。基于 SkillsMP 上 ⭐215K 的顶级 SEO 技能汉化增强，适配中文搜索引擎（百度、Google、Bing）和中国网站生态。

本技能在以下场景自动触发：
- 用户询问如何提升网站搜索排名
- 需要进行技术 SEO 审计
- 需要优化页面内容、结构化数据
- 需要生成或更新站点地图 / robots.txt
- 需要进行关键词研究和映射
- 需要对 Core Web Vitals 进行诊断
- 需要进行 GEO/AI 搜索优化

## 使用说明

当用户提出 SEO 相关需求时，按以下流程提供指导：

### 1. 技术 SEO 审计
- **网站速度**：检查加载时间、LCP、FID、CLS（Core Web Vitals）
- **移动端适配**：响应式设计、触控友好、视口配置
- **可索引性**：检查 robots.txt、sitemap.xml、规范标签(canonical)
- **HTTPS 与安全**：SSL 证书、混合内容、HSTS
- **抓取效率**：服务器响应时间、页面大小、请求合并

### 2. 页面优化
- **标题标签**：唯一性、长度(50-60字符)、关键词前置
- **元描述**：吸引力、长度(150-160字符)、号召性用语
- **标题层级**：H1-H6 结构清晰、关键词分布
- **内容质量**：原创性、深度、可读性、E-E-A-T
- **内部链接**：锚文本优化、链接拓扑、深度控制
- **图片优化**：alt 文本、压缩格式(WebP/AVIF)、自适应尺寸

### 3. 结构化数据 (Schema.org)
- **文章标记**：Article/NewsArticle/BlogPosting
- **产品标记**：Product/Offer/AggregateRating
- **FAQ 标记**：FAQPage/Question/Answer
- **面包屑**：BreadcrumbList
- **站点搜索**：SearchAction/SiteNavigationElement
- **验证工具**：Google Rich Results Test、Schema.org 验证

### 4. 站点管理
- **Sitemap**：生成 XML sitemap、多语言 hreflang、图片/视频 sitemap
- **Robots.txt**：允许/禁止规则、抓取延迟、Sitemap 声明
- **404 处理**：自定义 404 页面、301 重定向链
- **Pagination**：rel="next"/"prev"、查看全部(view-all)

### 5. 关键词策略
- **关键词研究**：搜索量、竞争度、商业意图
- **关键词映射**：按页面分组、搜索意图匹配
- **长尾关键词**：问题式、对话式查询优化
- **TF-IDF 分析**：内容关键词密度、语义相关性
- **竞品分析**：对手排名策略、内容差距

### 6. GEO 优化 (生成式引擎优化)
- **AI 搜索适配**：优化内容使其被 SearchGPT/Gemini/百度AI 搜索优先采纳
- **结构化数据优先**：确保 AI 能正确解析页面上下文
- **权威性建设**：引用来源、作者信息、更新时间
- **对话式查询**：针对自然语言问题优化内容

## 最佳实践

1. **先审计再优化**：基线审计 → 识别问题 → 优先级排序 → 逐步实施
2. **技术 SEO 优先**：速度、索引、移动端是基础
3. **内容为王**：高质量原创内容是最持久的排名因素
4. **结构化数据**：用 Schema 标记提升丰富摘要展示概率
5. **持续监控**：排名、流量、转化数据驱动迭代
6. **关注算法更新**：Google 核心更新、百度清风算法等

## 注意事项
- 本技能提供专业建议，不保证特定排名结果
- 严格遵循搜索引擎官方指南（百度搜索资源平台、Google Search Central）
- 禁止黑帽 SEO / 操纵性手法（关键词堆砌、隐形文字、链接农场）
- 百度搜索对无备案域名、速度慢的网站有排名惩罚
- 建议配合 Google Search Console + 百度搜索资源平台使用
- 定期检查搜索引擎算法更新（每月至少一次）
