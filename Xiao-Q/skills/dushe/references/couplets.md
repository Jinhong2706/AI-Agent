# 古寺楹联素材库

当心境与以下楹联契合时，可在毒舌安慰中引用。**宁缺毋滥**——不是每次都要引，只有真正契合时才用。

## 楹联与心境映射

| 楹联 | 适用心境 | 翻转方向 |
|------|---------|---------|
| 半生风雨半生悟，一世清欢一世安 | 苦尽甘来、转折时刻 | 风雨本身就是悟的过程 |
| 看淡世间繁华起落，守住本心一世安然 | 焦虑攀比、浮躁 | 不比繁华比安然 |
| 凡尘俗事不必心生焦躁，内心清净自然岁月安然 | 事情多尾巴、烦乱 | 焦躁不能加速解决，清净反而更快 |
| 人生难得事事皆如意，世事只求半分自称心 | 追求完美、不甘心 | 半分称心已是上签 |
| 得失随缘莫强求，聚散随心少烦忧 | 失去、离别 | 随缘不是放弃，是不和不可控较劲 |
| 烦恼皆由心上起，平和只在自心中 | 内耗、纠结 | 心是自己能管的为数不多的东西 |
| 心中放下千般事，眉间再无半点愁 | 放不下、扛太多 | 放下不是不管，是分清哪些真该管 |
| 世事浮沉皆常态，放宽胸怀少忧愁 | 无常、变故 | 浮沉是常态，不是你的错 |

## 引用规则

1. **不硬凑**：心境不匹配时坚决不引，生硬引用比不引更尴尬
2. **引后翻转**：引用楹联不是结束，要接着翻转——用楹联的点拨方向，落到具体行动
3. **语气自然**：融入行文中，不要"引用古联："这种格式，像随口说出来的
4. **最多一条**：每次毒舌安慰最多引用一条，多了变国学课
