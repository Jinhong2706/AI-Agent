# OpenClaw系统优化 技能
## 功能说明
每日定时自动执行两项核心运维操作，完成后自动发送结构化排版的飞书通知，无需人工干预，支持直接在对话框修改执行时间，无需后台操作：
1. **文件健康检查**：自动检查memory目录、skills目录、核心配置文件完整性，发现缺失自动修复
2. **内存优化清理**：自动清理Python缓存、大于10M冗余日志、临时文件、无用安装包、用户缓存，释放系统内存
3. **交互修改时间**：直接在对话框发送指令即可修改执行时间，无需登录后台操作
## 安装使用
1. 下载本技能zip包，上传到OpenClaw后台安装
2. 安装完成后添加定时任务，cron表达式默认推荐：`0 0 * * *`（每日0点执行，北京时间）
3. 任务触发后会自动给当前飞书对话用户发送通知
## 使用说明
### 🔄 修改执行时间
直接在飞书对话框发送指令：
```
修改系统优化执行时间 <cron表达式>
```
例如：
```
修改系统优化执行时间 0 2 * * *
```
表示将执行时间修改为每天凌晨2点（北京时间），系统会自动校验表达式合法性，修改成功后发送确认通知。
### 📅 cron表达式格式说明
```
分 时 日 月 周
```
常用示例：
- 每天0点：`0 0 * * *`
- 每天凌晨2点：`0 2 * * *`
- 每周一早上8点：`0 8 * * 1`
- 每月1号凌晨1点：`0 1 1 * *`
