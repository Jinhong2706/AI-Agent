# OpenClaw系统优化
## 基本信息
- 名称：OpenClaw系统优化
- 版本：1.1.0
- 作者：曾小闲
- 分类：系统工具
- 协议：MIT
## 功能概述
每日定时自动执行系统健康检查+内存优化清理，完成后自动发送结构化飞书通知，支持对话框直接修改执行时间，无需后台操作：
1. **文件健康检查**：自动检查memory、skills目录和核心配置文件完整性，发现缺失自动修复
2. **内存优化清理**：自动清理Python缓存、大于10M冗余日志、临时文件、无用安装包，释放系统内存
3. **交互配置**：直接在飞书对话框发送指令即可修改执行时间，零门槛操作
## 安装要求
- OpenClaw运行环境
- 飞书插件已完成授权
## 使用方法
1. 安装技能后添加定时任务，默认推荐cron表达式：`0 0 * * *`（每日0点执行，北京时间）
2. 修改执行时间：直接在对话中发送指令 `修改系统优化执行时间 <cron表达式>`，例如：
```
修改系统优化执行时间 0 2 * * *
```
表示将执行时间修改为每天凌晨2点（北京时间），系统会自动校验表达式合法性并返回确认通知。
