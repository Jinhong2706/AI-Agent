# 语言识人术

> 通过语言判断一个人性格的识人工具 🎯

## 核心定位

从语速语气、用词习惯、说话逻辑、矛盾反应、表达边界5个维度分析性格特征。

## 5大分析维度

| 维度 | 核心 | 分析内容 |
|------|------|---------|
| **语速语气** 🗣️ | 性子急慢 | 语速快慢、音量大小、语气强弱 |
| **用词习惯** 📝 | 自我/合群 | "我"多还是"我们"多、正面词还是负面词 |
| **说话逻辑** 🧠 | 理性/感性 | 先结论还是先铺垫、讲事实还是聊感受 |
| **矛盾反应** ⚡ | 冲突处理 | 直接吵、冷处理、习惯性道歉、推卸责任 |
| **表达边界** 🛡️ | 独立/讨好 | 敢拒绝还是不好意思拒绝 |

## 经典性格分类

### 内向型 vs 外向型
- **内向**：话少、精简、慢热、深思熟虑
- **外向**：话多、爱分享、自来熟、表达欲强

### 四维性格
- **理性果断型**：说话直接、讲逻辑、重结果
- **感性温柔型**：说话柔软、顾及感受、共情强
- **乐观活泼型**：说话轻松爱开玩笑、极少抱怨
- **谨慎多虑型**：说话反复斟酌、顾虑多

### 4种基础人格
- **胆汁质**：语速快、脾气急、说话直接
- **多血质**：能说会道、幽默健谈、擅长交际
- **粘液质**：说话平缓、情绪稳定、不争不抢
- **抑郁质**：话少内敛、心思细腻、共情极强

### MBTI 8大倾向
- E外向 / I内向
- S务实 / N理想
- T理性 / F感性
- J计划 / P随性

## 快速识人公式

```
看语速 = 性子急慢
看用词 = 乐观/悲观、自我/合群
看逻辑 = 理性/感性
看冲突反应 = 包容/尖锐、隐忍/直白
看边界 = 独立/讨好
```

## 示例

**新同事，说话很快，常用"我觉得"，爱打断**：
- 性格：胆汁质（火爆直爽型）+ 理性果断型
- 沟通建议：直接了当，不要绕弯子，给TA决策权

**伴侣，语速慢，"都行随便"，聊感受多，遇矛盾沉默**：
- 性格：抑郁质（敏感细腻型）+ 感性温柔型
- 沟通建议：耐心倾听，主动询问真实想法，关注感受

## 核心原则

1. **语言即性格** - 说话方式是内心的投射
2. **没有好坏** - 每种性格都有优势和劣势
3. **因人施教** - 根据性格调整沟通方式
4. **尊重差异** - 接受不同，不要试图改变
