# Unity 编辑器中英术语对照

以下为 Unity 2022.3 中文版编辑器界面的标准翻译对照。以 Unity 官方中文手册为准，部分术语在不同版本中叫法略有差异，以最常见译名为主。

---

## 主要窗口

| English | 中文（官方） | 常见别名 |
|---------|------------|---------|
| Hierarchy | 层级窗口 / 层级视图 | 层级面板、层次面板 |
| Inspector | 检视面板 / Inspector 窗口 | 检查器、属性面板 |
| Scene View | 场景视图 | Scene 窗口 |
| Game View | 游戏视图 | Game 窗口 |
| Project Window | 项目窗口 | 工程面板、Project 窗口 |
| Console | 控制台窗口 | Console 窗口 |
| Toolbar | 工具栏 | — |
| Package Manager | 包管理器 | Package Manager |
| Profiler | 性能分析器 | Profiler 窗口 |
