# Project 窗口

**打开方式：** `Window > General > Project`，快捷键 `Ctrl+5`（Windows）/ `Cmd+5`（macOS）

---

## 布局

### 两栏布局（默认）
- **左侧面板：** Favorites（收藏夹）+ 文件夹层级树
- **右侧面板：** 资源内容面板（图标或列表视图）
- 通过右上角 `⋮` 菜单切换为 **单栏布局**

### 导航栏（面包屑路径）
- 显示当前所在文件夹路径
- 点击路径中任意一段快速跳转到对应文件夹

---

## 左侧面板

### Favorites 收藏夹
- 位于左侧面板最顶部（文件夹层级之上）
- **拖拽**文件夹/资源到此处快速收藏
- 也可以**保存搜索查询**到收藏夹（搜索后点击工具栏保存按钮）

### 文件夹层级
- 按层级显示所有项目文件夹
- 点击三角展开/折叠
- **按住 Alt 点击三角**：递归展开/折叠所有子文件夹

---

## 工具栏

| 工具栏元素 | 功能 |
|-----------|------|
| **Create 菜单** | 在当前选中文件夹中创建新资源或文件夹（等同于 Assets > Create 菜单） |
| **搜索栏** | 按名称搜索资源。可选择搜索范围：All / 顶级文件夹 / 当前文件夹 / Asset Store |
| **按类型搜索** | 点击打开类型菜单筛选，添加 `t:` 过滤条件 |
| **按标签搜索** | 点击打开标签菜单筛选，添加 `l:` 过滤条件 |
| **保存搜索** | 将当前搜索保存到 Favorites 收藏夹 |

---

## 搜索语法

可在搜索栏直接输入过滤条件：

| 语法 | 含义 | 示例 |
|------|------|------|
| `t:类型` | 按资源类型过滤 | `t:Material`、`t:Prefab`、`t:Scene` |
| `l:标签` | 按标签过滤 | `l:Enemy`、`l:UI` |
| 普通文本 | 按名称搜索 | `Player` |
| 多种类型 | OR 组合 | `t:Prefab t:Scene`（找预制体或场景） |
| 类型+标签 | AND 组合 | `t:Material l:Terrain`（同时匹配两者） |

## 资源内容面板

### 图标/列表切换
- 底部滑块调整图标大小，左端为列表视图

### 快捷键（Project 窗口获得焦点时）

| 快捷键 | 功能 |
|--------|------|
| F | 在当前文件夹中定位选中资源 |
| Tab | 两栏布局中在两栏间切换焦点 |
| Ctrl/Cmd + F | 聚焦搜索栏 |
| Ctrl/Cmd + A | 全选 |
| Ctrl/Cmd + D | 复制选中资源 |
| Delete（Win） | 删除（弹出确认） |
| Shift + Delete（Win） | 直接删除（不弹确认） |
| F2（Win） / Enter（Mac） | 重命名 |
| Enter（Win） / Cmd+↓（Mac） | 打开选中资源 |
| Backspace（Win） / Cmd+↑（Mac） | 跳到父文件夹 |
| 右箭头 | 展开选中项 |
| 左箭头 | 折叠选中项 |

---

## Asset Labels（资源标签）

- 在 Project 窗口选中资源后，Inspector 底部预览面板显示标签图标
- 点击标签图标管理标签
- 输入新标签名后按 Space 或 Enter 创建新标签
- 标签可通过 `l:` 搜索语法查询

---

## Unity Search（高级搜索）— Ctrl+K

比 Project 窗口自带搜索更强大，可跨多个范围搜索：

| 前缀 | 搜索范围 | 示例 |
|------|---------|------|
| `p:` | Project 资源 | `p:Player` |
| `h:` | Hierarchy 物体 | `h:Main Camera` |
| `set:` | 设置 | `set:VFX` |
| `m:` | 菜单项 | `m:Prefab` |
| `?` | 帮助 | `?asset` |
| `find:` | 文件系统 | `find:Paint Mat` |
| `pkg:` | 包 | `pkg:vector` |
| `store:` | Asset Store | `store:texture` |

### 高级搜索功能
- **收藏结果：** 点击搜索结果的星标固定到顶部
- **保存搜索：** 保存为用户（个人）或项目（共享）搜索
- **搜索历史：** Alt+↑ / Alt+↓ 切换
- **视图切换：** 列表/网格/表格视图
