# Unity 快捷键汇总

**快捷键管理器：** `Edit > Shortcuts`（可查看和自定义所有快捷键）

---

## 工具快捷键

| 快捷键 | 工具 |
|--------|------|
| **Q** | 手形工具（Pan/View） |
| **W** | 移动（Translate） |
| **E** | 旋转（Rotate） |
| **R** | 缩放（Scale） |
| **T** | 矩形工具（Rect Tool） |
| **Y** | 变换工具（Move+Rotate+Scale） |

---

## Scene 视图导航

| 快捷键/操作 | 功能 |
|------------|------|
| **鼠标右键按住 + WASD** | 飞行模式（透视） |
| **鼠标中键拖拽** | 平移 |
| **Alt + 左键拖拽** | 环绕旋转 |
| **鼠标滚轮** | 缩放 |
| **Alt + 右键拖拽** | 缩放 |
| **F** | 聚焦选中物体 |
| **Shift + F** | 锁定视图到选中物体 |
| **方向键** | 摄像机前后左右移动 |
| **Shift + 方向键** | 加速移动 |
| **`**（反引号） | 切换 Overlays 菜单 |

---

## 文件与编辑

| 快捷键 | 功能 |
|--------|------|
| **Ctrl + N** | 新建场景 |
| **Ctrl + O** | 打开场景 |
| **Ctrl + S** | 保存场景 |
| **Ctrl + Shift + S** | 另存为 |
| **Ctrl + Shift + B** | 打开 Build Settings |
| **Ctrl + Z** | 撤销 |
| **Ctrl + Y**（Win）/ **Cmd + Shift + Z**（Mac） | 重做 |
| **Ctrl + X** | 剪切 |
| **Ctrl + C** | 复制 |
| **Ctrl + V** | 粘贴 |
| **Ctrl + D** | 原地复制（Duplicate） |
| **Shift + Del** | 删除 |
| **Ctrl + A** | 全选 |
| **F2** | 重命名 |
| **Delete** | 删除（带确认） |

---

## 游戏对象操作

| 快捷键 | 功能 |
|--------|------|
| **Ctrl + Shift + N** | 创建空对象 |
| **Alt + Shift + N** | 创建子空对象 |
| **Ctrl + Alt + F** | 移动物体到视图中心 |
| **Ctrl + Shift + F** | 物体与视图对齐 |
| **Alt + Shift + A** | 切换激活状态 |

## 组件操作

| 快捷键 | 功能 |
|--------|------|
| **Ctrl + Shift + A** | 打开 Add Component 菜单 |

---

## 对齐/V 键吸附

| 快捷键 | 功能 |
|--------|------|
| **V（按住）** | 顶点吸附（移动工具下按住 V 选择顶点） |
| **Ctrl（按住）** | 网格吸附（移动时按 Ctrl 吸附到网格） |

---

## 预制体操作

| 快捷键 | 功能 |
|--------|------|
| **双击 Hierarchy 中的 Prefab** | 进入预制体编辑模式 |
| **点击 Hierarchy 中的 ← 箭头** | 退出预制体编辑模式 |

---

## 常用鼠标操作

| 操作 | 效果 |
|------|------|
| 在 Hierarchy 中拖拽 GameObject | 改变父子关系 |
| 在 Project 窗口中拖拽资源到 Hierarchy | 实例化资源到场景 |
| 在 Project 窗口中拖拽资源到 Inspector | 赋值给对应类型字段 |
| 在 Scene 视图中拖拽 Gizmo 轴 | 沿轴移动/旋转/缩放 |
| 按住 Ctrl 拖拽移动 Gizmo | 按网格单位吸附移动 |
