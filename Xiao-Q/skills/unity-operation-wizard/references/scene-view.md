# Scene 视图

**打开方式：** `Window > General > Scene`，或双击 Hierarchy 中的场景

---

## 场景辅助图标（Scene Gizmo）

位于右上角。显示 Scene 摄像机的当前方向。

| 操作 | 效果 |
|------|------|
| 单击锥形臂（X/Y/Z 轴） | 切换到对应正交视图（如顶视图、前视图） |
| 右键单击立方体 | 弹出视角菜单（可回到 Free 默认视角） |
| 单击中心立方体 | 在 **Perspective（透视）** 和 **Orthographic（正交）** 之间切换 |
| Shift + 单击中心立方体 | 重置为默认透视图（侧面偏上） |
| 右上角挂锁图标 | 启用/禁用旋转。禁用后右键拖拽变为平移，而非旋转 |

## 视图导航

### 手形工具（快捷键 Q）

| 操作 | Windows | macOS |
|------|---------|-------|
| 平移 | 左键拖拽 | 左键拖拽 |
| 环绕/旋转 | **Alt + 左键拖拽** | **Option + 左键拖拽** |
| 缩放 | **Alt + 右键拖拽** | **Option + 右键拖拽** 或 **Control + 左键拖拽** |
| 加速 | 按住 **Shift** 的同时操作 | 按住 **Shift** |

### 通用导航（任意工具下可用）

| 操作 | 三键鼠标 | 两键鼠标/触控板 |
|------|----------|-----------------|
| 平移 | 按住**鼠标中键**拖拽 | Alt + Control + 左键拖拽 |
| 环绕 | Alt + 左键拖拽 | Alt + 左键拖拽 |
| 缩放 | 鼠标滚轮 或 Alt + 右键拖拽 | Alt + 右键拖拽 |

### 飞行模式（Flythrough Mode）

进入方式：**按住鼠标右键**

| 按键 | 效果 |
|------|------|
| W / S | 前 / 后 |
| A / D | 左 / 右 |
| Q / E | 上 / 下 |
| Shift（按住） | 加速 |
| 鼠标滚轮 | 动态调整移动速度 |

> 飞行模式仅在**透视模式**下有效。正交模式下按住右键为环绕。

## 聚焦与锁定

| 快捷键 | 效果 |
|--------|------|
| **F** | Scene 视图居中到选中对象 |
| **Shift + F** | 锁定视图跟随选中对象（对象移动时视角跟随） |
| Edit > Frame Selected | 等同于 F |
| Edit > Lock View to Selected | 等同于 Shift + F |

## View Options 工具栏

Scene 视图左上方的工具栏按钮：

| 按钮 | 功能 |
|------|------|
| **Draw Mode 下拉** | 渲染模式：Shaded（着色）/ Wireframe（线框）/ Shaded Wireframe（着色+线框）等 |
| **2D 按钮** | 2D / 3D 视图切换 |
| **光照开关** | 开启/关闭 Scene 视图中场景光照 |
| **音频开关** | 开启/关闭 Scene 视图中音频效果 |
| **Effects 按钮** | 管理场景可视化效果选项（天空盒、雾、光晕等） |
| **摄像机图标** | 调整 Scene 摄像机移动速度（Camera Speed） |
| **Gizmos 按钮** | 打开 Gizmos 菜单 |

---

## Gizmos 菜单

单击 Scene 视图工具栏中的 Gizmos 按钮打开：

| 属性 | 功能 |
|------|------|
| **3D Icons** | 开启后图标按距离缩放且被遮挡；关闭后固定大小始终显示在最前 |
| **3D Icons 滑块** | 调整 3D 图标的整体大小 |
| **Show Grid** | 显示/隐藏场景网格（可在 Preferences > Colors 中改色） |
| **Selection Outline** | 显示/隐藏选中对象的橙色轮廓（可在 Preferences > Colors 中调色） |
| **Selection Wire** | 显示/隐藏选中对象的线框叠加 |
| **Built-in Components 列表** | 控制各内置组件（Camera、Light、Collider 等）的图标和 Gizmo 显示 |
| 图标列 | 点击切换图标显示（点亮=可见，灰=隐藏） |
| Gizmo 列 | 复选框控制 Gizmo 绘制（如碰撞器线框、摄像机视锥体） |

---

## Overlays 菜单

按 **`**（反引号键，键盘左上角）打开或关闭 Overlays 菜单。可自定义 Scene 视图中的叠加工具栏。

---

## 方向键导航

| 按键 | 效果 |
|------|------|
| ↑ / ↓ | 摄像机前/后移动 |
| ← / → | 左/右平移 |
| Shift + 方向键 | 加速移动 |
