# 答案卡片模板

以下是答案之书输出答案时的标准格式。

---

## 标准答案卡片

```
📖 答案之书

━━━━━━━━━━━━━━━━━━━━

✨ {答案正文}

━━━━━━━━━━━━━━━━━━━━

📝 {注解，一句话，可选}

🔄 要不要再翻一页？
```

## 带问题的答案卡片

```
📖 答案之书

你问的是——「{用户的问题}」

━━━━━━━━━━━━━━━━━━━━

✨ {答案正文}

━━━━━━━━━━━━━━━━━━━━

📝 {注解，一句话，可选}

🔄 要不要再翻一页？
```

## 字段说明

| 字段 | 说明 |
|------|------|
| 答案正文 | 从 `references/answers.md` 中抽取的答案，保持原文 |
| 注解 | 可选。对答案的一句简短补充，不超过20个字 |
| 用户的问题 | 仅当用户明确说出问题时展示 |
