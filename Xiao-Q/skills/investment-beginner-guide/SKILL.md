# Investment Beginner Guide — 理财入门指南

Use when the user mentions 理财入门、怎么理财、投资新手、investment beginner、how to invest、financial planning, or asks about basic investing. NOT for 具体投资建议、股票推荐、基金推荐、代客理财.

## 描述

为理财小白提供基础知识科普，解释常见理财产品（存款、基金、债券、保险等）的特点和风险等级，帮助用户建立正确的理财观念，理解"风险与收益成正比"等核心原则，制定适合自身情况的入门理财计划。

## 重要限制（请提前告知用户）

- **不提供具体投资建议**：不推荐任何具体产品、股票或基金，只做知识科普
- **不保证收益**：任何理财都有风险，过往收益不代表未来表现
- **不替代专业顾问**：复杂财务规划请咨询持牌理财师
- **信息有时效性**：利率、政策等可能随时变化，以官方最新公告为准

- 解释理财术语（年化收益率、复利、风险等级等）

- 纠正常见理财误区

## 执行步骤

### Step 1: 了解用户情况
1. 确认理财知识水平（零基础/有一点了解）
2. 了解月收入和固定支出
3. 了解可用于理财的金额
4. 评估风险偏好（保守/稳健/进取）

### Step 2: 知识科普
1. 解释用户关心的理财产品
2. 说明风险与收益的关系
3. 对比不同产品的特点
4. 纠正可能存在的误区

### Step 3: 给出配置思路
1. 根据情况建议资金分配比例
2. 说明每部分资金的作用
3. 强调应急资金的重要性
4. 给出循序渐进的学习路径

## 输出原则

1. **中立客观**：只陈述事实和原理，不推荐具体产品
2. **风险优先**：先说风险再说收益，培养风险意识
3. **因人而异**：根据用户实际情况给建议，不搞一刀切
4. **通俗易懂**：避免过多专业术语，用大白话解释
5. **免责声明**：每次输出都提醒"不构成投资建议"

## 常见问题（FAQ）

**Q: 理财和存银行有什么区别？**
A: 存银行是最简单的理财方式之一，利率固定风险极低。其他理财产品收益可能更高，但也有本金损失的风险。

**Q: 月入5000能理财吗？**
A: 完全可以。先留出生活费和应急金，哪怕每月只投500元到货币基金或指数基金定投，长期也能积累可观的财富。

**Q: 年化收益率怎么理解？**
A: 就是投1万块钱放一整年能赚多少。年化5%意味着1万元一年赚500元。注意"年化"是换算值，实际持有时间不同收益不同。

**Q: 基金定投是什么意思？**
A: 每月固定时间投固定金额买基金，类似"强制储蓄"。好处是分散买入时点的风险，不用纠结什么时候买。

**Q: 理财有没有稳赚不赔的？**
A: 国债和存款保险范围内的银行存款几乎无风险，但收益较低。收益越高风险越大，这是铁律。号称"稳赚高收益"的多为骗局。

**Q: 什么时候开始理财最好？**
A: 越早越好。时间是复利最好的朋友——25岁开始每月投500元，比35岁开始每月投1000元的终值还高。

## 最佳实践

1. **先学后投**：花1-2周了解基础概念再动手，不要急于入场
2. **从小额开始**：初期用少量资金实践，积累经验后再增加
3. **坚持记账**：了解自己的收支结构是理财第一步
4. **定期复盘**：每季度回顾一次资产配置，根据情况微调
5. **持续学习**：关注基本经济常识，但不要被市场噪音左右

### 常见误用

- **误用 1**：把科普当投资建议直接买入 → 需要自行研究确认后再决策
- **误用 2**：只问"买什么赚钱" → 理财入门应先建立知识体系再行动
- **误用 3**：忽略风险提示 → 每种产品的风险说明都应认真对待
