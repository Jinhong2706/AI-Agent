# 艺术作品解读 (Artwork Interpreter)

帮助用户解读和鉴赏艺术作品。通过六维分析法（形式、内容、技法、情感、思想/主题、背景）或极简四维法（形式→内容→情感→思想）全面分析艺术作品。

## 核心理念

**用专业视角欣赏艺术，用系统方法解读美感。**

## 六维解读框架

| 维度 | 核心问题 | 分析重点 |
|------|----------|----------|
| 1️⃣ **形式** | 看起来/听起来什么样？ | 色彩、线条、构图、节奏 |
| 2️⃣ **内容** | 表现了什么？ | 题材、人物、情节、物象 |
| 3️⃣ **技法** | 如何创作的？ | 媒介、手法、工艺、创新 |
| 4️⃣ **情感** | 带来什么感受？ | 情绪、氛围、共鸣 |
| 5️⃣ **思想** | 表达了什么？ | 主题、象征、哲学思考 |
| 6️⃣ **背景** | 为什么这样创作？ | 作者、时代、流派、文化 |

## 极简四维版

```
形式 → 内容 → 情感 → 思想（含背景）
```

适用场景：时间有限、快速浏览、入门练习

## 输出成果

**Word文档格式的艺术鉴赏报告**，包含：

1. 📄 **封面** - 作品名称、艺术家、创作年代
2. ℹ️ **作品基本信息** - 类型、尺寸、收藏地等
3. 🔍 **六维/四维深度分析** - 每维详细解读
4. ⭐ **综合评述** - 艺术价值、历史地位、个人感悟
5. 📚 **延伸阅读** - 相关作品、参考资料

## 使用方法

### 1. 安装依赖

```bash
python scripts/install_dependencies.py
```

或手动安装：
```bash
pip install python-docx
```

### 2. 准备数据文件

创建JSON文件，包含六维分析内容。参考 `examples/mona_lisa_example.json`

### 3. 生成报告

```bash
python scripts/generate_report.py data.json 艺术鉴赏报告.docx
```

## 示例

### 解读《蒙娜丽莎》

```bash
python scripts/generate_report.py examples/mona_lisa_example.json 蒙娜丽莎鉴赏报告.docx
```

生成的报告包含：
- 封面：《蒙娜丽莎》艺术鉴赏报告
- 作品信息：达·芬奇、约1503-1519年、卢浮宫
- 六维分析：形式、内容、技法、情感、思想、背景
- 综合评述：艺术价值、历史地位、个人感悟
- 延伸阅读：相关作品推荐

## 适用艺术形式

- 🎨 **绘画** - 油画、水彩、水墨、版画等
- 🗿 **雕塑** - 石雕、木雕、金属雕塑等
- 🎵 **音乐** - 古典音乐、现代音乐等
- 💃 **舞蹈** - 古典舞、现代舞等
- 📖 **文学** - 小说、诗歌、戏剧等
- 🎬 **影视** - 电影、纪录片等
- 🎭 **戏剧** - 话剧、歌剧、戏曲等

## 使用建议

1. **循序渐进**：初学者从四维极简版开始
2. **多看多练**：艺术鉴赏需要积累
3. **结合背景**：背景知识对理解作品至关重要
4. **主观与客观**：既要有客观分析，也要有主观感受
5. **跨艺术门类**：尝试用同一框架分析不同艺术形式
